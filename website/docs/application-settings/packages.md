# Packages

![Packages Settings](/images/application-packages.png)

The Packages page shows installed extension packages and allows you to
manage them.

## Overview

Packages are extensions that add functionality to Rayforge. They can
provide:

- New operations
- Import/export filters
- Machine drivers
- UI enhancements
- Material libraries

## Installed Packages

The package list shows all installed packages with:

- **Name**: The package's display name
- **Version**: Current installed version
- **Description**: Brief description of functionality
- **Status**: Whether enabled or disabled

## Managing Packages

### Installing Packages

1. Click the **Install** button
2. Select a package file (`.zip` or package repository)
3. Confirm the installation
4. Wait for the installation to complete

### Updating Packages

When updates are available:

1. An update indicator appears next to the package
2. Click the **Update** button to install the update
3. Restart Rayforge if prompted

### Removing Packages

1. Select the package to remove
2. Click the **Remove** button
3. Confirm the removal

:::warning
Removing a package may affect documents that use its features.
:::

## Package Sources

Packages can be installed from:

- **Local files**: `.zip` archives downloaded manually
- **Repositories**: Git repositories containing packages
- **Official registry**: Curated packages from the Rayforge team
