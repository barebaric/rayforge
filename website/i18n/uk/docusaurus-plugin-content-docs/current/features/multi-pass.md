# Багатопрохідність

Багатопрохідність повторює шлях різання або гравіювання кілька разів, з опціональним опусканням по Z між проходами. Це корисно для товстих матеріалів або створення глибшого гравіювання.

## Як це працює

Кожен прохід трасує той самий шлях знову. При увімкненому Z-кроку лазер переміщується ближче до матеріалу між проходами, поступово ріжучи глибше.

## Налаштування

### Кількість проходів

Скільки разів повторити весь крок (1-100). Кожен прохід слідує тим самим шляхом.

- **1 прохід:** Одинарний різ (за замовчуванням)
- **2-3 проходи:** Типово для середньо-товстих матеріалів
- **4+ проходів:** Дуже товсті або тверді матеріали

### Z-крок за прохід

Відстань опускання осі Z між проходами (0-50 мм). Працює тільки якщо ваша машина має контроль Z-осі.

- **0 мм:** Всі проходи на тій самій глибині (за замовчуванням)
- **Товщина матеріалу ÷ проходи:** Поступове різання на глибину
- **Малі кроки (0.1-0.5мм):** Тонкий контроль для глибокого гравіювання

:::warning Потрібна вісь Z
Z-крок працює тільки з машинами, які мають моторизований контроль Z-осі. Для машин без Z-осі всі проходи відбуваються на тій самій висоті фокусу.
:::

## Коли використовувати багатопрохідність

**Різання товстих матеріалів:**

Кілька проходів на тій самій глибині часто ріжуть чистіше, ніж один повільний прохід. Перший прохід створює керф, а наступні проходи слідують тим самим шляхом ефективніше.

**Глибоке гравіювання:**

З Z-кроком ви можете вирізувати глибокі рельєфні патерни або гравіювання, які були б неможливі за один прохід.

**Покращена якість краю:**

Кілька швидших проходів часто дають чистіші краї, ніж один повільний прохід, особливо на матеріалах, які легко обгорають.

## Поради

- Почніть з 2-3 проходів на вашій звичайній швидкості різання
- Для товстих матеріалів збільште кількість проходів замість уповільнення
- Увімкніть Z-крок тільки якщо ваша машина підтримує це
- Тестуйте на відходах матеріалу, щоб знайти оптимальну кількість проходів
