# Відстеження використання

Rayforge включає необов'язкове анонімне відстеження використання, щоб допомогти нам зрозуміти, як використовується програма, та пріоритезувати майбутній розвиток. Ця сторінка пояснює, що ми відстежуємо, як це працює, та вашу приватність.

## Повністю необов'язково

Відстеження використання є **повністю необов'язковим**. При першому запуску Rayforge вас запитають, чи хочете ви взяти участь:

- **Так**: Анонімні дані про використання будуть надіслані на наш сервер аналітики
- **Ні**: Жодні дані ніколи не збираються та не передаються

Ви можете змінити цей вибір у будь-який час у загальних налаштуваннях.

## Що ми відстежуємо

Коли увімкнено, ми збираємо лише анонімні дані про перегляд сторінок –
подібно до веб-аналітики. Ось що ми можемо бачити:

| Дані                         | Приклад                   |
| ---------------------------- | ------------------------- |
| Роздільна здатність екрану   | 1920x1080                 |
| Мовні налаштування           | uk-UA                     |
| Переглянуті сторінки/діалоги | /machine-settings/general |
| Час на сторінці              | 6хв 3с                    |
| ID сесії                     | a1b2c3d4-e5f6-...         |

ID сесії – це випадковий ідентифікатор, що генерується кожного разу при
запуску Rayforge. Він дозволяє нам розрізняти різні сесії використання,
зберігаючи відстеження повністю анонімним.

## Що ми бачимо

Ось приклад того, як виглядає панель аналітики:

| Шлях                      | Відвідувачів | Візитів | Переглядів | Показник відмов | Тривалість візиту |
| ------------------------- | ------------ | ------- | ---------- | --------------- | ----------------- |

## Що ми НЕ відстежуємо

Ми віддані вашій приватності:

- **Без особистої інформації** – Без імен, електронних пошт або облікових записів
- **Без вмісту файлів** – Ваші проекти та дизайн залишаються приватними
- **Без ідентифікаторів машини** – Без серійних номерів або унікальних ID
- **Без збереження IP-адрес** – Ми використовуємо Umami Analytics, який не зберігає IP
- **Без міжсайтового відстеження** – Дані ізольовані лише для Rayforge

## Чому ми відстежуємо

Дані про використання допомагають нам:

- **Визначати популярні функції** – Знати, що добре працює
- **Знаходити проблемні місця** – Бачити, де користувачі витрачають час або застрягають
- **Пріоритезувати розробку** – Зосередитися на функціях, які люди фактично використовують
- **Розуміти різноманітність** – Знати, які мови та розміри екрану підтримувати

## Як це працює

Rayforge використовує [Umami](https://umami.is/), платформу аналітики з відкритим вихідним кодом, орієнтовану на приватність. Відстеження:

- Надсилає невеликі HTTP-запити у фоновому режимі
- Не впливає на продуктивність програми
- Працює офлайн (невдалі запити ігноруються безшумно)
- Використовує загальний User-Agent для запобігання фінгерпринтингу

## Вимкнення відстеження

Ви можете вимкнути відстеження у будь-який час:

1. Відкрийте **Налаштування** → **Загальні**
2. Вимкніть **Надсилати анонімну статистику використання**

Коли вимкнено, абсолютно жодні дані не надсилаються.
