# Режим Симуляції

![Режим Симуляції](/screenshots/main-simulation.png)

Режим симуляції показує, як буде виконуватися ваше лазерне завдання перед запуском на машині. Ви можете переглядати G-код крок за кроком і бачити точно, що відбудеться.

## Активація Режиму Симуляції

- **Клавіатура**: Натисніть <kbd>F11</kbd>
- **Меню**: Перейдіть до **Вид → Симулювати Виконання**
- **Панель інструментів**: Натисніть кнопку симуляції

## Візуалізація

### Теплокарта Швидкості

Операції забарвлюються за швидкістю:

| Швидкість     | Колір     |
| ------------- | --------- |
| Найповільніша | Синій     |
| Повільна      | Бірюзовий |
| Середня       | Зелений   |
| Швидка        | Жовтий    |
| Найшвидша     | Червоний  |

Кольори відносні до діапазону швидкостей вашого завдання - синій є мінімумом, червоний - максимумом.

### Прозорість Потужності

Непрозорість ліній показує потужність лазера:

- **Тьмяні лінії** = Низька потужність (переміщення, легке гравірування)
- **Суцільні лінії** = Висока потужність (різання)

## Елементи Керування Відтворенням

Використовуйте елементи керування внизу полотна:

- **Відтворити/Пауза** (<kbd>Пробіл</kbd>): Запустити або зупинити автоматичне відтворення
- **Повзунок прогресу**: Перетягуйте для прокручування завдання
- **Стрілки**: Переглядати інструкції по одній

Симуляція та перегляд G-коду залишаються синхронізованими - перегляд симуляції виділяє відповідний G-код, а натискання на рядки G-коду переходить до цієї точки в симуляції.

## Редагування Під Час Симуляції

Ви можете редагувати робочі деталі під час симуляції. Переміщуйте, масштабуйте або обертайте об'єкти, і симуляція оновлюється автоматично.
