# 缝隙补偿

缝隙是切割过程中激光束移除的材料。缝隙补偿调整刀具路径以考虑这一点，确保切割零件与其设计尺寸匹配。

## 什么是缝隙？

**缝隙** = 切割过程移除的材料宽度。

**示例：**
- 激光光斑大小：0.2mm
- 材料相互作用：每边增加约 0.1mm
- **总缝隙：** 约 0.4mm

---

## 缝隙补偿如何工作

缝隙补偿**向内或向外偏移刀具路径**以考虑材料移除：

**对于外部切割（切割零件）：**
- **向外**偏移路径半个缝隙宽度
- 结果：最终零件具有正确的尺寸

**对于内部切割（切割孔）：**
- **向内**偏移路径半个缝隙宽度
- 结果：最终孔具有正确的尺寸

**0.4mm 缝隙的示例：**

```
原始路径：  50mm 方形
补偿：     向外偏移 0.2mm（半缝隙）
激光跟随：  50.4mm 方形
切割后：   零件测量 50.0mm（完美！）
```

---

## 测量缝隙

**准确的缝隙测量过程：**

1. **创建测试文件：**
   - 绘制 50mm x 50mm 方形
   - 绘制一个圆形（任何大小，用于内部切割测试）

2. **切割测试：**
   - 使用正常的切割设置
   - 完全切割穿
   - 让材料冷却

3. **测量：**
   - **外部方形（零件）：** 用卡尺测量
     - 如果 < 50mm，缝隙向外移除
     - 缝隙 = (50 - 测量值) x 2
   - **内部圆形（孔）：** 测量直径
     - 如果 > 设计直径，缝隙向内移除
     - 缝隙 = (测量值 - 设计值) / 2

4. **平均：** 使用多次测量的平均值

**影响缝隙的变量：**
- 激光功率（越高 = 越宽）
- 切割速度（越慢 = 越宽）
- 材料类型和密度
- 对焦距离
- 空气辅助压力

## 手动缝隙补偿

如果自动缝隙补偿不可用，在设计软件中补偿：

**Inkscape：**

1. **选择路径**
2. **路径 → 动态偏移**（Ctrl+J）
3. **拖动偏移**半个缝隙测量值
   - 零件向外（使路径更大）
   - 孔向内（使路径更小）
4. **路径 → 对象转路径**完成

**Illustrator：**

1. **选择路径**
2. **对象 → 路径 → 偏移路径**
3. **输入偏移值：**（缝隙 / 2）
   - 正值向外，负值向内
4. **确定**应用

**Fusion 360 / CAD：**

- 导出前偏移草图实体
- 使用缝隙/偏移尺寸
