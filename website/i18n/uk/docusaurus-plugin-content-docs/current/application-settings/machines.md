# Машини

![Налаштування машин](/screenshots/application-machines.png)

Сторінка "Машини" у налаштуваннях додатку дозволяє керувати профілями
машин. Кожен профіль містить всю конфігурацію для конкретної
лазерної машини.

## Профілі машин

Профілі машин зберігають повну конфігурацію для лазерного різака або
гравера, включаючи:

- **Загальні налаштування**: Назва, швидкості, прискорення
- **Налаштування обладнання**: Розміри робочої області, конфігурація осей
- **Налаштування лазера**: Діапазон потужності, частота PWM
- **Налаштування пристрою**: Послідовний порт, швидкість передачі, тип прошивки
- **Налаштування G-code**: Власні параметри діалекту G-code
- **Налаштування камери**: Калібрування та вирівнювання камери

## Керування машинами

### Додавання нової машини

1. Натисніть кнопку **Додати нову машину**
2. Введіть описову назву для вашої машини
3. Налаштуйте параметри машини (див.
    [Налаштування машини](../machine/general) для деталей)
4. Натисніть **Зберегти** для створення профілю

### Перемикання між машинами

Використовуйте випадаючий список вибору машини у головному вікні для перемикання між
налаштованими машинами. Всі налаштування, включаючи вибрану машину,
зберігаються між сесіями.

### Дублювання машини

Для створення схожого профілю машини:

1. Виберіть машину для дублювання
2. Натисніть кнопку **Дублювати**
3. Перейменуйте нову машину та налаштуйте параметри за потреби

### Видалення машини

1. Виберіть машину для видалення
2. Натисніть кнопку **Видалити**
3. Підтвердьте видалення

:::увага
Видалення профілю машини не можна скасувати. Переконайтеся, що ви
записали всі важливі налаштування перед видаленням.
:::
