# Оптимізація шляху

Оптимізація шляху переставляє сегменти різання для мінімізації відстані переміщення. Лазер переміщається ефективно між різами замість хаотичних стрибків по робочій зоні.

## Як це працює

Без оптимізації шляхи ріжуться в порядку їх появи у вашому файлі дизайну. Оптимізація аналізує всі сегменти шляху та переставляє їх так, щоб лазер проходив найкоротшу загальну відстань між різами.

**До оптимізації:** Лазер стрибає туди-сюди по матеріалу  
**Після оптимізації:** Лазер переміщається послідовно від різу до різу

## Налаштування

### Увімкнути оптимізацію

Увімкніть або вимкніть оптимізацію шляху. Увімкнено за замовчуванням для більшості операцій.

## Коли використовувати оптимізацію

**Увімкнути для:**

- Дизайнів з багатьма окремими формами
- Зменшення загального часу роботи
- Мінімізації зносу системи переміщення
- Складних вкладених макетів

**Вимкнути для:**

- Дизайнів, де порядок різання важливий (наприклад, внутрішні елементи перед зовнішніми)
- Налагодження проблем зі шляхами
- Коли потрібен передбачуваний, повторюваний порядок виконання

## Як це впливає на вашу роботу

**Економія часу:** Може зменшити час роботи на 20-50% для дизайнів з багатьма окремими різами.

**Ефективність переміщення:** Менше швидких переміщень означає менший знос ременів, моторів та підшипників.

**Розподіл тепла:** Оптимізовані шляхи можуть концентрувати тепло в одній області. Для чутливих до тепла матеріалів врахуйте, чи має значення порядок.

:::tip
Оптимізація запускається автоматично. Просто увімкніть її, і програма зробить усе інше.
:::
