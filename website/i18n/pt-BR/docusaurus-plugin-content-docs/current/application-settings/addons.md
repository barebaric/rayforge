# Addons

![Configurações de Addons](/screenshots/application-addons.png)

A página Addons mostra addons de extensão instalados e permite
gerenciá-los.

## Visão Geral

Addons são extensões que adicionam funcionalidade ao Rayforge. Eles podem
fornecer:

- Novas operações
- Filtros de importação/exportação
- Drivers de máquina
- Melhorias na interface
- Bibliotecas de materiais

## Addons Instalados

A lista de addons mostra todos os addons instalados com:

- **Nome**: O nome de exibição do addon
- **Versão**: Versão atual instalada
- **Descrição**: Breve descrição da funcionalidade
- **Status**: Se está ativado ou desativado

## Gerenciando Addons

### Instalando Addons

1. Clique no botão **Instalar**
2. Selecione um arquivo de addon (`.zip` ou repositório de addon)
3. Confirme a instalação
4. Aguarde a conclusão da instalação

### Atualizando Addons

Quando atualizações estão disponíveis:

1. Um indicador de atualização aparece ao lado do addon
2. Clique no botão **Atualizar** para instalar a atualização
3. Reinicie o Rayforge se solicitado

### Removendo Addons

1. Selecione o addon a remover
2. Clique no botão **Remover**
3. Confirme a remoção

:::warning
Remover um addon pode afetar documentos que usam seus recursos.
:::

## Fontes de Addons

Addons podem ser instalados de:

- **Arquivos locais**: Arquivos `.zip` baixados manualmente
- **Repositórios**: Repositórios Git contendo addons
- **Registro oficial**: Addons selecionados pela equipe Rayforge
