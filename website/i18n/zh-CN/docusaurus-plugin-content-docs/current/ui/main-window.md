# 主窗口

Rayforge 主窗口是您创建和管理激光作业的主要工作区。

## 窗口布局

![主窗口](/screenshots/main-standard.png)

### 1. 菜单栏

通过组织菜单访问所有 Rayforge 功能：

- **文件**：打开、保存、导入、导出和最近文件
- **编辑**：撤销、重做、复制、粘贴、首选项
- **视图**：缩放、网格、标尺、面板和视图模式
- **对象**：添加、编辑和管理操作
- **机器**：连接、点动、归零、开始/停止作业
- **帮助**：关于、捐赠、保存调试日志

### 2. 工具栏

快速访问常用控件：

- **机器下拉菜单**：选择您的机器、查看连接状态以及在作业期间查看预计剩余时间
- **WCS 下拉菜单**：选择活动工作坐标系（G53-G59）
- **仿真切换**：启用/禁用作业仿真模式
- **聚焦激光**：切换激光对焦模式
- **作业控制**：归零、框选、发送、暂停和取消按钮

机器下拉菜单在工具栏中直接显示机器的连接状态和当前状态（例如空闲、运行）。在作业执行期间，它还显示预计剩余时间。

WCS 下拉菜单允许您在坐标系之间快速切换。有关更多信息，请参阅[工作坐标系](../general-info/coordinate-systems)。

工件、标签、相机画面、空走移动和其他元素的可见性切换已移至画布本身的叠加按钮上，因此它们在您工作时始终触手可及。

### 3. 画布

主要工作区，您在这里：

- 导入和排列设计
- 预览刀具路径
- 相对于机器原点定位对象
- 测试框选边界

**画布控制：**

- **平移**：中键拖动或 <kbd>space</kbd> + 拖动
- **缩放**：鼠标滚轮或 <kbd>ctrl+"+"</kbd> / <kbd>ctrl+"-"</kbd>
- **重置视图**：<kbd>ctrl+0</kbd> 或视图 → 重置缩放

### 4. 图层面板

管理操作和图层分配：

- 查看项目中的所有操作
- 为设计元素分配操作
- 重新排序操作执行
- 启用/禁用单个操作
- 配置操作参数

### 5. 属性面板

配置选定对象或操作的设置：

- 操作类型（轮廓、光栅等）
- 功率和速度设置
- 通过次数
- 高级选项（过扫描、缝隙、标签）

### 6. 底部面板

底部面板提供控制台、G 代码查看器和文档资产（材料和草图）的标签页。点动控制和 WCS 管理始终在右侧可见。预计作业时间显示在图层面板上方的图层列表标题中。

有关详细信息，请参阅[底部面板](bottom-panel)。

## 窗口管理

### 面板

根据需要显示/隐藏面板：

- **底部面板**：视图 → 底部面板（<kbd>ctrl+l</kbd>）

### 全屏模式

全屏专注于您的工作：

- 进入：<kbd>f11</kbd> 或视图 → 全屏
- 退出：<kbd>f11</kbd> 或 <kbd>esc</kbd>

## 自定义

在 **编辑 → 设置** 中自定义界面：

- **主题**：浅色、深色或系统
- **单位**：毫米或英寸
- **网格**：显示/隐藏和配置网格间距
- **标尺**：显示/隐藏画布上的标尺
