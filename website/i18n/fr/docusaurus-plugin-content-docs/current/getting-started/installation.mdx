# Installation

import InstallGuide from '@site/src/components/InstallGuide';

Rayforge est disponible pour Linux, Windows et macOS. Choisissez votre plateforme ci-dessous
et suivez les instructions d'installation.

## Configuration Système Requise

| Ressource     | Installé   | Recommandé               |
|---------------|------------|--------------------------|
| Espace disque | < 1 Go     | Variable selon le projet |
| Mémoire RAM   | 1 Go       | Variable selon le projet |

Les besoins réels en disque et en mémoire dépendent de la complexité du projet.
Les projets simples peuvent nécessiter peu de ressources au-delà de l'installation,
tandis que les projets complexes peuvent facilement consommer 10 Go de RAM ou plus
et nécessiter un espace disque supplémentaire important.
