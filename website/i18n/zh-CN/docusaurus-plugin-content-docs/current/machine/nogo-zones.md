# 禁区

禁区定义了工作表面上激光不应进入的限制区域。在运行或导出作业之前，Rayforge 会检查是否有刀具路径进入已启用的禁区，并在检测到碰撞时向您发出警告。

![禁区](/screenshots/machine-nogo-zones.png)

## 添加禁区

打开 **设置 → 机器**，导航到 **禁区** 页面。点击添加按钮创建新区域，然后选择其形状和位置。

每个区域有以下设置：

- **名称**：区域的描述性标签
- **启用**：在不删除的情况下开启或关闭区域
- **形状**：矩形、长方体或圆柱体
- **位置（X、Y、Z）**：区域在工作表面上的放置位置
- **尺寸**：宽度、高度和深度（或圆柱体的半径）

## 碰撞警告

当您运行或导出作业时，Rayforge 会根据已启用的禁区检查所有刀具路径。如果刀具路径穿过禁区，将出现警告对话框，您可以选择取消或自行承担风险继续。

## 可见性

禁区在 2D 和 3D 画布上都以半透明叠加显示。使用画布叠加中的禁区切换按钮来显示或隐藏它们。可见性设置在会话之间保持记忆。
