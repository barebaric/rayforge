# 框架轮廓

框架轮廓围绕整个设计创建一个简单的矩形切割路径。这是添加干净边框或从材料板材上切割作品的最快方法。

## 概述

框架轮廓操作：

- 围绕所有内容创建矩形边界
- 添加可配置的偏移/边距
- 支持缝隙补偿以获得准确的尺寸
- 适用于画布上任何对象组合

![框架轮廓步骤设置](/screenshots/step-settings-frame-outline-general.png)

## 何时使用框架轮廓

使用框架轮廓用于：

- 围绕设计添加装饰边框
- 从材料板材上切割作品
- 创建简单的矩形边界
- 无需复杂路径计算的快速框架

**不要使用框架轮廓用于：**

- 围绕多个对象的不规则形状（改用[收缩包裹](shrink-wrap)）
- 切割单个零件（改用[轮廓](contour)）
- 遵循设计的精确形状

## 创建框架轮廓操作

### 第 1 步：排列设计

1. 将所有对象放在画布上
2. 相对于框架定位它们
3. 框架将围绕所有内容的边界框计算

### 第 2 步：添加框架轮廓操作

- **菜单：** 操作 → 添加框架轮廓
- **右键点击：** 上下文菜单 → 添加操作 → 框架轮廓

### 第 3 步：配置设置

配置框架参数：

- **功率和速度：** 匹配材料的切割要求
- **偏移：** 从内容边缘到框架的距离
- **路径偏移：** 内部、外部或中心线切割

## 关键设置

### 功率和速度

**功率（%）：**

- 切割框架的激光强度
- 匹配材料的切割要求

**速度（mm/min）：**

- 激光移动的速度
- 较厚的材料较慢

**次数：**

- 切割框架的次数
- 通常 1-2 次
- 为较厚的材料添加次数

### 偏移距离

**偏移（mm）：**

- 从设计边界框到框架的距离
- 在作品周围创建边距/边框

**典型值：**

- **0mm：** 框架接触设计边缘
- **2-5mm：** 小边距，外观干净
- **10mm+：** 用于安装或处理的大边框

### 路径偏移（切割侧）

控制激光相对于框架路径切割的位置：

| 切割侧       | 描述                   | 用于                               |
| ------------ | ---------------------- | ---------------------------------- |
| **中心线**   | 直接在路径上切割       | 标准切割                           |
| **外部**     | 在框架路径外切割       | 使框架略微变大                     |
| **内部**     | 在框架路径内切割       | 使框架略微变小                     |

### 缝隙补偿

框架轮廓支持缝隙补偿：

- 自动调整激光束宽度
- 确保准确的最终尺寸
- 使用激光头设置中的缝隙值

## 后处理选项

![框架轮廓后处理设置](/screenshots/step-settings-frame-outline-post.png)

### 多次遍

多次切割框架：

- **次数：** 重复次数
- **Z 下步：** 遍之间降低 Z（需要 Z 轴）
- 用于厚材料

### 固定标签

添加标签以保持框架件连接：

- 防止零件在切割期间掉落
- 配置标签宽度、高度和间距
- 有关详细信息，请参阅[固定标签](../holding-tabs)

## 用例

### 装饰边框

**场景：** 围绕牌匾或标志添加干净的矩形边框

**过程：**

1. 设计内容（文本、标志等）
2. 添加 3-5mm 偏移的框架轮廓
3. 以装饰划线设置切割（低功率）

**结果：** 专业外观的带框作品

### 从板材上切割出来

**场景：** 从材料板材上移除完成的作品

**过程：**

1. 完成所有其他操作（雕刻、轮廓切割）
2. 添加框架轮廓作为最后一个操作
3. 设置偏移以包括小边距

**好处：**

- 从板材干净分离
- 一致的边缘质量
- 作为最后一步容易执行

### 批量处理边界

**场景：** 为多个嵌套零件创建切割边界

**过程：**

1. 在画布上排列所有零件
2. 为零件添加单独的轮廓操作
3. 围绕所有内容添加框架轮廓
4. 框架最后切割（在单独图层中）

**顺序：** 雕刻 → 零件轮廓 → 框架轮廓

## 提示和最佳实践

### 图层顺序

**最佳实践：**

- 将框架轮廓放在自己的图层中
- **最后**执行框架
- 这确保所有其他工作首先完成

**为什么最后？**

- 材料在其他操作期间保持固定
- 防止零件移动
- 更干净的最终结果

### 偏移选择

**选择偏移：**

- **0-2mm：** 紧密配合，最小材料浪费
- **3-5mm：** 标准边距，外观专业
- **10mm+：** 用于处理/安装的额外材料

**考虑：**

- 作品的最终用途
- 边缘是否可见
- 材料成本和可用性

### 质量设置

**为了干净的框架切割：**

- 使用空气辅助
- 确保适当的对焦
- 多次较快的遍通常比一次慢遍更好
- 保持材料平坦并固定

## 与其他操作结合

### 框架 + 雕刻 + 轮廓

完成作品的典型工作流程：

1. **图层 1：** 雕刻细节（文本、图像）
2. **图层 2：** 轮廓切割单个零件
3. **图层 3：** 框架轮廓（切割出来）

**执行顺序确保：**

- 雕刻在材料平坦固定时发生
- 零件细节在最终分离前切割
- 框架最后切割所有内容

### 框架与收缩包裹

| 功能         | 框架轮廓                   | 收缩包裹             |
| ------------ | -------------------------- | -------------------- |
| **形状**     | 始终是矩形                 | 跟随对象轮廓         |
| **速度**     | 非常快（4 条线）           | 取决于复杂度         |
| **用例**     | 简单边框、切割出来         | 高效材料使用         |
| **灵活性**   | 固定矩形                   | 适应设计             |

**选择框架轮廓当：**

- 想要矩形边框
- 首选简单性
- 从板材上切割出来

**选择收缩包裹当：**

- 想要最小化材料浪费
- 设计有不规则形状
- 效率很重要

## 故障排除

### 框架太紧/太松

- **调整：** 偏移距离设置
- **检查：** 路径偏移（内部/外部/中心线）
- **验证：** 缝隙补偿正确

### 框架不出现

- **检查：** 画布上有对象
- **验证：** 操作已启用
- **查看：** 框架可能在可见区域外（缩小）

### 框架切割到设计中

- **增加：** 偏移距离
- **检查：** 正确定位对象
- **验证：** 边界框计算包括所有对象

### 切割深度不一致

- **检查：** 材料平坦
- **验证：** 对焦距离正确
- **尝试：** 较低功率的多次遍

## 技术细节

### 边界框计算

框架轮廓使用以下组合边界框：

- 画布上的所有工件
- 它们的最终变换位置
- 包括任何应用的旋转/缩放

### 路径生成

1. 计算组合边界框
2. 应用偏移距离
3. 应用路径偏移（内部/外部/中心线）
4. 应用缝隙补偿
5. 生成矩形 G 代码路径

### G 代码示例

```gcode
G0 X5 Y5           ; 移动到框架开始（带偏移）
M3 S200            ; 激光开，80% 功率
G1 X95 Y5 F500     ; 切割底边
G1 X95 Y95         ; 切割右边
G1 X5 Y95          ; 切割顶边
G1 X5 Y5           ; 切割左边（完成）
M5                 ; 激光关
```
