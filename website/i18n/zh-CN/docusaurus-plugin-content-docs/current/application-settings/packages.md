# 包

![包设置](/screenshots/application-packages.png)

包页面显示已安装的扩展包并允许您管理它们。

## 概述

包是为 Rayforge 添加功能的扩展。它们可以提供：

- 新操作
- 导入/导出过滤器
- 机器驱动程序
- UI 增强
- 材料库

## 已安装的包

包列表显示所有已安装的包，包括：

- **名称**：包的显示名称
- **版本**：当前安装的版本
- **描述**：功能的简要描述
- **状态**：启用或禁用

## 管理包

### 安装包

1. 点击 **安装** 按钮
2. 选择包文件（`.zip` 或包存储库）
3. 确认安装
4. 等待安装完成

### 更新包

当有更新可用时：

1. 包旁边会出现更新指示器
2. 点击 **更新** 按钮安装更新
3. 如果提示，重启 Rayforge

### 移除包

1. 选择要移除的包
2. 点击 **移除** 按钮
3. 确认移除

:::warning
移除包可能会影响使用其功能的文档。
:::

## 包来源

包可以从以下位置安装：

- **本地文件**：手动下载的 `.zip` 存档
- **存储库**：包含包的 Git 存储库
- **官方注册表**：Rayforge 团队策划的包
