# Машини

![Налаштування машин](/screenshots/application-machines.png)

Сторінка «Машини» в налаштуваннях додатку показує список усіх налаштованих
машин. Кожен запис містить назву машини та кнопки для редагування або
видалення. Поточна активна машина позначена іконкою галочки.

## Додавання машини

1. Натисніть кнопку **Add Machine** у нижній частині списку
2. Виберіть профіль пристрою зі списку як шаблон — кожен профіль
   попередньо налаштовує параметри машини та діалект G-code

![Додавання машини](/screenshots/app-settings-machines-add.png)

3. Відкриється [діалог налаштувань машини](../machine/general), де ви
   зможете скоригувати конфігурацію

Крім того:

- Натисніть **Other Device…**, щоб запустити
  [Майстер конфігурації](../machine/config-wizard), який автоматично
  виявить налаштування підключеного пристрою
- Натисніть **Import from File...** у виборі профілю, щоб додати
  машину з раніше експортованого профілю

## Редагування машини

Натисніть іконку редагування поруч із машиною, щоб відкрити
[діалог налаштувань машини](../machine/general).

## Перемикання активної машини

Використовуйте випадаючий список машин у заголовку головного вікна для
перемикання між налаштованими машинами. Вибір зберігається між сесіями.

## Видалення машини

1. Натисніть іконку видалення поруч із машиною
2. Підтвердьте видалення

:::warning
Видалення машини не можна скасувати. Експортуйте профіль перед видаленням,
якщо хочете зберегти конфігурацію.
:::
