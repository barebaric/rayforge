# Керівництво з позиціонування заготовки

Це керівництво охоплює всі методи, доступні в Rayforge для точного
позиціонування вашої заготовки та вирівнювання ваших дизайнів перед різанням
або гравіюванням.

## Огляд

Точне позиціонування заготовки є важливим для:

- **Запобігання відходам**: Уникнути різання в неправильному місці
- **Точне вирівнювання**: Розміщувати дизайни на попередньо надрукованих
  матеріалах
- **Відтворювані результати**: Виконувати одне й те саме завдання багато
  разів послідовно
- **Багатодетальні завдання**: Вирівнювати кілька деталей на одному аркуші

Rayforge надає кілька взаємодоповнювальних інструментів для позиціонування:

| Метод                 | Призначення                  | Найкраще для                                 |
| --------------------- | ---------------------------- | -------------------------------------------- |
| **Режим фокусу**      | Бачити позицію лазера        | Швидке візуальне вирівнювання                |
| **Обрамлення**        | Попередній перегляд меж      | Перевірка, чи дизайн поміщається на матеріал |
| **Нуль СКР**          | Встановити початок координат | Відтворюване позиціонування                  |
| **Накладання камери** | Візуальне розміщення дизайну | Точне вирівнювання на існуючих елементах     |

## Режим фокусу (лазерний вказівник)

Режим фокусу вмикає лазер на низькому рівні потужності, діючи як "лазерний
вказівник", щоб допомогти вам точно бачити, де розміщена головка лазера.

### Увімкнення режиму фокусу

1. **Підключіться до вашої машини**
2. **Натисніть кнопку Фокус** на панелі інструментів (іконка лазера)
3. Лазер вмикається на налаштованому рівні потужності фокусу
4. **Перемістіть головку лазера** щоб бачити позицію променя на вашому
   матеріалі
5. **Натисніть кнопку Фокус знову** щоб вимкнути після завершення

:::warning Безпека
Навіть при низькій потужності лазер може пошкодити очі. Ніколи не дивіться
прямо в промінь і не спрямовуйте його на відбиваючі поверхні. Носіть
відповідний захист очей.
:::

### Налаштування потужності фокусу

Потужність фокусу визначає, наскільки яскраво erscheint лазерна точка:

1. Перейдіть до **Налаштування → Машина → Лазер**
2. Знайдіть налаштування **Потужність фокусу**
3. Встановіть значення, яке робить точку видимою без позначення вашого
   матеріалу
   - Типові значення: 1-5% для більшості матеріалів
   - Встановіть 0, щоб вимкнути функцію

:::tip Пошук правильної потужності
Почніть з 1% і поступово збільшуйте. Точка має бути видимою, але не
залишати жодної позначки на вашому матеріалі. Темніші матеріали можуть
вимагати вищої потужності, щоб чітко бачити точку.
:::

### Коли використовувати режим фокусу

- **Швидкі перевірки вирівнювання**: Перевірити, чи лазер приблизно там, де
  ви очікуєте
- **Пошук країв матеріалу**: Перемістити до кутів, щоб перевірити розміщення
  матеріалу
- **Встановлення початку СКР**: Розмістити лазер у бажаній нульовій точці
  перед встановленням СКР
- **Перевірка домашньої позиції**: Перевірити, що базування спрацювало
  правильно

## Обрамлення

Обрамлення відстежує обмежувальний прямокутник вашого завдання при низькій
(або нульовій) потужності, показуючи точно де ваш дизайн буде вирізаний або
гравійований.

### Як обрамляти

1. **Завантажте та розмістіть ваш дизайн** в Rayforge
2. **Натисніть Машина → Обрамити** або натисніть `Ctrl+F`
3. Головка лазера відстежує обмежувальну рамку вашого завдання
4. **Перевірте контур**, що він поміщається в вашому матеріалі

### Налаштування обрамлення

Налаштувати поведінку обрамлення в **Налаштування → Машина → Лазер**:

- **Швидкість обрамлення**: Наскільки швидко головка рухається під час
  обрамлення (повільніше = легше бачити)
- **Потужність обрамлення**: Потужність лазера під час обрамлення
  - Встановіть 0 для повітряного обрамлення (лазер вимкнено, тільки рух)
  - Встановіть 1-5% для видимого сліду на матеріалі

:::tip Повітряне обрамлення проти низької потужності

- **Повітряне обрамлення (0% потужність)**: Безпечно для будь-якого
  матеріалу, але ви бачите тільки рух головки
- **Обрамлення низької потужності**: Залишає слабку видиму позначку, корисно
  для точного вирівнювання на темних матеріалах
  :::

### Коли обрамляти

- **Перед кожним завданням**: Швидка перевірка, що дизайн поміщається
- **Після змін позиції**: Підтвердити, що нове розміщення правильне
- **Дорогі матеріали**: Перевірити двічі перед виконанням
- **Багатодетальні завдання**: Перевірити, що всі деталі поміщаються на
  матеріал

Дивіться [Обрамлення вашого завдання](framing-your-job) для деталей.

## Встановлення нуля СКР (Система координат робочого простору)

Системи координат робочого простору (СКР) дозволяють вам визначати
"нульові точки" для ваших завдань. Це полегшує вирівнювання завдань до
позиції вашого матеріалу.

### Швидке налаштування СКР

1. **Перемістіть головку лазера** до кута вашого матеріалу (або бажаної
   точки початку)
2. **Відкрийте Панель керування** (`Ctrl+L`)
3. **Виберіть СКР** (G54 - це система координат робочого простору за
   замовчуванням)
4. **Натисніть Нуль X і Нуль Y** щоб встановити поточну позицію як початок
5. Точка (0,0) вашого дизайну тепер буде вирівняна з цією позицією

### Розуміння систем координат

Rayforge використовує кілька систем координат:

| Система     | Опис                                                     |
| ----------- | -------------------------------------------------------- |
| **G53**     | Координати машини (фіксовані, не можуть бути змінені)    |
| **G54**     | Система координат робочого простору 1 (за замовчуванням) |
| **G55-G59** | Додаткові системи координат робочого простору            |

:::tip Кілька робочих зон
Використовуйте різні слоти СКР для різних позицій кріплення. Наприклад:

- G54 для лівої сторони вашого столу
- G55 для правої сторони
- G56 для ротаційного кріплення
  :::

### Коли встановлювати нуль СКР

- **Нове розміщення матеріалу**: Вирівняти початок до кута матеріалу
- **Робота з кріпленням**: Встановити початок до контрольної точки кріплення
- **Відтворювані завдання**: Те саме завдання, різні позиції
- **Виробничі партії**: Послідовне позиціонування через кілька деталей

Дивіться [Системи координат робочого простору](../general-info/coordinate-systems)
для повної документації.

## Позиціонування на основі камери

Накладання камери показує пряму трансляцію вашого матеріалу з вашим дизайном
накладеним зверху, що дозволяє точне візуальне вирівнювання.

### Налаштування камери

1. **Підключіть USB-камеру** над вашою робочою зоною
2. Перейдіть до **Налаштування → Камера** і додайте ваш пристрій камери
3. **Увімкніть камеру** щоб бачити накладання на вашому полотні
4. **Вирівняйте камеру** використовуючи процедуру вирівнювання (необхідно для
   точного позиціонування)

### Вирівнювання камери

Вирівнювання камери відображає пікселі камери на реальні координати:

1. Відкрийте **Камера → Вирівняти камеру**
2. Розмістіть маркери вирівнювання у відомих позиціях (принаймні 4 точки)
3. Введіть реальні координати X/Y для кожної точки
4. Натисніть **Застосувати** щоб обчислити трансформацію

:::tip Точність вирівнювання

- Використовуйте точки, розподілені по всій вашій робочій зоні
- Ретельно виміряйте світові координати лінійкою
- Використовуйте позиції машини (перемістіть до відомих координат) для
  найкращої точності
  :::

### Позиціонування з накладанням камери

1. **Увімкніть накладання камери** щоб бачити ваш матеріал
2. **Імпортуйте ваш дизайн**
3. **Перетягніть дизайн** щоб вирівняти з елементами, видимими в камері
4. **Тонке налаштування** використовуючи клавіші стрілок для піксельно
   точного розміщення
5. **Обраміть для перевірки** перед виконанням завдання

### Коли використовувати позиціонування з камерою

- **Попередньо надруковані матеріали**: Вирівнювати різання до існуючих
  відбитків
- **Нерегулярні матеріали**: Розміщувати на непрямокутних деталях
- **Точне розміщення**: Вимоги до суб-міліметрової точності
- **Складні компонування**: Кілька елементів з специфічним інтервалом

Дивіться [Інтеграція камери](../machine/camera) для повної документації.

## Рекомендовані робочі процеси

### Базовий робочий процес позиціонування

Для простих завдань на прямокутних матеріалах:

1. **Розмістіть матеріал** на лазерному столі
2. **Увімкніть режим фокусу** і перемістіть щоб перевірити позицію матеріалу
3. **Встановіть нуль СКР** на куті матеріалу
4. **Розмістіть ваш дизайн** на полотні
5. **Обраміть завдання** щоб перевірити розміщення
6. **Виконайте завдання**

### Робочий процес точного вирівнювання

Для точного розміщення на попередньо надрукованих або позначених матеріалах:

1. **Налаштуйте та вирівняйте камеру** (одноразове налаштування)
2. **Розмістіть матеріал** на лазерному столі
3. **Увімкніть накладання камери** щоб бачити матеріал
4. **Імпортуйте та розмістіть дизайн** візуально на зображенні камери
5. **Вимкніть камеру** і обраміть для перевірки
6. **Виконайте завдання**

### Виробничий робочий процес

Для виконання кількох ідентичних завдань:

1. **Налаштуйте кріплення** на лазерному столі
2. **Встановіть нуль СКР** вирівняний з кріпленням (наприклад, G54)
3. **Завантажте та налаштуйте** ваш дизайн
4. **Обраміть для перевірки** вирівнювання з кріпленням
5. **Виконайте завдання**
6. **Замініть матеріал** і повторіть (СКР залишається тим самим)

### Робочий процес кількох позицій

Для виконання того самого завдання в різних місцях:

1. **Налаштуйте кілька позицій СКР**:
   - Перемістіть до позиції 1, встановіть нуль G54
   - Перемістіть до позиції 2, встановіть нуль G55
   - Перемістіть до позиції 3, встановіть нуль G56
2. **Завантажте ваш дизайн** (той самий дизайн для всіх позицій)
3. **Виберіть G54**, обраміть і виконайте
4. **Виберіть G55**, обраміть і виконайте
5. **Виберіть G56**, обраміть і виконайте

## Вирішення проблем

### Лазерна точка не видима в режимі фокусу

- **Збільште потужність фокусу** в налаштуваннях лазера
- **Темні матеріали** можуть вимагати вищої потужності (5-10%)
- **Перевірте підключення лазера** і переконайтеся, що машина відповідає
- **Перевірте, що потужність фокусу** не встановлена на 0

### Накладання камери невирівняне

- **Повторно запустіть вирівнювання камери** з більшою кількістю контрольних
  точок
- **Перевірте кріплення камери** - воно могло зміститися
- **Перевірте, що світові координати** були виміряні точно
- **Дивіться вирішення проблем камери** в документації Інтеграції камери
