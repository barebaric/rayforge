# Tastaturkürzel

Vollständige Referenz der Tastaturkürzel in Rayforge.

:::note Plattformkonventionen
- **Linux/Windows:** `Strg`-Taste
- **macOS:** `⌘ (Befehl)`-Taste
- Dokumentation verwendet `Strg` – macOS-Benutzer ersetzen durch `⌘`
:::


---

## Dateioperationen

| Kürzel          | Aktion          | Beschreibung                      |
| --------------- | --------------- | --------------------------------- |
| `Strg+N`        | Neue Skizze     | Neue Skizze erstellen             |
| `Strg+O`        | Öffnen          | Bestehendes Projekt öffnen        |
| `Strg+S`        | Speichern       | Aktuelles Projekt speichern       |
| `Strg+Umschalt+S`| Speichern unter | Projekt als neue Datei speichern  |
| `Strg+I`        | Importieren     | Dateiimport-Dialog öffnen         |
| `Strg+E`        | Exportieren     | G-Code exportieren                |
| `Strg+Q`        | Beenden         | Rayforge beenden                  |
| `Strg+,`        | Einstellungen   | Einstellungsdialog öffnen         |

## Bearbeiten & Zwischenablage

| Kürzel               | Aktion               | Beschreibung                            |
| -------------------- | -------------------- | --------------------------------------- |
| `Strg+Z`             | Rückgängig           | Letzte Aktion rückgängig machen         |
| `Strg+Y`             | Wiederholen          | Letzte rückgängige Aktion wiederholen   |
| `Strg+Umschalt+Z`    | Wiederholen (alt.)   | Alternative Wiederholen-Verknüpfung     |
| `Strg+X`             | Ausschneiden         | Auswahl in Zwischenablage ausschneiden  |
| `Strg+C`             | Kopieren             | Auswahl in Zwischenablage kopieren      |
| `Strg+V`             | Einfügen             | Aus Zwischenablage einfügen             |
| `Strg+A`             | Alles auswählen      | Alle Elemente in aktiver Ebene auswählen|
| `Strg+D`             | Duplizieren          | Ausgewählte Elemente duplizieren        |
| `Entf`               | Entfernen            | Ausgewählte Elemente löschen            |
| `Strg+Umschalt+Entf` | Löschen              | Alle Elemente aus Dokument entfernen    |

## Schnellreferenz nach Kategorie

### Meistgenutzt (Top 10)

1. `Strg+Z` / `Strg+Y` - Rückgängig/Wiederholen
2. `Strg+C` / `Strg+V` - Kopieren/Einfügen
3. `Strg+D` - Duplizieren
4. `Entf` - Entfernen
5. `Strg+A` - Alles auswählen
6. `Strg+N` - Neue Skizze
7. `Strg+O` - Öffnen
8. `Strg+S` - Speichern
9. `H` - Werkstücke umschalten
10. `F12` - 3D-Ansicht

### Ansicht & Visualisierung

- `H` - Werkstücke ausblenden/anzeigen
- `T` - Halterungen ausblenden/anzeigen
- `Alt+C` - Kamera umschalten
- `F12` - 3D-Ansicht
- `F11` - Simulationsmodus
- `1`, `2`, `7` - 3D-Ansicht-Presets
- `P` - Perspektive umschalten

### Bearbeiten & Transformieren

- `Strg+Z` / `Strg+Y` - Rückgängig/Wiederholen
- `Strg+X` / `Strg+C` / `Strg+V` - Ausschneiden/Kopieren/Einfügen
- `Strg+D` - Duplizieren
- `Entf` - Entfernen
- `Umschalt+H` / `Umschalt+V` - Horizontal/Vertikal spiegeln
- `Strg+G` / `Strg+U` - Gruppieren/Gruppierung aufheben

### Ausrichtung

- `Umschalt+Pfeiltasten` - An Kanten ausrichten
- `Umschalt+Pos1` / `Umschalt+Ende` - Zentrieren
- `Strg+Umschalt+H` / `Strg+Umschalt+V` - Verteilen
- `Alt+A` - Pixelgenau

## Tastaturkürzel anpassen

:::note Benutzerdefinierte Kürzel
Tastaturkürzel sind derzeit fest codiert. Eine Konfiguration benutzerdefinierter Kürzel könnte in zukünftigen Versionen hinzugefügt werden.
:::


**Aktuelle Einschränkungen:**

- Kürzel können nicht ohne Quellcode-Änderung geändert werden
- Keine GUI für Kürzel-Anpassung
- Einigen Aktionen sind möglicherweise keine Kürzel zugewiesen

**Funktionswunsch:** Wenn du benutzerdefinierte Kürzel benötigen, öffnen Sie bitte ein Issue auf GitHub.

## Tipps & Tricks

### Effizienz-Tipps

1. **Lernen Sie zuerst die Bearbeitungskürzel** – `Strg+Z/Y/C/V/D` werden ständig verwendet
2. **Verwende Ein-Tasten-Umschaltungen** – `H`, `T`, `P` für schnelle Ansichtswechsel
3. **3D-Ansicht-Kürzel** – `1`, `2`, `7` für sofortigen Ansichtswechsel
4. **Ausrichtungskürzel** – `Umschalt+Pfeil` schneller als Anklicken von Ausrichtungs-Buttons
5. **Simulationsmodus** – `Strg+Umschalt+S` zur schnellen Überprüfung der Ausführung

### Workflow-Kürzel

**Schneller Bearbeitungszyklus:**

```
1. Neue Skizze (Strg+N) oder Öffnen (Strg+O)
2. Elemente anordnen (Pfeiltasten, Umschalt+Pfeil für Ausrichtung)
3. Teile duplizieren (Strg+D)
4. In 3D überprüfen (F12)
5. Simulieren (F11)
6. Speichern (Strg+S) oder Exportieren (Strg+E)
```

**Präzise Positionierung:**

```
1. Element auswählen
2. Pfeiltasten für 1mm-Verschiebungen
3. Umschalt+Pfeil für 10mm-Sprünge
4. Umschalt+Pos1/Ende zum Zentrieren
```

### Versteckte Kürzel

Einige weniger bekannte Kürzel:

- `Strg+,` - Schneller Zugriff auf Einstellungen (Standard auf macOS, funktioniert auch hier)
- `Strg+Umschalt+Z` - Alternative für Wiederholen (für Benutzer, die dies gegenüber Strg+Y bevorzugen)
- `Alt+A` - Pixelgenaues Einrasten für präzise Platzierung
- `Alt+T` - Schnelle äquidistante Halterungsplatzierung

## Plattformunterschiede

### Linux

- Verwendet Standard-`Strg`-Modifikator
- Alle Kürzel funktionieren wie dokumentiert
- GTK-Standardkürzel gelten in Textfeldern

### Windows

- Verwendet Standard-`Strg`-Modifikator
- Identisch mit Linux-Kürzeln
- Windows-spezifische Tasten (Win-Taste) werden nicht verwendet

### macOS

- `Strg` wird zu `⌘ (Befehl)`
- `Alt` wird zu `⌥ (Option)`
- Standard-macOS-Konventionen gelten
- `Cmd+Q` zum Beenden (statt Strg+Q)

## Fehlerbehebung bei Kürzeln

### Kürzel funktioniert nicht

**Häufige Probleme:**

1. **Fokus auf falschem Element** – Stelle sicher, dass Canvas oder Hauptfenster den Fokus hat, nicht ein Textfeld
2. **Kollision mit anderer Anwendung** – Eine andere App fängt das Kürzel möglicherweise ab
3. **Desktop-Umgebungs-Kürzel** – Systemkürzel könnten Vorrang haben (z.B. Alt+F4)
4. **Numpad vs. Zahlenreihe** – Verwende die Zahlenreihe für 3D-Ansicht-Kürzel, nicht das Numpad

**Lösungen:**

- Auf den Canvas klicken, um sicherzustellen, dass er den Fokus hat
- System-Tastaturkürzel auf Konflikte prüfen
- Stattdessen die Menüaktion versuchen, um die Funktionalität zu überprüfen

### Modifikatortasten

**Wenn Strg nicht funktioniert:**

- Stelle sicher, dass Feststelltaste aus ist (kann auf einigen Systemen stören)
- Strg auf beiden Tastaturseiten versuchen
- Tastaturlayout-Einstellungen prüfen

**Wenn Alt nicht funktioniert:**

- Einige Fenstermanager fangen Alt für Fensterziehen ab
- Versuche, Fenstermanager-Alt-Kürzel zu deaktivieren
- Verwende Menüpunkte als Alternative
