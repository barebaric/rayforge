# 扩展

![扩展设置](/screenshots/application-addons.png)

扩展页面显示已安装的扩展并允许您管理它们。

## 概述

扩展是为 Rayforge 添加功能的扩展。它们可以提供：

- 新操作
- 导入/导出过滤器
- 机器驱动程序
- UI 增强
- 材料库

## 已安装的扩展

扩展列表显示所有已安装的扩展，包括：

- **名称**：扩展的显示名称
- **版本**：当前安装的版本
- **描述**：功能的简要描述
- **状态**：启用或禁用

## 管理扩展

### 安装扩展

1. 点击 **安装** 按钮
2. 选择扩展文件（`.zip` 或扩展存储库）
3. 确认安装
4. 等待安装完成

### 更新扩展

当有更新可用时：

1. 扩展旁边会出现更新指示器
2. 点击 **更新** 按钮安装更新
3. 如果提示，重启 Rayforge

### 移除扩展

1. 选择要移除的扩展
2. 点击 **移除** 按钮
3. 确认移除

:::warning
移除扩展可能会影响使用其功能的文档。
:::

## 扩展来源

扩展可以从以下位置安装：

- **本地文件**：手动下载的 `.zip` 存档
- **存储库**：包含扩展的 Git 存储库
- **官方注册表**：Rayforge 团队策划的扩展
