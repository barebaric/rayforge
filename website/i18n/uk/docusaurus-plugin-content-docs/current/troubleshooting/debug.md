# Усунення неполадок та звітування про проблеми

Якщо ви зіткнулися з проблемами з Rayforge, особливо з підключенням або керуванням вашою машиною, ми тут щоб допомогти. Найкращий спосіб отримати підтримку - надати детальний звіт про debug. Rayforge має вбудований інструмент, який робить це легко.

## Як створити звіт про debug

Виконайте ці прості кроки щоб згенерувати та поділитися звітом:

#### 1. Збережіть звіт

Перейдіть до **Допомога → Зберегти debug-лог** у рядку меню. Це упакує всю необхідну діагностичну інформацію в один файл `.zip`. Збережіть цей файл у пам'ятне місце, наприклад, на Робочий стіл.

#### 2. Створіть GitHub Issue

Перейдіть на нашу [сторінку GitHub Issues](https://github.com/barebaric/rayforge/issues/new/choose) і створіть нове issue. Будь ласка, надайте чітку назву та детальний опис проблеми:

- **Що ви робили?** (наприклад, "Я спробував підключитися до мого лазера після запуску програми.")
- **Що ви очікували, що станеться?** (наприклад, "Я очікував успішного підключення.")
- **Що насправді сталося?** (наприклад, "Він залишився відключеним, а лог показав помилки тайм-ауту.")

#### 3. Прикріпіть звіт

**Перетягніть файл `.zip`**, який ви зберегли, у поле опису GitHub issue. Це завантажить і прикріпить його до вашого звіту.

## Що міститься у звіті про debug?

Згенерований файл `.zip` містить технічну інформацію, яка допомагає нам швидко діагностувати проблему. Він включає:

- **Налаштування машини та додатку:** Ваші збережені конфігурації машини та налаштування додатку, що допомагає нам відтворити ваше налаштування.
- **Логи комунікації:** Детальний запис даних, надісланих між Rayforge та вашим лазером.
- **Системна інформація:** Ваша операційна система та версії Rayforge і ключових встановлених бібліотек.
- **Стан додатку:** Інша внутрішня інформація, яка може допомогти виявити джерело помилки.

> **Примітка про приватність:** Звіт **не** включає жодних ваших файлів дизайну (SVG, DXF тощо.) або особисті дані операційної системи. Він містить лише інформацію, безпосередньо пов'язану з додатком Rayforge та його з'єднанням з вашим лазером.

Дякуємо, що допомагаєте нам покращувати Rayforge
