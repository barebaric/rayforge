# Рамковий контур

Рамковий контур створює простий прямокутний шлях різання навколо всього вашого дизайну. Це найшвидший спосіб додати чисту рамку або вирізати вашу роботу з листа матеріалу.

## Огляд

Операції рамкового контуру:

- Створюють прямокутну межу навколо всього вмісту
- Додають налаштовуване зміщення/відступ від дизайну
- Підтримують компенсацію керфа для точного розміру
- Працюють з будь-якою комбінацією об'єктів на полотні

![Налаштування кроку рамкового контуру](/screenshots/step-settings-frame-outline-general.png)

## Коли використовувати рамковий контур

Використовуйте рамковий контур для:

- Додавання декоративної рамки навколо вашого дизайну
- Вирізання вашої роботи з листа матеріалу
- Створення простої прямокутної межі
- Швидкого обведення без складних розрахунків шляху

**Не використовуйте рамковий контур для:**

- Неправильних форм навколо кількох об'єктів (використовуйте [Обтікання](shrink-wrap))
- Різання окремих деталей (використовуйте [Контур](contour))
- Слідування точною формі вашого дизайну

## Створення операції рамкового контуру

### Крок 1: Розташуйте ваш дизайн

1. Розмістіть всі об'єкти на полотні
2. Позиціонуйте їх там, де ви хочете відносно рамки
3. Рамка буде розрахована навколо обмежувального прямокутника всього вмісту

### Крок 2: Додайте операцію рамкового контуру

- **Меню:** Операції → Додати рамковий контур
- **Права кнопка:** Контекстне меню → Додати операцію → Рамковий контур

### Крок 3: Налаштуйте параметри

Налаштуйте параметри рамки:

- **Потужність та швидкість:** Відповідайте вимогам різання вашого матеріалу
- **Зміщення:** Відстань від краю вмісту до рамки
- **Зміщення шляху:** Різання всередині, зовні або по центральній лінії

## Основні налаштування

### Потужність та швидкість

**Потужність (%):**

- Інтенсивність лазера для різання рамки
- Відповідайте вимогам різання вашого матеріалу

**Швидкість (мм/хв):**

- Як швидко рухається лазер
- Повільніше для товстих матеріалів

**Проходи:**

- Кількість разів для різання рамки
- Зазвичай 1-2 проходи
- Додайте проходи для товстих матеріалів

### Відстань зміщення

**Зміщення (мм):**

- Відстань від обмежувального прямокутника дизайну до рамки
- Створює відступ/рамку навколо вашої роботи

**Типові значення:**

- **0мм:** Рамка торкається краю дизайну
- **2-5мм:** Малий відступ для чистого вигляду
- **10мм+:** Велика рамка для монтажу або обробки

### Зміщення шляху (бік різу)

Контролює, де лазер ріже відносно шляху рамки:

| Бік різу         | Опис                       | Використовувати для                 |
| ---------------- | -------------------------- | ----------------------------------- |
| **Центральна лінія** | Ріже безпосередньо на шляху | Стандартне різання               |
| **Ззовні**       | Ріже зовні шляху рамки     | Зробити рамку трохи більшою        |
| **Всередині**    | Ріже всередині шляху рамки | Зробити рамку трохи меншою         |

### Компенсація керфа

Рамковий контур підтримує компенсацію керфа:

- Автоматично коригує за ширину лазерного променя
- Забезпечує точні кінцеві розміри
- Використовує значення керфа з налаштувань вашої лазерної головки

## Параметри постобробки

![Налаштування постобробки рамкового контуру](/screenshots/step-settings-frame-outline-post.png)

### Багатопрохідність

Ріжте рамку кілька разів:

- **Проходи:** Кількість повторень
- **Z-крок:** Опускайте Z між проходами (вимагає Z-осі)
- Корисно для товстих матеріалів

### Утримуючі лапки

Додайте лапки, щоб утримувати обрамлену деталь прикріпленою:

- Запобігає падінню деталей під час різання
- Налаштуйте ширину, висоту та інтервал лапок
- Дивіться [Утримуючі лапки](../holding-tabs) для деталей

## Випадки використання

### Декоративна рамка

**Сценарій:** Додайте чисту прямокутну рамку навколо дошки або знаку

**Процес:**

1. Розробіть ваш вміст (текст, логотипи тощо)
2. Додайте рамковий контур зі зміщенням 3-5мм
3. Ріжте з налаштуваннями декоративного надрізування (низька потужність)

**Результат:** Професійно виглядаюча обрамлена деталь

### Вирізання з листа

**Сценарій:** Видаліть вашу завершену роботу з листа матеріалу

**Процес:**

1. Завершіть всі інші операції (гравіювання, контурні різи)
2. Додайте рамковий контур як останню операцію
3. Встановіть зміщення, щоб включити малий відступ

**Переваги:**

- Чисте відділення від листа
- Послідовна якість краю
- Легко виконати як фінальний крок

### Межа пакетної обробки

**Сценарій:** Створіть межу різання для кількох вкладених деталей

**Процес:**

1. Розташуйте всі деталі на полотні
2. Додайте індивідуальні операції контуру для деталей
3. Додайте рамковий контур навколо всього
4. Рамка ріже останньою (в окремому шарі)

**Порядок:** Гравіювання → Контури деталей → Рамковий контур

## Поради та найкращі практики

### Порядок шарів

**Найкраща практика:**

- Розмістіть рамковий контур у власному шарі
- Виконуйте рамку як **останню** операцію
- Це гарантує, що вся інша робота завершиться першою

**Чому останньою?**

- Матеріал залишається закріпленим під час інших операцій
- Запобігає зміщенню деталей
- Чистіший кінцевий результат

### Вибір зміщення

**Вибір зміщення:**

- **0-2мм:** Щільне прилягання, мінімальні відходи матеріалу
- **3-5мм:** Стандартний відступ, виглядає професійно
- **10мм+:** Додатковий матеріал для обробки/монтажу

**Враховуйте:**

- Кінцеве використання деталі
- Чи будуть краї видимі
- Вартість та наявність матеріалу

### Налаштування якості

**Для чистих різів рамки:**

- Використовуйте повітряний ассистент
- Забезпечте правильний фокус
- Кілька швидших проходів часто краще за один повільний
- Тримайте матеріал пласким і закріпленим

## Поєднання з іншими операціями

### Рамка + Гравіювання + Контур

Типовий робочий процес для завершеної деталі:

1. **Шар 1:** Гравіювання деталей (текст, зображення)
2. **Шар 2:** Контурний різ окремих деталей
3. **Шар 3:** Рамковий контур (відрізання)

**Порядок виконання гарантує:**

- Гравіювання відбувається, поки матеріал плаский і закріплений
- Деталі деталей вирізаються перед фінальним відділенням
- Рамка відрізає все в кінці

### Рамка проти Обтікання

| Функція          | Рамковий контур              | Обтікання                 |
| ---------------- | ---------------------------- | ------------------------- |
| **Форма**        | Завжди прямокутна            | Слідує контурам об'єктів  |
| **Швидкість**    | Дуже швидка (4 лінії)        | Залежить від складності   |
| **Випадок використання** | Прості рамки, вирізання | Ефективне використання матеріалу |
| **Гнучкість**    | Фіксований прямокутник       | Адаптується до дизайну    |

**Виберіть рамковий контур коли:**

- Ви хочете прямокутну рамку
- Перевага віддається простоті
- Вирізання з листа

**Виберіть обтікання коли:**

- Ви хочете мінімізувати відходи матеріалу
- Дизайн має неправильну форму
- Ефективність важлива

## Виправлення проблем

### Рамка занадто щільна/вільна

- **Скоригуйте:** Налаштування відстані зміщення
- **Перевірте:** Зміщення шляху (всередину/зовні/центральна лінія)
- **Підтвердіть:** Компенсація керфа правильна

### Рамка не з'являється

- **Перевірте:** Об'єкти на полотні
- **Підтвердіть:** Операція увімкнена
- **Подивіться:** Рамка може бути поза видимою областю (зменште масштаб)

### Рамка врізається в дизайн

- **Збільште:** Відстань зміщення
- **Перевірте:** Об'єкти правильно позиціоновані
- **Підтвердіть:** Розрахунок обмежувального прямокутника включає всі об'єкти

### Нестабільна глибина різу

- **Перевірте:** Матеріал плаский
- **Підтвердіть:** Відстань фокусування правильна
- **Спробуйте:** Кілька проходів на нижчій потужності

## Технічні деталі

### Розрахунок обмежувального прямокутника

Рамковий контур використовує комбінований обмежувальний прямокутник:

- Всі заготовки на полотні
- Їхні кінцеві трансформовані позиції
- Включаючи будь-які застосовані обертання/масштабування

### Генерація шляху

1. Розрахувати комбінований обмежувальний прямокутник
2. Застосувати відстань зміщення
3. Застосувати зміщення шляху (всередину/зовні/центральна лінія)
4. Застосувати компенсацію керфа
5. Згенерувати прямокутний шлях G-коду

### Приклад G-коду

```gcode
G0 X5 Y5           ; Переміщення до початку рамки (зі зміщенням)
M3 S200            ; Лазер увімкнено на 80% потужності
G1 X95 Y5 F500     ; Різ нижнього краю
G1 X95 Y95         ; Різ правого краю
G1 X5 Y95          ; Різ верхнього краю
G1 X5 Y5           ; Різ лівого краю (завершення)
M5                 ; Лазер вимкнено
```
