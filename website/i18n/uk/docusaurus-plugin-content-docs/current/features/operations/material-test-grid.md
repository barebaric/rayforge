# Тестова сітка матеріалу

Генератор тестової сітки матеріалу створює параметричні тестові патерни, щоб допомогти вам знайти оптимальні налаштування лазера для різних матеріалів.

## Огляд

Тестування матеріалу є необхідним для лазерної роботи — різні матеріали вимагають різних налаштувань потужності та швидкості. Тестова сітка матеріалу автоматизує цей процес шляхом:

- Генерації тестових сіток з налаштовуваними діапазонами швидкості/потужності
- Надання пресетів для поширених типів лазерів (Діодний, CO2)
- Оптимізації порядку виконання для безпеки (найшвидші швидкості першими)
- Додавання міток для ідентифікації налаштувань кожної тестової комірки

## Створення тестової сітки матеріалу

### Крок 1: Відкрийте генератор

Доступ до генератора тестової сітки матеріалу:

- Меню: **Інструменти → Тестова сітка матеріалу**
- Це створює спеціальну заготовку, яка генерує тестовий патерн

### Крок 2: Виберіть пресет (опціонально)

Rayforge включає пресети для поширених сценаріїв:

| Пресет               | Діапазон швидкості  | Діапазон потужності | Використовувати для        |
| -------------------- | ------------------- | ------------------- | -------------------------- |
| **Діодне гравіювання** | 1000-10000 мм/хв    | 10-100%             | Гравіювання діодним лазером |
| **Діодне різання**   | 100-5000 мм/хв      | 50-100%             | Різання діодним лазером    |
| **CO2 гравіювання**  | 3000-20000 мм/хв    | 10-50%              | Гравіювання CO2 лазером    |
| **CO2 різання**      | 1000-20000 мм/хв    | 30-100%             | Різання CO2 лазером        |

Пресети — це відправні точки — ви можете коригувати всі параметри після вибору одного.

### Крок 3: Налаштуйте параметри

Скоригуйте параметри тестової сітки в діалозі налаштувань:

![Налаштування тестової сітки матеріалу](/screenshots/material-test-grid.png)

#### Тип тесту

- **Гравіювання**: Заповнює квадрати растровим патерном
- **Різання**: Ріже контур квадратів

#### Діапазон швидкості

- **Мін. швидкість**: Найповільніша швидкість для тестування (мм/хв)
- **Макс. швидкість**: Найшвидша швидкість для тестування (мм/хв)
- Стовпці в сітці представляють різні швидкості

#### Діапазон потужності

- **Мін. потужність**: Найнижча потужність для тестування (%)
- **Макс. потужність**: Найвища потужність для тестування (%)
- Рядки в сітці представляють різні рівні потужності

#### Розміри сітки

- **Стовпці**: Кількість варіацій швидкості (зазвичай 3-7)
- **Рядки**: Кількість варіацій потужності (зазвичай 3-7)

#### Розмір та інтервал

- **Розмір форми**: Розмір кожного тестового квадрата в мм (за замовчуванням: 20мм)
- **Інтервал**: Проміжок між квадратами в мм (за замовчуванням: 5мм)

#### Мітки

- **Включити мітки**: Увімкнути/вимкнути мітки осей, що показують значення швидкості та потужності
- Мітки з'являються на лівому та верхньому краях
- Мітки гравіюються при 10% потужності, 1000 мм/хв

### Крок 4: Згенеруйте сітку

Натисніть **Згенерувати**, щоб створити тестовий патерн. Сітка з'явиться на вашому полотні як спеціальна заготовка.

## Розуміння макета сітки

### Організація сітки

```
Потужність (%)   Швидкість (мм/хв) →
    ↓        1000   2500   5000   7500   10000
  100%     [  ]   [  ]   [  ]   [  ]   [  ]
   75%     [  ]   [  ]   [  ]   [  ]   [  ]
   50%     [  ]   [  ]   [  ]   [  ]   [  ]
   25%     [  ]   [  ]   [  ]   [  ]   [  ]
   10%     [  ]   [  ]   [  ]   [  ]   [  ]
```

- **Стовпці**: Швидкість збільшується зліва направо
- **Рядки**: Потужність збільшується знизу вгору
- **Мітки**: Показують точні значення для кожного рядка/стовпця

### Розрахунок розміру сітки

**Без міток:**

- Ширина = стовпці × (розмір_форми + інтервал) - інтервал
- Висота = рядки × (розмір_форми + інтервал) - інтервал

**З мітками:**

- Додайте 15мм відступ зліва та зверху для простору міток

**Приклад:** Сітка 5×5 з квадратами 20мм та інтервалом 5мм:

- Без міток: 120мм × 120мм
- З мітками: 135мм × 135мм

## Порядок виконання (оптимізація ризику)

Rayforge виконує тестові комірки в **оптимізованому за ризиком порядку**, щоб запобігти пошкодженню матеріалу:

1. **Найвища швидкість першою**: Швидкі швидкості безпечніші (менше накопичення тепла)
2. **Найнижча потужність у межах швидкості**: Мінімізує ризик на кожному рівні швидкості

Це запобігає підгорянню або пожежі від початку з повільними, високопотужними комбінаціями.

**Приклад порядку виконання для сітки 3×3:**

```
Порядок:  1  2  3
          4  5  6  ← Найвища швидкість, потужність збільшується
          7  8  9

(Найшвидша швидкість/найнижча потужність виконується першою)
```

## Використання результатів тестування матеріалу

### Крок 1: Запустіть тест

1. Завантажте ваш матеріал у лазер
2. Сфокусуйте лазер правильно
3. Запустіть роботу тестової сітки матеріалу
4. Моніторте тест — зупиніть, якщо будь-яка комірка викликає проблеми

### Крок 2: Оцініть результати

Після завершення тесту огляньте кожну комірку:

- **Занадто світло**: Збільште потужність або зменште швидкість
- **Занадто темно/обгоріло**: Зменште потужність або збільште швидкість
- **Ідеально**: Запишіть комбінацію швидкість/потужність

### Крок 3: Запишіть налаштування

Документуйте ваші успішні налаштування для подальшого використання:

- Тип та товщина матеріалу
- Тип операції (гравіювання або різання)
- Комбінація швидкості та потужності
- Кількість проходів
- Будь-які спеціальні примітки

:::tip База даних матеріалів
Розгляньте можливість створення довідкового документа з результатами тестування матеріалів для швидкого пошуку в майбутніх проектах.
:::

## Розширене використання

### Поєднання з іншими операціями

Тестові сітки матеріалу — це звичайні заготовки — ви можете поєднувати їх з іншими операціями:

**Приклад робочого процесу:**

1. Створіть тестову сітку матеріалу
2. Додайте контурний різ навколо всієї сітки
3. Запустіть тест, відріжте, оцініть результати

Це корисно для відрізання тестового шматка від матеріалу заготовки.

### Власні діапазони тестування

Для точного налаштування створіть вузькодіапазонні тести:

**Грубий тест** (знайти діапазон):

- Швидкість: 1000-10000 мм/хв (5 стовпців)
- Потужність: 10-100% (5 рядків)

**Точний тест** (оптимізувати):

- Швидкість: 4000-6000 мм/хв (5 стовпців)
- Потужність: 35-45% (5 рядків)

### Різні матеріали, та сама сітка

Запустіть ту саму конфігурацію сітки на різних матеріалах, щоб створити вашу бібліотеку матеріалів швидше.

## Поради та найкращі практики

### Дизайн сітки

✅ **Почніть з пресетів** - Гарні відправні точки для поширених сценаріїв
✅ **Використовуйте сітки 5×5** - Гарний баланс деталізації та часу тестування
✅ **Увімкніть мітки** - Необхідні для ідентифікації результатів
✅ **Тримайте квадрати ≥20мм** - Легше бачити та вимірювати результати

### Стратегія тестування

✅ **Тестуйте відходи спочатку** - Ніколи не тестуйте на фінальному матеріалі
✅ **Одна змінна за раз** - Тестуйте діапазон швидкості АБО потужності, не обидва екстреми
✅ **Дозвольте охолонути** - Чекайте між тестами на тому самому матеріалі
✅ **Послідовний фокус** - Та сама відстань фокусу для всіх тестів

### Безпека

⚠️ **Моніторте тести** - Ніколи не залишайте запущені тести без нагляду
⚠️ **Починайте консервативно** - Починайте з нижчих діапазонів потужності
⚠️ **Перевірте вентиляцію** - Забезпечте належне видалення парів
⚠️ **Нагляд за вогнем** - Майте вогнегасник наготові

## Виправлення проблем

### Тестові комірки виконуються в неправильному порядку

- Rayforge використовує оптимізований за ризиком порядок (найшвидші швидкості першими)
- Це навмисно і не може бути змінено
- Дивіться [Порядок виконання](#порядок-виконання-оптимізація-ризику) вище

### Результати непослідовні

- **Перевірте**: Матеріал плаский і правильно закріплений
- **Перевірте**: Фокус послідовний по всій області тестування
- **Перевірте**: Потужність лазера стабільна (перевірте джерело живлення)
- **Спробуйте**: Меншу сітку, щоб зменшити область тестування
