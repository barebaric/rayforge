# 合并线条

当您导入包含重叠路径的设计时，激光可能会多次切割同一条线。这不仅
浪费时间，还可能导致过度烧焦，并使切缝超出预期宽度。

**合并线条**后处理器可检测重叠和重合的路径段，并将其合并为单次
切割。激光仅沿每条唯一的线切割一次。

## 适用场景

以下情况最为常见：

- 导入的 SVG 或 DXF 中图形共享边缘（例如网格图案或镶嵌图案）
- 合并多个轮廓重叠的工件
- 设计软件导出了重复路径

## 不适用场景

如果重叠切割是有意为之——例如对同一条线进行多次切割以切透较厚的
材料——请保持"合并线条"禁用。在这种情况下，您可能需要改用
[多次切割](multi-pass)功能，它可以让您明确控制切割次数。
