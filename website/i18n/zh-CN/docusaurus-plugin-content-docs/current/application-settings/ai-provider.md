# AI 提供商

![AI 提供商设置](/screenshots/application-ai.png)

配置插件可用于为 Rayforge 添加智能功能的 AI 提供商。

## 工作原理

插件可以使用已配置的 AI 提供商，而无需自己的 API 密钥。这集中了
您的 AI 配置，让您可以控制哪些提供商对插件可用。

## 添加提供商

1. 点击 **添加提供商** 创建新的提供商配置
2. 输入 **名称** 以标识此提供商
3. 将 **基础 URL** 设置为您的 AI 服务的 API 端点
4. 输入您的 **API 密钥** 进行身份验证
5. 指定要与此提供商一起使用的 **默认模型**
6. 点击 **测试** 验证您的配置是否有效

## 提供商类型

### OpenAI 兼容

此提供商类型适用于任何使用 OpenAI API 格式的服务。这包括各种
云提供商和自托管解决方案。

默认基础 URL 设置为 OpenAI 的 API，但您可以将其更改为任何
兼容的服务。

## 管理提供商

- **启用/禁用**：在不删除提供商的情况下启用或禁用它
- **设为默认**：点击复选图标将提供商设为默认
- **删除**：删除您不再需要的提供商

:::warning
您的 API 密钥本地存储在您的计算机上，永远不会与第三方共享。
:::
