# 主窗口

Rayforge 主窗口是您创建和管理激光作业的主要工作区。

## 窗口布局

![主窗口](/screenshots/main-standard.png)

### 1. 菜单栏

通过组织菜单访问所有 Rayforge 功能：

- **文件**：打开、保存、导入、导出和最近文件
- **编辑**：撤销、重做、复制、粘贴、首选项
- **视图**：缩放、网格、标尺、面板和视图模式
- **操作**：添加、编辑和管理操作
- **机器**：连接、点动、归零、开始/停止作业
- **帮助**：文档、关于和支持

### 2. 工具栏

快速访问常用工具：

- **选择工具**：选择和移动对象
- **平移工具**：导航画布
- **缩放工具**：放大/缩小特定区域
- **测量工具**：测量距离和角度
- **对齐工具**：对齐和分布对象
- **WCS 下拉菜单**：选择活动工作坐标系（G53-G59）

WCS 下拉菜单允许您在坐标系之间快速切换。有关更多信息，请参阅[工作坐标系](../general-info/coordinate-systems)。

### 3. 画布

主要工作区，您在这里：

- 导入和排列设计
- 预览刀具路径
- 相对于机器原点定位对象
- 测试框架边界

**画布控制：**

- **平移**：中键拖动或 <kbd>space</kbd> + 拖动
- **缩放**：鼠标滚轮或 <kbd>ctrl+"+"</kbd> / <kbd>ctrl+"-"</kbd>
- **重置视图**：<kbd>ctrl+0</kbd> 或视图 → 重置缩放

### 4. 图层面板

管理操作和图层分配：

- 查看项目中的所有操作
- 为设计元素分配操作
- 重新排序操作执行
- 启用/禁用单个操作
- 配置操作参数

### 5. 属性面板

配置选定对象或操作的设置：

- 操作类型（轮廓、光栅等）
- 功率和速度设置
- 通过次数
- 高级选项（过扫描、缝隙、标签）

### 6. 控制面板

窗口底部的控制面板提供：

- **点动控制**：手动机器移动和定位
- **机器状态**：实时位置和连接状态
- **日志视图**：G 代码通信和操作历史
- **WCS 管理**：工作坐标系选择和归零

有关详细信息，请参阅[控制面板](control-panel)。

## 窗口管理

### 面板

根据需要显示/隐藏面板：

- **图层面板**：视图 → 图层面板（<kbd>ctrl+l</kbd>）
- **属性面板**：视图 → 属性面板（<kbd>ctrl+i</kbd>）

### 全屏模式

全屏专注于您的工作：

- 进入：<kbd>f11</kbd> 或视图 → 全屏
- 退出：<kbd>f11</kbd> 或 <kbd>esc</kbd>

## 自定义

在 **编辑 → 首选项** 中自定义界面：

- **主题**：浅色、深色或系统
- **单位**：毫米或英寸
- **网格**：显示/隐藏和配置网格间距
- **标尺**：显示/隐藏画布上的标尺
- **工具栏**：自定义可见按钮
