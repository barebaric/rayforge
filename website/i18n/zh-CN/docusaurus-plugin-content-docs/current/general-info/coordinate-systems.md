# 坐标系统

理解 Rayforge 如何处理坐标系统对于正确定位您的工作至关重要。

## 工作坐标系（WCS）与机器坐标

Rayforge 使用两个主要坐标系统：

### 工作坐标系（WCS）

WCS 是您作业的坐标系统。当您在画布上将设计定位在 (50, 100) 时，这些就是 WCS 坐标。

- **原点**：由您定义（默认为 G54）
- **目的**：设计和作业定位
- **多个系统**：G54-G59 可用于不同的设置

### 机器坐标

机器坐标是相对于机器原点位置的绝对位置。

- **原点**：机器原点 (0,0,0) - 由硬件固定
- **目的**：床上的物理定位
- **固定**：无法通过软件更改

**关系**：WCS 偏移定义您的作业坐标如何映射到机器坐标。如果 G54 偏移为 (100, 50, 0)，那么您在 WCS (0, 0) 的设计将在机器位置 (100, 50) 切割。

## 在 Rayforge 中配置坐标

### 设置 WCS 原点

要在机器上定位您的作业：

1. 首先将机器**归零**（`$H` 命令或归零按钮）
2. 将激光头**点动**到所需的作业原点
3. 使用控制面板**设置 WCS 零点**：
   - 点击"零 X"将当前 X 设置为原点
   - 点击"零 Y"将当前 Y 设置为原点
4. 您的作业现在将从此位置开始

### 选择 WCS

Rayforge 支持 G54-G59 工作坐标系：

| 系统 | 用例             |
| ---- | ---------------- |
| G54  | 默认，主要工作区域 |
| G55-G59 | 附加夹具位置   |

在控制面板中选择活动的 WCS。每个系统存储其自己的距机器原点的偏移。

### Y 轴方向

某些机器的 Y 向下增加而不是向上。在以下位置配置：

**设置 → 机器 → 硬件 → 轴**

如果您的作业垂直镜像，请切换 Y 轴方向设置。

## 常见问题

### 作业位置错误

- **检查 WCS 偏移**：发送 `G10 L20 P1` 查看 G54 偏移
- **验证归零**：机器必须归零以进行一致的定位
- **检查 Y 轴方向**：可能反转

### 作业之间坐标漂移

- **作业前始终归零**：建立一致的参考
- **检查 G92 偏移**：使用 `G92.1` 命令清除
