# Матеріали

![Налаштування матеріалів](/screenshots/application-materials.png)

Бібліотеки матеріалів у Rayforge дозволяють організовувати та керувати колекціями
матеріалів для ваших проектів лазерного різання та гравірування. Цей посібник
пояснює різницю між основними та користувацькими бібліотеками, а також як створити
власні бібліотеки та додати до них матеріали.

:::примітка
  Бібліотеки матеріалів наразі не мають функціонального використання - призначення матеріалу
  впливає лише на естетичний вигляд. У майбутніх версіях матеріали
  будуть використовуватися для отримання функціональних параметрів.
  :::


## Створення нової бібліотеки

Для створення власної бібліотеки матеріалів:

1. Відкрийте меню **Налаштування** та виберіть **Матеріали**
2. Натисніть кнопку **Додати нову бібліотеку** для створення нової бібліотеки
3. Введіть описову назву для вашої бібліотеки (наприклад, "Матеріали моєї майстерні")
4. Натисніть **Створити** для завершення

Ваша нова бібліотека буде створена в каталозі даних користувача і буде доступна негайно.


## Додавання матеріалів до бібліотек

### Створення нового матеріалу

1. Виберіть бібліотеку, куди хочете додати матеріал
2. Натисніть кнопку **Додати новий матеріал** у списку матеріалів
3. Заповніть властивості матеріалу:
    - **Назва**: Зрозуміла назва
    - **Категорія**: Категорія групування (наприклад, "Дерево", "Акрил")
    - **Вигляд**: Візуальні властивості (колір)
4. Натисніть **Зберегти** для додавання матеріалу до бібліотеки

### Пояснення властивостей матеріалу

#### Назва
- Зрозуміла назва, що відображається в інтерфейсі
- Може містити пробіли та спеціальні символи

#### Категорія
- Використовується для організації матеріалів у бібліотеці
- Типові категорії включають: Дерево, Акрил, Метал, Папір, Шкіра
- Ви можете створювати власні категорії за потреби

#### Колір

Колір використовується лише для візуального вигляду на робочій поверхні - він не
впливає на шлях лазера жодним чином.


## Керування існуючими матеріалами

### Редагування матеріалів

1. Виберіть матеріал, який хочете відредагувати
2. Натисніть кнопку **Редагувати**
3. Змініть бажані властивості
4. Натисніть **Зберегти** для застосування змін

### Видалення матеріалів

1. Виберіть матеріал, який хочете видалити
2. Натисніть кнопку **Видалити**
3. Підтвердьте видалення в діалозі

:::увага
Видалення матеріалу є остаточним і не може бути скасовано.
:::
