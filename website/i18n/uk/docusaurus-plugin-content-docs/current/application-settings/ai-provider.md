# Постачальник ШІ

![Налаштування Постачальника ШІ](/screenshots/application-ai.png)

Налаштуйте постачальників штучного інтелекту, які аддони можуть
використовувати для додавання інтелектуальних функцій до Rayforge.

## Як це працює

Аддони можуть використовувати налаштованих постачальників ШІ без потреби
у власних ключах API. Це централізує вашу конфігурацію ШІ та дозволяє
контролювати, які постачальники доступні для аддонів.

## Додати постачальника

1. Натисніть **Додати постачальника**, щоб створити нову конфігурацію
2. Введіть **Назву** для ідентифікації цього постачальника
3. Встановіть **Базову URL-адресу** на кінцеву точку API вашого сервісу ШІ
4. Введіть ваш **Ключ API** для автентифікації
5. Вкажіть **Модель за замовчуванням** для використання з цим постачальником
6. Натисніть **Тестувати**, щоб перевірити, чи працює ваша конфігурація

## Типи постачальників

### Сумісний з OpenAI

Цей тип постачальника працює з будь-яким сервісом, що використовує формат
API OpenAI. Це включає різні хмарні постачальники та власні рішення.

Базова URL-адреса за замовчуванням встановлена на API OpenAI, але ви
можете змінити її на будь-який сумісний сервіс.

## Керування постачальниками

- **Увімкнути/Вимкнути**: Увімкніть або вимкніть постачальника без його
  видалення
- **Встановити за замовчуванням**: Натисніть значок галочки, щоб зробити
  постачальника типовим
- **Видалити**: Видаліть постачальника, який вам більше не потрібен

:::warning
Ваші ключі API зберігаються локально на вашому комп'ютері та ніколи не
передаються третім сторонам.
:::
