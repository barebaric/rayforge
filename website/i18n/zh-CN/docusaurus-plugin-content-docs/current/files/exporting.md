# 从 Rayforge 导出

Rayforge 支持多种导出选项用于不同目的：

- **G 代码** - 用于运行作业的机器控制输出
- **对象导出** - 将单个工件导出为矢量格式
- **文档导出** - 将所有工件导出为单个文件

---

## 导出对象

您可以将任何工件导出为矢量格式，用于设计软件、CAD 应用程序或存档。

### 如何导出

1. 在画布上**选择工件**
2. **选择对象 → 导出对象...**（或右键点击 → 导出对象...）
3. **选择格式**和保存位置

### 可用格式

| 格式    | 扩展名 | 描述                                                   |
| ------- | ------ | ------------------------------------------------------ |
| **RFS** | `.rfs` | Rayforge 的原生参数化草图格式。保留所有约束，可以重新导入编辑。|
| **SVG** | `.svg` | 可缩放矢量图形。与 Inkscape、Illustrator 和网络浏览器等设计软件广泛兼容。|
| **DXF** | `.dxf` | 图形交换格式。与 AutoCAD、FreeCAD 和 LibreCAD 等大多数 CAD 应用程序兼容。|

### 导出说明

- **SVG 和 DXF** 导出解析的几何（不是参数约束）
- 导出使用**毫米单位**
- 几何缩放到实际尺寸（世界空间）
- 多个子路径（不连续的形状）保留为单独的元素

### 用例

**共享设计：**

- 导出为 SVG 以与 Inkscape 用户共享
- 导出为 DXF 以与 CAD 软件用户共享

**进一步编辑：**

- 导出到 SVG/DXF，在外部软件中编辑，重新导入

**存档：**

- 使用 RFS 用于基于草图的设计以保留可编辑性
- 使用 SVG/DXF 用于长期存储或非 Rayforge 用户

## 导出文档

您可以将文档中的所有工件导出为单个矢量文件。这对于共享完整项目或以标准格式创建备份很有用。

### 如何导出

1. **选择文件 → 导出文档...**
2. **选择格式**（SVG 或 DXF）
3. **选择保存位置**

### 可用格式

| 格式    | 扩展名 | 描述                                                   |
| ------- | ------ | ------------------------------------------------------ |
| **SVG** | `.svg` | 可缩放矢量图形。与 Inkscape、Illustrator 和网络浏览器等设计软件广泛兼容。|
| **DXF** | `.dxf` | 图形交换格式。与 AutoCAD、FreeCAD 和 LibreCAD 等大多数 CAD 应用程序兼容。|

### 导出说明

- 所有图层的所有工件合并为单个文件
- 工件位置保留
- 空工件跳过
- 边界框包含所有几何

### 用例

- **项目共享**：导出整个项目以进行协作
- **备份**：创建工作的可视存档
- **进一步编辑**：将整个设计带到 Inkscape 或 CAD 软件中

## 导出 G 代码

生成的 G 代码包含将发送到机器的所有内容。
确切的格式、命令、数字精度等取决于当前选择的机器及其 G 代码方言的设置。

---

### 导出方法

### 方法 1：文件菜单

**文件 → 导出 G 代码**（Ctrl+E）

- 打开文件保存对话框
- 选择位置和文件名
- 生成并保存 G 代码

### 方法 2：命令行

```bash
# 从命令行导出（如果支持）
rayforge --export output.gcode input.svg
```

---

### G 代码输出

生成的 G 代码包含将发送到机器的所有内容。
确切的格式、命令、数字精度等取决于当前选择的机器及其 G 代码方言的设置。
