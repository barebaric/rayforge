# Рецепти та налаштування

![Налаштування рецептів](/screenshots/application-recipes.png)

Rayforge надає потужну систему рецептів, яка дозволяє створювати,
керувати та застосовувати узгоджені налаштування у ваших проектах лазерного різання.
Цей посібник охоплює повний шлях користувача від створення рецептів у
загальних налаштуваннях до їх застосування до операцій та керування налаштуваннями на
рівні кроку.

## Огляд

Система рецептів складається з трьох основних компонентів:

1. **Керування рецептами**: Створення та керування багаторазовими пресетами налаштувань
2. **Керування заготовками**: Визначення властивостей матеріалу та товщини
3. **Налаштування кроків**: Застосування та точне налаштування параметрів для окремих операцій

## Керування рецептами

### Створення рецептів

Рецепти - це іменовані пресети, що містять всі налаштування, необхідні для конкретних операцій.
Ви можете створювати рецепти через інтерфейс основних налаштувань:

#### 1. Доступ до менеджера рецептів

Меню: Редагування → Налаштування → Рецепти

#### 2. Створення нового рецепту

Натисніть "Додати новий рецепт" щоб відкрити діалог редактора рецептів.

**Вкладка "Загальні"** - Встановіть назву та опис рецепту:

![Редактор рецептів - Вкладка "Загальні"](/screenshots/recipe-editor-general.png)

Заповніть базову інформацію:

- **Назва**: Описова назва (наприклад, "Різання фанери 3мм")
- **Опис**: Опціональний детальний опис

#### 3. Визначення критеріїв застосовності

**Вкладка "Застосовність"** - Визначте, коли цей рецепт має пропонуватися:

![Редактор рецептів - Вкладка "Застосовність"](/screenshots/recipe-editor-applicability.png)

- **Тип завдання**: Виберіть тип операції (Різання, Гравірування тощо)
- **Машина**: Виберіть конкретну машину або залиште "Будь-яка машина"
- **Матеріал**: Виберіть тип матеріалу або залиште відкритим для будь-якого матеріалу
- **Діапазон товщини**: Встановіть мінімальні та максимальні значення товщини

#### 4. Налаштування параметрів

**Вкладка "Налаштування"** - Налаштуйте потужність, швидкість та інші параметри:

![Редактор рецептів - Вкладка "Налаштування"](/screenshots/recipe-editor-settings.png)

- Налаштуйте потужність, швидкість та інші параметри
- Налаштування автоматично адаптуються на основі вибраного типу завдання

### Система підбору рецептів

Rayforge автоматично пропонує найбільш відповідні рецепти на основі:

- **Сумісність машини**: Рецепти можуть бути специфічними для машини
- **Відповідність матеріалу**: Рецепти можуть націлюватися на конкретні матеріали
- **Діапазони товщини**: Рецепти застосовуються в межах визначених обмежень товщини
- **Відповідність можливостей**: Рецепти прив'язані до конкретних типів операцій

Система використовує алгоритм оцінки специфічності для пріоритизації найбільш релевантних рецептів:

1. Специфічні для машини рецепти мають вищий рейтинг, ніж загальні
2. Специфічні для лазерної головки рецепти мають вищий рейтинг
3. Специфічні для матеріалу рецепти мають вищий рейтинг
4. Специфічні для товщини рецепти мають вищий рейтинг
