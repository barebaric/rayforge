# 键盘快捷键

Rayforge 中键盘快捷键的完整参考。

:::note 平台约定
- **Linux/Windows：** `Ctrl` 键
- **macOS：** `⌘（Command）` 键
- 文档使用 `Ctrl` - macOS 用户替换为 `⌘`
:::


---

## 文件操作

| 快捷键         | 操作       | 描述               |
| -------------- | ---------- | ------------------ |
| `Ctrl+N`       | 新建草图   | 创建新草图         |
| `Ctrl+O`       | 打开       | 打开现有项目       |
| `Ctrl+S`       | 保存       | 保存当前项目       |
| `Ctrl+Shift+S` | 另存为     | 将项目保存为新文件 |
| `Ctrl+I`       | 导入       | 打开文件导入对话框 |
| `Ctrl+E`       | 导出       | 导出 G 代码        |
| `Ctrl+Q`       | 退出       | 退出 Rayforge      |
| `Ctrl+,`       | 首选项     | 打开首选项对话框   |

## 提示和技巧

### 效率提示

1. **首先学习编辑快捷键** - `Ctrl+Z/Y/C/V/D` 经常使用
2. **使用单键切换** - `H`、`T`、`P` 用于快速视图更改
3. **3D 视图快捷键** - `1`、`2`、`7` 用于即时视图切换
4. **对齐快捷键** - `Shift+Arrow` 比点击对齐按钮更快
5. **模拟模式** - `Ctrl+Shift+S` 快速检查执行

### 工作流程快捷键

**快速编辑循环：**

```
1. 新建草图（Ctrl+N）或打开（Ctrl+O）
2. 排列项目（箭头键，Shift+Arrow 用于对齐）
3. 复制零件（Ctrl+D）
4. 在 3D 中检查（F12）
5. 模拟（F11）
6. 保存（Ctrl+S）或导出（Ctrl+E）
```

**精确定位：**

```
1. 选择项目
2. 箭头键用于 1mm 微调
3. Shift+Arrow 用于 10mm 跳转
4. Shift+Home/End 用于居中
```

### 隐藏的快捷键

一些不太知名的快捷键：

- `Ctrl+,` - 快速访问首选项（macOS 上是标准的，这里也有效）
- `Ctrl+Shift+Z` - 备用重做（对于喜欢这个而不是 Ctrl+Y 的用户）
- `Alt+A` - 精确放置的像素完美捕捉
- `Alt+T` - 快速等距标签放置

## 平台差异

### Linux

- 使用标准的 `Ctrl` 修饰符
- 所有快捷键按文档工作
- GTK 标准快捷键适用于文本字段

### Windows

- 使用标准的 `Ctrl` 修饰符
- 与 Linux 快捷键相同
- 不使用 Windows 特定键（Win 键）

### macOS

- `Ctrl` 映射到 `⌘（Command）`
- `Alt` 映射到 `⌥（Option）`
- 应用标准 macOS 约定
- `Cmd+Q` 退出（而不是 Ctrl+Q）

## 快捷键故障排除

### 快捷键不起作用

**常见问题：**

1. **焦点在错误的元素上** - 确保画布或主窗口有焦点，而不是文本字段
2. **应用程序冲突** - 另一个应用可能正在拦截快捷键
3. **桌面环境快捷键** - 系统快捷键可能会覆盖（例如 Alt+F4）
4. **数字键盘与数字行** - 3D 视图快捷键使用数字行，而不是数字键盘

**解决方案：**

- 点击画布以确保它有焦点
- 检查系统键盘快捷键冲突
- 尝试菜单操作以验证功能

### 修饰键

**如果 Ctrl 不起作用：**

- 验证 Caps Lock 已关闭（可能在某些系统上干扰）
- 尝试键盘两侧的 Ctrl
- 检查键盘布局设置

**如果 Alt 不起作用：**

- 某些窗口管理器捕获 Alt 用于窗口拖动
- 尝试禁用窗口管理器 Alt 快捷键
- 使用菜单项作为替代
