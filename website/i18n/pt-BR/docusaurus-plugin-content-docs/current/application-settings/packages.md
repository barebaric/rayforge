# Pacotes

![Configurações de Pacotes](/screenshots/application-packages.png)

A página Pacotes mostra pacotes de extensão instalados e permite
gerenciá-los.

## Visão Geral

Pacotes são extensões que adicionam funcionalidade ao Rayforge. Eles podem
fornecer:

- Novas operações
- Filtros de importação/exportação
- Drivers de máquina
- Melhorias na interface
- Bibliotecas de materiais

## Pacotes Instalados

A lista de pacotes mostra todos os pacotes instalados com:

- **Nome**: O nome de exibição do pacote
- **Versão**: Versão atual instalada
- **Descrição**: Breve descrição da funcionalidade
- **Status**: Se está ativado ou desativado

## Gerenciando Pacotes

### Instalando Pacotes

1. Clique no botão **Instalar**
2. Selecione um arquivo de pacote (`.zip` ou repositório de pacote)
3. Confirme a instalação
4. Aguarde a conclusão da instalação

### Atualizando Pacotes

Quando atualizações estão disponíveis:

1. Um indicador de atualização aparece ao lado do pacote
2. Clique no botão **Atualizar** para instalar a atualização
3. Reinicie o Rayforge se solicitado

### Removendo Pacotes

1. Selecione o pacote a remover
2. Clique no botão **Remover**
3. Confirme a remoção

:::warning
Remover um pacote pode afetar documentos que usam seus recursos.
:::

## Fontes de Pacotes

Pacotes podem ser instalados de:

- **Arquivos locais**: Arquivos `.zip` baixados manualmente
- **Repositórios**: Repositórios Git contendo pacotes
- **Registro oficial**: Pacotes selecionados pela equipe Rayforge
