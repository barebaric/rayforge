# 设置

![常规设置](/screenshots/application-general.png)

自定义 Rayforge 以匹配您的工作流程和偏好。通过 **编辑 → 设置** 或按
<kbd>ctrl+comma</kbd> 打开设置对话框。

## 常规

常规页面包含应用程序范围设置。

### 外观

在**系统**、**浅色**或**深色**主题之间选择，以匹配您的桌面环境或个人
偏好。您还可以配置**操作颜色**，使用激光颜色或图层颜色在画布上进行视觉
区分。

### 单位

配置整个应用程序中使用的显示单位。您可以为**长度**（毫米、英寸等）、
**速度**（mm/min、mm/sec、inches/min 等）和**加速度**（mm/s² 等）
分别设置单位。

### 行为

默认情况下，操作在每次更改后会自动重新计算。如果您在较慢的机器上工作
或处理非常复杂的文档，可以禁用**自动更新操作**，改为通过工具栏按钮手动
触发重新计算。

Rayforge 可以在启动时**自动检查更新**。启用后，当有新版本可用时会通知
您。

您还可以配置**启动行为** — 以空白工作区启动、重新打开上次的项目或始终
打开特定的项目文件。请注意，命令行指定的文件将始终覆盖这些设置。

### 隐私

Rayforge 可以发送匿名使用数据以帮助改进应用程序。不会收集任何个人信息。
您可以随时开启或关闭**报告匿名使用**。请参阅
[使用追踪](https://rayforge.org/docs/general-info/usage-tracking)页面，
了解更多关于收集哪些数据以及如何使用这些数据的信息。

## 其他设置

设置对话框还包含管理应用程序其他部分的页面。每个页面都有自己的专门文档：

- [机器](../application-settings/machines) — 添加、移除和配置您的激光切割机
- [材料](../application-settings/materials) — 管理您的材料库
- [配方](../application-settings/recipes) — 管理保存的操作配方
- [AI 提供者](../application-settings/ai-provider) — 配置供扩展使用的 AI 提供者
- [扩展](../application-settings/addons) — 安装、更新和移除扩展
