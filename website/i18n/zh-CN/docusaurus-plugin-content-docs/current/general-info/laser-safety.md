# 激光安全

激光切割和雕刻设备存在严重的安全风险，包括火灾、有毒烟雾和永久性眼部损伤。

:::danger 关键安全警告
**始终遵循激光制造商的安全指南和操作说明。** 本页面提供一般安全原则，但不能替代适当的培训和您机器的特定安全文档。
:::


## 您的主要安全资源

**在操作任何激光设备之前，完整阅读激光制造商手册。** 它包含：

- 您机器的特定安全要求
- 所需的个人防护装备（PPE）
- 紧急程序
- 维护和检查计划
- 保修和责任信息

Rayforge 是控制软件 - 它无法覆盖激光硬件的物理安全要求。

## 基本安全原则

### 切勿让激光无人看管

**最重要的规则：** 切勿让运行中的激光无人看管，即使几秒钟也不行。

**原因：** 激光可以瞬间点燃材料。小火焰可能在几秒钟内变成严重的火灾。

**始终：**

- 保持在紧急停止范围内
- 持续观察切割过程
- 保持灭火器可及

### 消防安全

**每次作业前：**

- 清除工作区易燃材料
- 从切割床清除碎屑
- 准备好灭火器（ABC 或 CO2 类型）
- 知道紧急停止按钮在哪里

**如果发生火灾：**

1. 立即按下紧急停止
2. 如果火焰很小，保持盖子关闭（切断氧气）
3. 如果火灾继续，使用灭火器
4. 如果火灾蔓延，撤离并拨打紧急服务

### 通风是强制性的

**所有激光切割都会产生有毒烟雾。** 适当的通风不是可选的。

**要求：**

- 排气系统排放到室外（不循环）
- 足够的气流用于您的机器尺寸
- 所有激光操作期间运行
- 定期更换过滤器（如果使用过滤）

**切勿在没有适当通风的情况下操作** - 您冒着严重的健康影响风险，并将损坏激光的光学器件。

### 禁止材料

**切勿切割或雕刻这些材料：**

| 材料               | 危险                         |
| ------------------ | ---------------------------- |
| **PVC / 乙烯基**   | 产生氯气（有毒、腐蚀性）     |
| **ABS 塑料**       | 产生氰化物气体（致命）       |
| **聚碳酸酯**       | 有毒烟雾，效果差             |
| **玻璃纤维**       | 玻璃颗粒损害肺部和光学器件   |
| **任何含氯的材料** | 有毒且腐蚀性                 |

**如有疑问：** 检查材料安全数据表（MSDS）或在极好的通风条件下测试微小样本。

### 眼部保护

大多数封闭式激光系统在正常操作期间不需要护目镜，**如果外壳关闭并具有适当的联锁。**

**需要眼部保护的情况：**

- 操作期间打开外壳
- 执行维护或对齐
- 机器有观察窗但没有激光安全过滤
- 制造商指定

**始终检查制造商的要求** - 它们因机器设计和激光类型而异。

## 紧急程序

### 紧急停止

**何时使用：**

- 可见火焰
- 异常声音或烟雾
- 材料危险地移动
- 任何紧急情况

**如何：** 按下机器上的红色紧急停止按钮。所有运动和激光功率立即停止。

### 火灾响应

**小火（受控）：**

1. 紧急停止
2. 如可能保持盖子关闭
3. 如需要使用灭火器

**大火：**

1. 紧急停止
2. 立即撤离
3. 拨打紧急服务
4. 不要试图扑灭大火

### 医疗紧急情况

**眼部暴露：**

- 立即寻求医疗关注
- 不要揉眼睛
- 将激光规格带给医生

**烟雾吸入：**

- 移至新鲜空气处
- 如症状持续，寻求医疗关注
- 如严重，拨打毒物控制

## 培训和责任

**操作前：**

- 完成任何所需的安全培训
- 完整阅读制造商手册
- 理解紧急程序
- 了解您的特定激光的危险等级和要求

**持续：**

- 定期审查安全程序
- 维护安全设备
- 了解新材料和危险
- 切勿允许未经培训的用户操作设备

## 关键安全规则

:::warning 关键安全规则
1. **操作期间切勿让激光无人看管**
2. **切勿在没有适当通风的情况下操作**
3. **切勿切割禁止材料（PVC、乙烯基、ABS 等）**
4. **始终在触手可及的范围内备有灭火器**
5. **始终遵循制造商的安全指南**
6. **知道紧急停止在哪里**
7. **运行前在模拟模式中预览作业**
:::
