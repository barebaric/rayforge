# 投影模式

投影模式在一个单独的窗口中显示您的切割区域，设计用于在外部投影仪或辅助显示器上显示。通过将刀具路径直接投影到您的材料上，您可以准确看到激光切割的位置，使对齐变得简单。

投影窗口以亮绿色在黑色背景上显示您的工作件。它显示机器的轴范围框架和工作原点，以便您可以看到完整的切割区域和原点位置。当您在主画布上移动或修改工作件时，视图会实时更新。

## 打开投影窗口

从**视图 - 显示投影对话框**打开投影窗口。该窗口作为独立的窗口打开，您可以将其拖动到连接到系统的任何显示器上。

一个开关控制投影窗口——相同的菜单项可关闭它，当投影窗口处于焦点状态时按 Escape 键也可以关闭它。

## 全屏模式

点击投影窗口标题栏中的**全屏**按钮进入全屏模式。这会隐藏窗口装饰并填充整个显示器。点击**退出全屏**（同一个按钮）返回窗口模式。

投影到材料上时，全屏是预期使用的模式，因为它去除了分散注意力的窗口边框，并使用整个显示表面。

## 不透明度

标题栏中的不透明度按钮在四个级别之间循环：100%、80%、60% 和 40%。降低不透明度会使投影窗口变为半透明，这在桌面显示器上可能有用，可以看到后面的窗口。每次点击都会前进到下一个不透明度级别并循环返回。

## 投影显示的内容

投影显示渲染文档的简化视图。工作件显示为亮绿色轮廓，展示计算出的刀具路径——与将发送到激光的路径相同。工作件的基底图像不会显示，保持显示集中在切割路径上。

机器范围框架显示为边框，代表机器轴的全部行程区域。工作原点十字线显示坐标系原点在该区域内的位置。当您更改机器上的工件坐标系偏移时，两者都会自动更新。
