# Згладжування шляху

Згладжування шляху зменшує зубчасті краї та різкі переходи у ваших шляхах різання, що призводить до більш плавних кривих та м'якшого руху машини.

## Як це працює

Згладжування застосовує фільтр до вашої геометрії шляху, який заокруглює кутові кути та згладжує нерівні краї. Лазер слідує м'якшою траєкторією замість різких змін напрямку.

## Налаштування

### Увімкнути згладжування

Увімкніть або вимкніть згладжування для цієї операції. Згладжування вимкнено за замовчуванням.

### Гладкість

Контролює, наскільки сильно згладжується шлях (0-100). Вищі значення створюють більш заокруглені криві, але можуть більше відхилятися від оригінального шляху.

- **Низька (0-30):** Мінімальне згладжування, зберігає гострі деталі
- **Середня (30-60):** Збалансоване згладжування для більшості дизайнів
- **Висока (60-100):** Агресивне згладжування, найкраще для органічних форм

### Поріг кута кута

Кути гостріші за це значення зберігаються як кути замість згладжування (0-179 градусів). Це запобігає заокругленню важливих гострих елементів.

- **Нижчі значення:** Більше кутів згладжується, більш заокруглений результат
- **Вищі значення:** Більше кутів зберігається, більш гострий результат

## Коли використовувати згладжування

**Добре для:**

- Дизайнів, імпортованих з піксельних джерел зі сходинками
- Зменшення механічного навантаження при різких змінах напрямку
- Покращення якості різу на кривих
- Дизайнів з багатьма малими відрізками ліній

**Не потрібно для:**

- Чистої векторної графіки з плавними кривими Безьє
- Дизайнів, де гострі кути повинні бути збережені точно
- Технічних креслень, що вимагають точної геометрії
