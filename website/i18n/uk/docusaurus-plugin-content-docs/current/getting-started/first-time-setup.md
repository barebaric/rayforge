# Перше налаштування

Після встановлення Rayforge вам потрібно налаштувати ваш лазерний різак або гравер. Цей посібник проведе вас через створення першої машини та встановлення з'єднання.

## Крок 1: Запуск Rayforge

Запустіть Rayforge з меню програм або виконавши `rayforge` у терміналі. Ви повинні побачити головний інтерфейс з порожнім полотном.

## Крок 2: Створення машини

Перейдіть до **Налаштування → Машини** або натисніть <kbd>ctrl+кома</kbd>, щоб відкрити діалог налаштувань, потім виберіть сторінку **Машини**.

Натисніть **Додати машину**, щоб створити нову машину. Ви можете:

1. **Вибрати вбудований профіль** - Виберіть із попередньо визначених шаблонів машин
2. **Вибрати "Користувацький"** - Почніть з порожньої конфігурації

Після вибору відкриється діалог налаштувань машини для вашої нової машини.

![Налаштування машини](/screenshots/application-machines.png)

## Крок 3: Налаштування загальних параметрів

Сторінка **Загальні** містить базову інформацію про машину, вибір драйвера та налаштування з'єднання.

![Загальні налаштування](/screenshots/machine-general.png)

### Інформація про машину

1. **Назва машини**: Дайте вашій машині описову назву (наприклад, "K40 Laser", "Ortur LM2")

### Вибір драйвера

Виберіть відповідний драйвер для вашого пристрою зі списку:

- **GRBL Serial** - Для пристроїв GRBL, підключених через USB/послідовний порт
- **GRBL Network** - Для пристроїв GRBL з WiFi/Ethernet підключенням
- **Smoothie** - Для пристроїв на базі Smoothieware

### Налаштування драйвера

Залежно від обраного драйвера, налаштуйте параметри з'єднання:

#### GRBL Serial (USB)

1. **Порт**: Виберіть ваш пристрій зі списку (наприклад, `/dev/ttyUSB0` на Linux, `COM3` на Windows)
2. **Швидкість передачі**: Виберіть `115200` (стандарт для більшості пристроїв GRBL)

:::info
Якщо ваш пристрій не з'являється в списку, перевірте, що він підключений і що у вас є необхідні дозволи. На Linux вам може знадобитися додати свого користувача до групи `dialout`.
:::

#### GRBL Network / Smoothie (WiFi/Ethernet)

1. **Хост**: Введіть IP-адресу вашого пристрою (наприклад, `192.168.1.100`)
2. **Порт**: Введіть номер порту (зазвичай `23` або `8080`)

### Швидкості та прискорення

Ці налаштування використовуються для оцінки часу роботи та оптимізації траєкторії:

- **Макс. швидкість переміщення**: Максимальна швидкість швидкого переміщення
- **Макс. швидкість різання**: Максимальна швидкість різання
- **Прискорення**: Використовується для оцінок часу та розрахунків оверскану

## Крок 4: Налаштування апаратних параметрів

Перейдіть на вкладку **Апаратне забезпечення**, щоб налаштувати фізичні розміри вашої машини.

![Налаштування апаратного забезпечення](/screenshots/machine-hardware.png)

### Розміри

- **Ширина**: Введіть максимальну ширину робочої області в міліметрах
- **Висота**: Введіть максимальну висоту робочої області в міліметрах

### Осі

- **Початок координат (0,0)**: Виберіть, де знаходиться початок координат вашої машини:
  - Нижній лівий кут (найпоширеніший для GRBL)
  - Верхній лівий кут
  - Верхній правий кут
  - Нижній правий кут

### Зміщення осей (необов'язково)

Налаштуйте зміщення X та Y, якщо ваша машина потребує їх для точного позиціонування.

## Крок 5: Автоматичне підключення

Rayforge автоматично підключається до вашої машини при запуску програми (якщо машина увімкнена та підключена). Вам не потрібно вручну натискати кнопку підключення.

Статус з'єднання відображається в нижньому лівому куті головного вікна з іконкою статусу та міткою, що показує поточний стан (Підключено, Підключення, Відключено, Помилка тощо).

:::success Підключено!
Якщо ваша машина показує статус "Підключено", ви готові почати використовувати Rayforge!
:::

## Додатково: Розширене налаштування

### Кілька лазерів

Якщо ваша машина має кілька лазерних модулів (наприклад, діодний та CO2), ви можете налаштувати їх на сторінці **Лазер**.

![Налаштування лазера](/screenshots/machine-laser.png)

Дивіться [Налаштування лазера](../machine/laser) для деталей.

### Налаштування камери

Якщо у вас є USB-камера для вирівнювання та позиціонування, налаштуйте її на сторінці **Камера**.

![Налаштування камери](/screenshots/machine-camera.png)

Дивіться [Інтеграція камери](../machine/camera) для деталей.

### Налаштування пристрою

Сторінка **Пристрій** дозволяє зчитувати та змінювати налаштування прошивки безпосередньо на підключеному пристрої (наприклад, параметри GRBL). Це розширена функція, і її слід використовувати з обережністю.

:::warning
Редагування налаштувань пристрою може бути небезпечним і може вивести вашу машину з ладу при введенні неправильних значень!
:::

## Виправлення проблем зі з'єднанням

### Пристрій не знайдено

- **Linux (Serial)**: Додайте свого користувача до групи `dialout`. Це
  необхідно для **обох Snap і не-Snap інсталяцій** на дистрибутивах на основі
  Debian, щоб уникнути повідомлень AppArmor DENIED:
  ```bash
  sudo usermod -a -G dialout $USER
  ```
  Вийдіть із системи та увійдіть знову, щоб зміни набули чинності.

- **Snap-пакет**: На додачу до групи `dialout` вище, переконайтеся, що ви
  надали дозволи на послідовний порт:
  ```bash
  sudo snap connect rayforge:serial-port
  ```

- **Windows**: Перевірте Диспетчер пристроїв, щоб підтвердити, що пристрій
  розпізнається, і запишіть номер COM-порту.

### У з'єднанні відмовлено

- Перевірте, що IP-адреса та номер порту правильні
- Переконайтеся, що ваша машина увімкнена та підключена до мережі
- Перевірте налаштування брандмауера при використанні мережевого з'єднання

### Машина не відповідає

- Спробуйте іншу швидкість передачі (деякі пристрої використовують `9600` або `57600`)
- Перевірте чи не роз'єднані кабелі або поганий контакт
- Перемкніть живлення лазерного різака і спробуйте знову

Для додаткової допомоги дивіться [Проблеми зі з'єднанням](../troubleshooting/connection).
