# 高级设置

机器设置中的高级页面包含用于专用用例的额外配置选项。

![高级设置](/screenshots/machine-advanced.png)

## 连接行为

控制 Rayforge 在连接期间如何与您的机器交互的设置。

### 连接时归零

启用后，Rayforge 在连接到机器时自动发送归零命令（$H）。

- **启用如果**：您的机器有可靠的限位开关
- **禁用如果**：您的机器无限位开关或归零不可靠

### 连接时清除警报

启用后，Rayforge 在连接时自动清除任何警报状态。

- **启用如果**：您的机器经常以警报状态启动
- **禁用如果**：您想在清除之前手动调查警报

### 允许单轴归零

启用后，您可以独立归零单个轴（X、Y 或 Z），而不需要所有轴一起归零。这对于某个轴已经正确定位的机器很有用。

## 弧线和曲线设置

控制曲线路径如何转换为 G-code 移动的设置。

### 支持弧线

启用后，Rayforge 为曲线路径生成弧线命令（G2/G3），而不是将其分解为许多小的线性移动。这在大多数控制器上产生更紧凑的 G-code 和更平滑的运动。

禁用时，所有曲线都转换为线性段（G1 命令），这为不支持弧线的控制器提供了最大的兼容性。

### 支持 Bézier 曲线

启用后，Rayforge 为曲线路径生成原生三次 Bézier 命令（如 LinuxCNC 使用的 G5 命令）。这在支持它的控制器上产生非常平滑的运动和紧凑的 G-code。如果您的机器固件不支持 Bézier 命令，您应该禁用此设置，此时曲线将被分解为线性段。

### 弧线和曲线公差

此设置控制将弧线和曲线拟合到曲线路径时允许的最大偏差，以毫米为单位。较小的值产生更准确的路径，但可能需要更多的命令。较大的值允许更多偏差，但生成更少的命令。

典型值从精密工作的 0.01mm 到更快处理的 0.1mm。
