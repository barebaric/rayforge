# 多图层工作流程

![图层面板](/screenshots/bottom-panel-layers.png)

Rayforge 的多图层系统允许您将作业组织到独立的处理阶段。每个
图层是工件的容器，并拥有自己的工作流程——一系列步骤，每个
步骤都有独立的激光设置。

:::tip 什么时候不需要多个图层
在很多情况下，一个图层就足够了。图层内的每个步骤都有自己的
激光、功率、速度和其他设置，因此您可以在同一图层中进行雕刻
和轮廓切割。只有当您需要用不同设置对一张图像的不同部分进行
轮廓切割，或者需要不同的 WCS 或旋转配置时，才需要单独的
图层。
:::

## 创建和管理图层

### 添加图层

点击图层面板中的 **+** 按钮。新文档默认包含三个空图层。

### 重新排列图层

在面板中拖放图层以更改执行顺序。图层从左到右依次处理。您
可以使用**鼠标中键拖动**在图层列表中平移。

### 重新排列工件

图层内的工件可以通过拖放重新排列，以控制它们的 Z 轴顺序。您可以使用 **Ctrl+点击** 切换选择单个项目，或使用 **Shift+点击** 选择一个范围。拖动选中的项目可一次移动所有选中的项目。

选中的工件在图层列中高亮显示，并且选区与画布保持同步。

### 删除图层

选择图层并点击删除按钮。图层中的所有工件将被移除。如有
需要，您可以撤销删除操作。

## 图层属性

每个图层具有以下设置，可通过图层列上的齿轮图标访问：

- **名称** — 显示在图层标题中
- **颜色** — 用于在画布上渲染该图层的操作
- **可见性** — 眼睛图标切换图层是否在画布和预览中显示。
  隐藏的图层仍会包含在生成的 G-code 中。
- **坐标系 (WCS)** — 为此图层分配工作坐标系。当设置为特定
  的 WCS（如 G54、G55）时，机器会在图层开始时切换到该坐标
  系统。选择**默认**以使用全局 WCS。
- **旋转模式** — 为此图层启用旋转附件模式，允许您在同一
  项目中混合平面和圆柱形加工。在图层设置中配置旋转模块和
  物体直径。

## 图层工作流程

每个图层都有一个**工作流程**——一系列步骤，显示为图层列中的
图标管道。每个步骤定义一个单独的操作（如轮廓切割、光栅雕刻），
拥有自己的激光、功率、速度和其他设置。

点击步骤进行配置。使用管道中的 **+** 按钮向图层添加更多
步骤。步骤可以通过拖放重新排列。

## 矢量文件导入

导入矢量文件（SVG、DXF、PDF）时，导入对话框提供三种方式来
处理源文件中的图层：

- **映射到现有图层** — 按位置将每个源图层导入到对应的文档
  图层
- **新建图层** — 为每个源图层创建新的文档图层
- **合并** — 将所有内容导入到当前活动图层

使用**映射到现有图层**或**新建图层**时，对话框会显示源文件
中的图层列表，并带有开关以选择要导入的图层。

## 将工件分配到图层

**拖放：** 在画布或图层面板中选择工件，然后拖动到目标图层。支持使用 Ctrl+点击和 Shift+点击进行多选，您可以在图层之间拖动项目。

**剪切粘贴：** 从当前图层剪切工件（Ctrl+X），选择目标图层，
然后粘贴（Ctrl+V）。

**上下文菜单：** 在图层标签中右键点击工件以打开上下文菜单，包含将其移动到其他图层、删除或打开其属性的选项。

## 执行顺序

在作业过程中，图层从左到右依次处理。在每个图层内，所有工件
处理完毕后才会进入下一个图层。标准工作流程是先雕刻后切割，
这样工件在雕刻过程中能保持在原位。
