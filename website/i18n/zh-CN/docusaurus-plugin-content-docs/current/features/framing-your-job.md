# 框定您的工作

了解如何使用框定功能预览工作边界，确保在切割前正确对齐。

## 概述

框定功能允许您预览激光工作的精确边界，方法是使用低功率激光或关闭激光来
描绘轮廓。这有助于验证定位并防止代价高昂的错误。

## 何时使用框定

- **首次设置**：验证材料放置
- **精确定位**：确保设计在材料边界内
- **多个工作**：在每次运行前确认对齐
- **昂贵材料**：在切割前仔细检查

## 如何框定

### 方法一：仅轮廓

在不开启激光的情况下描绘工作边界：

1. **在 Rayforge 中加载设计**
2. **将材料放置**在激光工作台上
3. **点击工具栏中的框定按钮**
4. **观察激光头**描绘边界矩形
5. **验证定位**并根据需要调整材料

### 方法二：低功率预览

部分机器支持低功率框定，带有可见光束：

1. **在机器设置中启用低功率模式**
2. **设置框定功率**（通常为 1-5%）
3. **运行框定操作**
4. **观察材料表面**描绘的轮廓

:::warning 检查您的机器
并非所有激光器都能安全支持低功率框定。使用此功能前请查阅机器文档。
:::

## 框定设置

在机器的激光头设置中配置框定行为：

- **框定速度**：框定期间激光头的移动速度。此设置针对每个激光头
  单独配置，因此如果您的机器有多个激光器，可以为每个设置不同的
  速度。
- **框定功率**：框定期间的激光功率（0 表示关闭，低百分比表示
  可见痕迹）
- **角落停留时间**：在框定轮廓的每个角落短暂停留。这让您有时间
  准确看到每个角落的位置——在较高框定速度下尤其有用。
- **重复次数**：描绘轮廓的次数。设置为大于一的值可以使框定路径
  更容易被肉眼追踪。

## 使用框定结果

框定完成后，您可以：

- **调整材料位置**（如需要）
- **重新框定**以验证新位置
- **确认位置后继续工作**

## 有效框定的技巧

- **标记角落**：在角落放置小块胶带作为参考
- **检查间隙**：确保设计周围有足够的空间
- **确认方向**：验证材料方向是否正确
- **考虑切缝**：请记住，切割将比轮廓线略宽

## 使用摄像头框定

如果您的机器支持摄像头，可以：

1. **拍摄材料放置**的摄像头图像
2. **在摄像头视图上叠加设计**
3. **在框定前虚拟调整位置**
4. **框定确认**物理对齐

详见[摄像头集成](../machine/camera)。

## 故障排除

**框定与设计不匹配**：检查工作原点和坐标系设置

**框定时激光发射**：禁用框定功率或检查机器设置

**框定太快看不清**：在设置中降低框定速度

**激光头无法到达角落**：验证设计是否在机器工作区域内

## 安全注意事项

- **框定期间切勿无人看管机器**
- **使用零功率框定时验证激光已关闭**
- **远离激光头运动路径**
- **注意可能干扰运动的障碍物**
