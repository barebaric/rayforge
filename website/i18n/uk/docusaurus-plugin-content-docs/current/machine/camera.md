# Інтеграція камери

Rayforge підтримує інтеграцію USB-камери для точного вирівнювання та
позиціонування матеріалу. Функція накладання камери дозволяє бачити точно, де
ваш лазер буде різати або гравірувати на матеріалі, усуваючи здогадки та
зменшуючи відходи матеріалу.

![Налаштування камери](/screenshots/machine-camera.png)

## Робочий процес налаштування

Налаштування камери відбувається в чотири кроки:

1. **Додати камеру** — Підключіть камеру та додайте її до конфігурації машини
2. **Налаштувати параметри зображення** — Відрегулюйте яскравість, контраст,
   баланс білого та зменшення шуму
3. **Відкалібрувати об'єктив** — Виправте дисторсію за допомогою майстра
   калібрування або ручних коефіцієнтів
4. **Вирівняти камеру** — Відобразіть пікселі камери на координати машини для
   точного позиціонування

Кроки 2–4 доступні з панелі властивостей камери, де значки статусу показують
прогрес з першого погляду:

- ✓ **Калібрування об'єктива** — Калібрування виконано
- ⚠ **Вирівнювання зображення** — Попередження, коли вирівнювання потрібно
  повторити (наприклад, після калібрування об'єктива)
- ✓ **Вирівнювання зображення** — Вирівнювання актуальне та дійсне

## Крок 1: Додавання камери

### Вимоги до обладнання

**Сумісні камери:**

- USB веб-камери (найпоширеніші)
- Вбудовані камери ноутбуків (якщо Rayforge запущено на ноутбуці поруч
  з машиною)
- Будь-яка камера, що підтримується Video4Linux2 (V4L2) на Linux або
  DirectShow на Windows

**Рекомендоване налаштування:**

- Камера, закріплена над робочою областю з чітким видом матеріалу
- Послідовні умови освітлення
- Камера позиціонована для захвату робочої області лазера
- Надійне кріплення для запобігання русі камери

### Додавання камери

1. **Підключіть камеру** до комп'ютера через USB

2. **Відкрийте налаштування камери:**
   - Перейдіть до **Налаштування → Налаштування → Камера**
   - Або використовуйте кнопку камери на панелі інструментів

3. **Додайте нову камеру:**
   - Натисніть кнопку "+"
   - Введіть описову назву (наприклад, "Верхня камера",
     "Камера робочої області")
   - Виберіть пристрій зі списку
     - На Linux: `/dev/video0`, `/dev/video1` тощо.
     - На Windows: Camera 0, Camera 1 тощо.

4. **Увімкніть камеру:**
   - Перемкніть перемикач увімкнення камери
   - Живе зображення з'явиться на вашому полотні

## Крок 2: Налаштування параметрів зображення

![Діалог налаштувань зображення](/screenshots/camera-image-settings.png)

Натисніть **Налаштувати** поруч з **Параметри зображення** у властивостях
камери, щоб відкрити діалог налаштувань зображення. Відрегулюйте ці параметри
для отримання чіткого зображення камери:

| Налаштування        | Опис                                                                                          |
| ------------------- | --------------------------------------------------------------------------------------------- |
| **Яскравість**      | Загальна яскравість зображення (-100 до +100)                                                 |
| **Контраст**        | Визначення країв та контраст (0 до 100)                                                       |
| **Переважати YUYV** | Використовувати нестиснутий YUYV замість MJPEG. Повільніше, але може виправити деякі проблеми |
| **Прозорість**      | Непрозорість накладання на полотні (0% непрозоре до 100% прозоре)                             |
| **Баланс білого**   | Корекція колірної температури (Авто або 2500-10000K)                                          |
| **Зменшення шуму**  | Часове зменшення шуму (0.0 до 0.95)                                                           |

Опція YUYV корисна, якщо ваша камера видає зеленуваті зображення з форматом
MJPEG за замовчуванням. Зверніть увагу, що YUYV не стиснутий і може зменшити
доступну роздільну здатність або частоту кадрів на USB 2.0 з'єднаннях.

## Крок 3: Калібрування об'єктива

Якщо ваша камера має ширококутний об'єктив або закріплена під кутом,
зображення може показувати видиму кривину — прямі лінії виглядають
вигнутими, особливо ближче до країв кадру. Це називається дисторсією
об'єктива, і вона може порушити вирівнювання навіть при ретельному
вимірюванні точок.

Rayforge включає майстер калібрування з підказками, який автоматично
виправляє цю дисторсію. Ви також можете налаштовувати коефіцієнти
дисторсії вручну.

### Діалог калібрування об'єктива

![Діалог калібрування об'єктива](/screenshots/camera-lens-calibration.png)

Відкрийте діалог калібрування об'єктива, натиснувши **Налаштувати**
поруч з **Калібрування об'єктива** у властивостях камери. Звідси ви
можете:

- **Налаштовувати коефіцієнти дисторсії вручну** — Точне налаштування
  параметрів радіальної (k1–k3) та тангенціальної (p1–p2) дисторсії
- **Запустити майстер калібрування** — Натисніть кнопку **Майстер**
  для автоматичного калібрування з підказками

Ручне налаштування корисне для точного доопрацювання після того, як
майстер обчислив початкове рішення, або коли ви знаєте приблизні
значення дисторсії для вашого об'єктива.

### Майстер калібрування

Майстер калібрування проведе вас через захоплення кількох зображень
надрукованої калібрувальної картки з різних позицій на столі. Потім
він автоматично обчислює модель дисторсії.

**Крок 1: Налаштування калібрувальної картки**

![Майстер — Налаштування
картки](/screenshots/camera-lens-calibration-wizard-card.png)

1. Натисніть **Майстер** у діалозі калібрування об'єктива, щоб почати
2. Встановіть **Ширину** та **Висоту** вашої надрукованої картки
3. Попередній перегляд оновлюється в реальному часі — картка має
   покривати близько 70% виду камери
4. Натисніть **Зберегти як PDF**, щоб експортувати картку для друку
5. Роздрукуйте картку та розмістіть її на робочому столі лазера

**Крок 2: Захоплення кадрів**

![Майстер —
Захоплення](/screenshots/camera-lens-calibration-wizard-capture.png)

1. Натисніть **Далі**, щоб увійти в режим захоплення
2. Розмістіть калібрувальну картку в різних місцях та під різними
   кутами в межах виду камери
3. Натисніть **Захопити кадр** для кожної позиції
4. Намагайтеся зробити щонайменше 8 захоплень, що покривають весь
   кадр, включаючи кути та краї
5. Індикатор прогресу та індикатори статусу показують якість захоплення

**Крок 3: Застосування калібрування**

1. Коли достатньо кадрів захоплено, натисніть **Калібрувати**
2. Обчислені коефіцієнти дисторсії автоматично застосовуються до камери
3. Накладання камери тепер показує виправлене, пряме зображення

## Крок 4: Вирівнювання зображення

![Діалог вирівнювання зображення](/screenshots/camera-image-alignment.png)

Вирівнювання камери калібрує співвідношення між пікселями камери та реальними
координатами, забезпечуючи точне позиціонування.

### Чому необхідне вирівнювання

Камера бачить робочу область зверху, але зображення може бути:

- Повернуте відносно осей машини
- Масштабоване по-різному в напрямках X та Y
- Спотворене перспективою об'єктива

Вирівнювання створює матрицю трансформації, яка відображає пікселі камери в
координати машини.

### Процедура вирівнювання

1. **Відкрийте діалог вирівнювання:**
   - Натисніть кнопку **Налаштувати** поруч з **Вирівнювання зображення** у
     властивостях камери
   - Діалог показує потік камери з поточним накладанням вирівнювання

2. **Розмістіть маркери вирівнювання:**
   - Вам потрібно щонайменше 3 опорні точки (4 рекомендується для кращої
     точності)
   - Точки вирівнювання мають бути розподілені по робочій області
   - Використовуйте відомі позиції, такі як:
     - Позиція home машини
     - Позначки лінійки
     - Попередньо вирізані отвори вирівнювання
     - Калібрувальна сітка

3. **Позначте точки зображення:**
   - Клацніть на зображенні камери, щоб розмістити точку у відомому місці
   - З'явиться віджет з координатами точки
   - Повторіть для кожної опорної точки

4. **Введіть світові координати:**
   - Для кожної точки зображення введіть реальні координати X/Y в мм
   - Це фактичні координати машини, де знаходиться кожна точка
   - Виміряйте точно лінійкою або використовуйте відомі позиції машини

5. **Застосуйте вирівнювання:**
   - Натисніть **Застосувати** для розрахунку трансформації
   - Накладання камери тепер буде правильно вирівняно

6. **Перевірте вирівнювання:**
   - Перемістіть лазерну головку у відому позицію
   - Перевірте, що точка лазера вирівнюється з очікуваною позицією у видові
     камери
   - Точно налаштуйте повторним вирівнюванням за потреби

### Статус вирівнювання

Панель властивостей камери показує статус вирівнювання за допомогою значка:

- **Галочка** — Вирівнювання актуальне та дійсне
- **Попередження** — Вирівнювання потрібно повторити. Це відбувається, коли
  оновлюється калібрування об'єктива, оскільки корекція дисторсії змінює
  зображення камери та робить існуюче вирівнювання недійсним. Ваші точки
  вирівнювання зберігаються — просто відкрийте діалог і натисніть
  **Застосувати** знову.

### Приклад робочого процесу

1. Перемістіть лазер у позицію home (0, 0) і позначте на камері
2. Перемістіть лазер у (100, 0) і позначте на камері
3. Перемістіть лазер у (100, 100) і позначте на камері
4. Перемістіть лазер у (0, 100) і позначте на камері
5. Введіть точні координати для кожної точки
6. Застосуйте та перевірте

:::tip Найкращі практики

- Використовуйте точки в кутах вашої робочої області для максимального покриття
- Уникайте скупчення точок в одній області
- Вимірюйте світові координати ретельно - точність тут визначає
  загальну якість вирівнювання
- Повторно вирівнюйте, якщо перемістили камеру або змінили дистанцію
  фокусування
- Повторно вирівнюйте після оновлення калібрування об'єктива
- Зберігайте ваше вирівнювання - воно зберігається між сесіями
  :::

## Використання накладання камери

Після вирівнювання накладання камери допомагає точно позиціонувати
завдання. Увімкніть його, натиснувши іконку камери на панелі
інструментів головного вікна.

---

### Кілька камер

Rayforge підтримує кілька камер для різних видів або машин:

- Додайте кілька камер у налаштуваннях
- Кожна камера може мати незалежне вирівнювання
- Перемикайтеся між камерами за допомогою селектора камер
- Випадки використання:
  - Вид зверху + вид збоку для 3D-об'єктів
  - Різні камери для різних машин
  - Ширококутна + детальна камера

## Виправлення проблем

### Камера не виявлена

**Проблема:** Камера не з'являється в списку пристроїв.

**Рішення:**

**Linux:**
Перевірте, чи камера розпізнається системою:

```bash
# Список відеопристроїв
ls -l /dev/video*

# Перевірте камеру з v4l2
v4l2-ctl --list-devices

# Перевірте іншим додатком
cheese  # або VLC тощо.
```

**Для користувачів Snap:**

```bash
# Надайте доступ до камери
sudo snap connect rayforge:camera
```

**Windows:**

- Перевірте Device Manager на наявність камери в "Cameras" або
  "Imaging devices"
- Переконайтеся, що жоден інший додаток не використовує камеру
  (закрийте Zoom, Skype тощо)
- Спробуйте інший USB-порт
- Оновіть драйвери камери

### Камера показує чорний екран

**Проблема:** Камера виявлена, але не показує зображення.

**Можливі причини:**

1. **Камера використовується іншим додатком** - Закрийте інші відеододатки
2. **Невірний пристрій вибрано** - Спробуйте різні ID пристроїв
3. **Дозволи камери** - На Linux Snap переконайтеся, що інтерфейс
   камери підключено
4. **Апаратна проблема** - Перевірте камеру іншим додатком

**Рішення:**

```bash
# Linux: Звільніть пристрій камери
sudo killall cheese  # або інші додатки камери

# Перевірте, який процес використовує камеру
sudo lsof /dev/video0
```

### Вирівнювання неточне

**Проблема:** Накладання камери не відповідає реальній позиції лазера.

**Діагностика:**

1. **Недостатньо точок вирівнювання** - Використовуйте щонайменше 4 точки
2. **Помилки вимірювання** - Перевірте світові координати
3. **Камера перемістилася** - Повторно вирівняйте, якщо позиція
   камери змінилася
4. **Нелінійна дисторсія** - Може знадобитися калібрування об'єктива

**Покращення точності:**

- Використовуйте більше точок вирівнювання (6-8 для дуже великих областей)
- Розподіліть точки по всій робочій області
- Вимірюйте світові координати дуже ретельно
- Використовуйте команди переміщення машини для точного
  позиціонування лазера у відомих координатах
- Повторно вирівнюйте після будь-яких коригувань камери

### Погана якість зображення

**Проблема:** Зображення камери розмите, темне або пересвічене.

**Рішення:**

1. **Налаштуйте яскравість/контраст** у налаштуваннях камери
2. **Покращіть освітлення** - Додайте послідовне освітлення робочої області
3. **Очистіть об'єктив камери** - Пил та сміття знижують чіткість
4. **Перевірте фокус** - Автофокус може працювати погано;
   використовуйте ручний, якщо можливо
5. **Тимчасово зменшіть прозорість**, щоб чіткіше бачити зображення камери
6. **Спробуйте різні налаштування балансу білого**
7. **Налаштуйте зменшення шуму**, якщо зображення з'являється зернистим

### Затримка або переривчастість камери

**Проблема:** Живе зображення камери переривчасте або затримане.

**Рішення:**

- Знизьте роздільну здатність камери у налаштуваннях пристрою (якщо доступно)
- Закрийте інші додатки, що використовують CPU/GPU
- Оновіть графічні драйвери
