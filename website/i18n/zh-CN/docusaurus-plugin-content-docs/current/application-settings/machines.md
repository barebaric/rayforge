# 机器

![机器设置](/screenshots/application-machines.png)

应用设置中的机器页面允许您管理机器配置文件。每个配置文件包含特定激光机器的所有配置。

## 机器配置文件

机器配置文件存储激光切割机或雕刻机的完整配置，包括：

- **常规设置**：名称、速度、加速度
- **硬件设置**：工作区域尺寸、轴配置
- **激光设置**：功率范围、PWM 频率
- **设备设置**：串口、波特率、固件类型
- **G 代码设置**：自定义 G 代码方言选项
- **相机设置**：相机校准和对齐

## 管理机器

### 添加新机器

1. 点击 **添加新机器** 按钮
2. 为机器输入描述性名称
3. 配置机器设置（有关详细信息，请参阅[机器设置](../machine/general)）
4. 点击 **保存** 创建配置文件

### 在机器之间切换

使用主窗口中的机器选择器下拉菜单在配置的机器之间切换。所有设置，包括所选机器，都会在会话之间记住。

### 复制机器

要创建类似的机器配置文件：

1. 选择要复制的机器
2. 点击 **复制** 按钮
3. 重命名新机器并根据需要调整设置

### 删除机器

1. 选择要删除的机器
2. 点击 **删除** 按钮
3. 确认删除

:::warning
删除机器配置文件无法撤销。确保在删除之前已记录任何重要设置。
:::
