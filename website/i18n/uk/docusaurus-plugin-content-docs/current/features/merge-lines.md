# Об'єднання ліній

Під час імпорту дизайну, що містить контури, які перекриваються, лазер може
опинитися в ситуації, коли одна й та сама лінія прорізається кілька разів. Це
марнує час, може спричинити надмірне обгоряння та розширити різ понад бажаний
розмір.

Постпроцесор **Об'єднання ліній** виявляє сегменти контуру, що перекриваються
або збігаються, та об'єднує їх в один прохід. Лазер проходить кожну унікальну
лінію лише один раз.

## Коли варто використовувати

Це найчастіше трапляється, коли:

- Ви імпортуєте SVG або DXF, де фігури мають спільні ребра (наприклад,
  сітчастий візерунок або мозаїка)
- Ви комбінуєте кілька деталей, контури яких перекриваються
- Ваша програма для дизайну експортує дубльовані контури

## Коли не варто використовувати

Якщо перекриття прорізів навмисне — наприклад, виконання кількох проходів по
одній лінії для прорізання товстішого матеріалу — залиште об'єднання ліній
вимкненим. У цьому випадку краще скористатися функцією
[Багаторазовий прохід](multi-pass), яка дає явний контроль над кількістю
проходів.
