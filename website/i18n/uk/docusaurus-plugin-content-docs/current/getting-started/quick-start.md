# Посібник швидкого старту

Тепер, коли Rayforge встановлено і ваша машина налаштована, давайте виконаємо першу лазерну роботу! Цей посібник проведе вас через імпорт дизайну, налаштування операцій та відправку G-коду на вашу машину.

## Крок 1: Імпорт дизайну

Rayforge підтримує різні формати файлів, включаючи SVG, DXF, PDF та растрові зображення (JPEG, PNG, BMP).

1. **Натисніть** **Файл → Відкрити** або натисніть <kbd>ctrl+o</kbd>
2. Перейдіть до файлу дизайну та виберіть його
3. Дизайн з'явиться на полотні

![Полотно з імпортованим дизайном](/screenshots/main-standard.png)

:::tip Немає дизайну?
Ви можете створювати прості форми за допомогою інструментів полотна або завантажити безкоштовні SVG-файли з таких сайтів, як [Flaticon](https://www.flaticon.com/) або [SVG Repo](https://www.svgrepo.com/).
:::


## Крок 2: Розміщення дизайну

Використовуйте інструменти полотна для розміщення та налаштування вашого дизайну:

- **Панорама**: Середня кнопка миші та перетягування, або утримуйте <kbd>пробіл</kbd> і перетягуйте
- **Масштаб**: Колесо миші, або <kbd>ctrl+"+"</kbd> / <kbd>ctrl+"-"</kbd>
- **Переміщення**: Натисніть і перетягніть ваш дизайн
- **Обертання**: Виділіть дизайн і використовуйте маркери обертання
- **Масштабування**: Виділіть дизайн і перетягніть кутові маркери

## Крок 3: Призначення операції

Операції визначають, як Rayforge оброблятиме ваш дизайн. Поширені операції включають:

- **Контур**: Різання по контуру форм
- **Растрове гравіювання**: Заповнення форм зигзагоподібними лініями (для гравіювання)
- **Гравіювання з глибиною**: Створення 3D-ефектів глибини із зображень

### Додавання операції

1. Виділіть ваш дизайн на полотні
2. Натисніть **Операції → Додати операцію** або натисніть <kbd>ctrl+shift+a</kbd>
3. Виберіть тип операції (наприклад, "Контур" для різання)
4. Налаштуйте параметри операції:
   - **Потужність**: Відсоток потужності лазера (почніть з низького значення і тестуйте!)
   - **Швидкість**: Швидкість переміщення в мм/хв
   - **Проходи**: Кількість повторень операції (корисно для різання товстих матеріалів)

![Налаштування операції](/screenshots/step-settings-contour-general.png)

:::warning Починайте з низької потужності
При роботі з новими матеріалами завжди починайте з нижчих налаштувань потужності та виконуйте тестові різи. Поступово збільшуйте потужність, поки не досягнете бажаного результату. Використовуйте функцію [Тестова сітка матеріалу](../features/operations/material-test-grid) для систематичного пошуку оптимальних налаштувань.
:::


## Крок 4: Попередній перегляд

Перед відправкою на машину перегляньте траєкторію інструменту в 3D:

1. Натисніть **Вид → 3D перегляд** або натисніть <kbd>ctrl+3</kbd>
2. Вікно 3D-перегляду показує повну траєкторію інструменту
3. Використовуйте мишу для обертання та масштабування перегляду
4. Переконайтеся, що траєкторія виглядає правильно

![3D перегляд](/screenshots/main-3d.png)

:::tip Виявляйте помилки заздалегідь
3D-перегляд допомагає виявити такі проблеми:

- Відсутні траєкторії
- Неправильний порядок
- Операції, застосовані до неправильних об'єктів
- Траєкторії, що виходять за межі робочої області
:::


## Крок 5: Відправка на машину

:::danger Безпека насамперед
- Переконайтеся, що робоча зона вільна
- Ніколи не залишайте машину без нагляду під час роботи
- Майте під рукою засоби пожежогасіння
- Носіть відповідний захист очей
:::


### Підготовка матеріалу

1. Розмістіть ваш матеріал на робочому столі лазера
2. Сфокусуйте лазер згідно з інструкціями вашої машини
3. Якщо використовуєте камеру, вирівняйте ваш дизайн за допомогою [накладки камери](../machine/camera)

### Запуск роботи

1. **Позиціонування лазера**: Використовуйте контролі переміщення, щоб перемістити лазер у початкову позицію
   - Натисніть **Вид → Панель керування** або натисніть <kbd>ctrl+l</kbd>
   - Використовуйте кнопки зі стрілками або стрілки клавіатури для переміщення лазера
   - Натисніть <kbd>home</kbd> для калібрування машини

2. **Обведення дизайну**: Виконайте функцію обведення для перевірки розміщення
   - Натисніть **Машина → Обвести** або натисніть <kbd>ctrl+f</kbd>
   - Лазер обведе обмежувальний прямокутник вашого дизайну з низькою/нульовою потужністю
   - Переконайтеся, що він поміщається на вашому матеріалі

3. **Запуск роботи**: Натисніть **Машина → Запустити роботу** або натисніть <kbd>ctrl+r</kbd>
4. Моніторте прогрес у рядку стану

### Під час роботи

- Права частина рядка стану показує поточний прогрес та оцінку загального часу виконання
- Ви можете призупинити роботу натисканням <kbd>ctrl+p</kbd> або кнопки Пауза
- Натисніть <kbd>esc</kbd> або кнопку Стоп, щоб скасувати роботу (аварійна зупинка)

## Крок 6: Завершення

Після завершення роботи:

1. Дочекайтеся, поки витяжний вентилятор видалить будь-які пари
2. Обережно вийміть готовий виріб
3. Очистіть робочий стіл лазера за необхідності

:::success Вітаємо!
Ви виконали свою першу роботу в Rayforge! Тепер ви можете досліджувати більш розширені функції.
:::


## Наступні кроки

Тепер, коли ви завершили першу роботу, дослідіть ці функції:

- **[Багатошарові операції](../features/multi-layer)**: Призначайте різні операції шарам
- **[Утримуючі лапки](../features/holding-tabs)**: Тримайте вирізані частини на місці під час різання
- **[Інтеграція камери](../machine/camera)**: Використовуйте камеру для точного вирівнювання
- **[Хуки та макроси](../machine/hooks-macros)**: Автоматизуйте повторювані завдання

## Поради для успіху

1. **Зберігайте свою роботу**: Використовуйте <kbd>ctrl+s</kbd> для регулярного збереження проекту
2. **Тестові різи**: Завжди виконуйте тестовий різ на відходах матеріалу спочатку
3. **База даних матеріалів**: Зберігайте нотатки про успішні налаштування потужності/швидкості для різних матеріалів
4. **Технічне обслуговування**: Тримайте лінзу лазера чистою та регулярно перевіряйте натягування ременя
5. **Повітряний ассистент**: Якщо ваша машина має повітряний ассистент, використовуйте його для запобігання обгоранню та покращення якості різу
