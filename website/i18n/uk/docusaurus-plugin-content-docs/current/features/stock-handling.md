# Обробка заготовок

Заготовки в Rayforge представляють фізичний матеріал, який ви будете різати або гравіювати. Заготовки — це **документ-глобальне** поняття: ваш документ може мати один або кілька елементів заготовки, і вони існують незалежно від шарів.

## Додавання заготовки

Заготовка представляє фізичний шматок матеріалу, з яким ви будете працювати. Щоб додати заготовку до вашого документа:

1. На панелі **Матеріал заготовки** у бічній панелі натисніть кнопку **Додати заготовку**
2. Нова заготовка буде створена з розмірами за замовчуванням (80% робочого простору вашої машини)
3. Заготовка з'явиться як прямокутник у робочому просторі, відцентрована на робочому столі машини

### Властивості заготовки

Кожна заготовка має наступні властивості:

- **Назва**: Описова назва для ідентифікації (автоматично нумерується як "Заготовка 1", "Заготовка 2" тощо)
- **Розміри**: Ширина та висота матеріалу заготовки
- **Товщина**: Товщина матеріалу (опціонально, але рекомендується для точного 3D перегляду)
- **Матеріал**: Тип матеріалу (призначається на наступному кроці)
- **Видимість**: Перемикач для показу/приховування заготовки в робочому просторі

### Керування заготовками

- **Перейменувати**: Відкрийте діалог Властивості заготовки та відредагуйте поле назви
- **Змінити розмір**: Виберіть заготовку в робочому просторі та перетягніть кутові маркери для зміни розміру
- **Перемістити**: Виберіть заготовку в робочому просторі та перетягніть для перепозиціонування
- **Видалити**: Натисніть кнопку видалення (іконка кошика) поруч із заготовкою на панелі Матеріал заготовки
- **Редагувати властивості**: Натисніть кнопку властивостей (іконка документа) щоб відкрити діалог Властивості заготовки
- **Перемкнути видимість**: Натисніть кнопку видимості (іконка ока) щоб показати/приховати заготовку

## Призначення матеріалу

Після визначення заготовки ви можете призначити їй матеріал:

1. На панелі **Матеріал заготовки** натисніть кнопку властивостей (іконка документа) на заготовці
2. У діалозі Властивості заготовки натисніть кнопку **Вибрати** поруч із полем Матеріал
3. Перегляньте ваші бібліотеки матеріалів та виберіть відповідний матеріал
4. Заготовка оновиться, щоб показати візуальний вигляд матеріалу

### Властивості матеріалу

Матеріали визначають візуальні властивості вашої заготовки:

- **Візуальний вигляд**: Колір та патерн для візуалізації
- **Категорія**: Групування (наприклад, "Дерево", "Акрил", "Метал")
- **Опис**: Додаткова інформація про матеріал

Примітка: Властивості матеріалу визначаються в бібліотеках матеріалів і не можуть бути відредаговані через діалог властивостей заготовки. Властивості заготовки дозволяють лише призначити матеріал заготовці.

## Перетворення заготовок на заготовки

Ви можете перетворити будь-яку заготовку на елемент заготовки. Це корисно, коли у вас є шматок матеріалу неправильної форми, і ви хочете використати його точний контур як межу заготовки.

Щоб перетворити заготовку на заготовку:

1. Клацніть правою кнопкою миші на заготовці на полотні або на панелі Документа
2. Виберіть **Перетворити на заготовку** з контекстного меню
3. Заготовка буде замінена новим елементом заготовки з тією ж формою та позицією

Новий елемент заготовки:

- Використовує геометрію заготовки як свою межу
- Наслідує назву заготовки
- Може мати призначений матеріал, як і будь-який інший елемент заготовки

## Автоматичне компонування

Функція автоматичного компонування допомагає вам ефективно розмістити елементи дизайну в межах заготовки:

1. Виберіть елементи, які ви хочете розмістити (або залиште нічого не вибраним для розміщення всіх елементів в активному шарі)
2. Натисніть кнопку **Розмістити** на панелі інструментів та виберіть **Автоматичне компонування (пакетувати заготовки)**
3. Rayforge автоматично розмістить елементи для оптимізації використання матеріалу

### Поведінка автоматичного компонування

Алгоритм автоматичного компонування розміщує елементи в межах видимих елементів заготовки у вашому документі:

- **Якщо елементи заготовки визначені**: Елементи розміщуються в межах видимих елементів заготовки
- **Якщо заготовка не визначена**: Елементи розміщуються по всьому робочому простору машини

Алгоритм враховує:

- **Межі елементів**: Поважає розміри кожного елемента дизайну
- **Обертання**: Може обертати елементи з кроком 90 градусів для кращого розміщення
- **Інтервал**: Підтримує відступ між елементами (за замовчуванням 0.5мм)
- **Межі заготовки**: Тримає всі елементи в визначених межах

### Альтернативи ручного компонування

Якщо ви віддаєте перевагу більшому контролю, Rayforge також пропонує інструменти ручного компонування:

- **Інструменти вирівнювання**: Вирівняти вліво, вправо, по центру, вгору, вниз
- **Інструменти розподілу**: Розподілити елементи горизонтально або вертикально
- **Індивідуальне позиціонування**: Клацніть та перетягніть елементи для розміщення вручну

## Поради для ефективної обробки заготовок

1. **Почніть з точних розмірів заготовки** - Виміряйте ваш матеріал точно для найкращих результатів
2. **Використовуйте описові назви** - Називайте ваші заготовки чітко (наприклад, "Березова фанера 3мм")
3. **Встановіть товщину матеріалу** - Це може бути корисно для майбутніх розрахунків та довідки
4. **Призначайте матеріали на початку** - Це забезпечує правильне візуальне представлення з самого початку
5. **Використовуйте заготовки неправильної форми для залишків** - Перетворюйте заготовки на заготовки, коли використовуєте залишковий матеріал з нестандартними формами
6. **Перевірте розміщення перед різанням** - Використовуйте 2D перегляд для перевірки, що все поміщається на вашому матеріалі заготовки

## Виправлення проблем

### Автоматичне компонування не працює як очікувалося

- Переконайтеся, що принаймні один елемент заготовки видимий
- Переконайтеся, що елементи не згруповані (спочатку розгрупуйте їх)
- Спробуйте зменшити кількість вибраних елементів за один раз
- Перевірте, що елементи поміщаються в межі заготовки
