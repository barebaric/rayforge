# 作业完整性检查

在运行或导出作业之前，Rayforge 会自动执行一系列完整性检查，并将结果呈现在结构化的对话框中。这有助于您在问题导致材料浪费之前及早发现它们。

![完整性检查对话框](/screenshots/sanity-check.png)

## 执行的检查

- **机器行程范围违规**：几何图形超出了机器物理上能够到达的范围，按轴和方向分别报告
- **工作区域违规**：工件位于配置的工作区域边界之外
- **禁区碰撞**：刀具路径穿过已启用的禁区

每项检查针对每个唯一的违规最多产生一个问题，使对话框即使在复杂项目中也能保持可读性。对话框区分错误和警告，您可以在决定是否继续之前查看所有内容。
