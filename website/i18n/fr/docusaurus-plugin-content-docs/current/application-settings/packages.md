# Paquets

![Paramètres des Paquets](/screenshots/application-packages.png)

La page Paquets affiche les paquets d'extension installés et vous permet de les gérer.

## Aperçu

Les paquets sont des extensions qui ajoutent des fonctionnalités à Rayforge. Ils peuvent fournir :

- De nouvelles opérations
- Des filtres d'import/export
- Des pilotes de machine
- Des améliorations de l'interface utilisateur
- Des bibliothèques de matériaux

## Paquets Installés

La liste des paquets affiche tous les paquets installés avec :

- **Nom** : Le nom d'affichage du paquet
- **Version** : Version actuellement installée
- **Description** : Brève description de la fonctionnalité
- **Statut** : Activé ou désactivé

## Gérer les Paquets

### Installer des Paquets

1. Cliquez sur le bouton **Installer**
2. Sélectionnez un fichier de paquet (`.zip` ou dépôt de paquet)
3. Confirmez l'installation
4. Attendez que l'installation se termine

### Mettre à Jour les Paquets

Quand des mises à jour sont disponibles :

1. Un indicateur de mise à jour apparaît à côté du paquet
2. Cliquez sur le bouton **Mettre à Jour** pour installer la mise à jour
3. Redémarrez Rayforge si demandé

### Supprimer des Paquets

1. Sélectionnez le paquet à supprimer
2. Cliquez sur le bouton **Supprimer**
3. Confirmez la suppression

:::warning
Supprimer un paquet peut affecter les documents qui utilisent ses fonctionnalités.
:::

## Sources de Paquets

Les paquets peuvent être installés depuis :

- **Fichiers locaux** : Archives `.zip` téléchargées manuellement
- **Dépôts** : Dépôts Git contenant des paquets
- **Registre officiel** : Paquets organisés par l'équipe Rayforge
