# Утримуючі лапки

Утримуючі лапки (також звані мостами або лапками) — це малі нерізані секції, що залишаються вздовж траєкторій різання, які утримують деталі прикріпленими до навколишнього матеріалу. Це запобігає переміщенню вирізаних частин під час роботи, що може спричинити неправильне вирівнювання, пошкодження або пожежонебезпеку.

## Чому використовувати утримуючі лапки?

При різанні наскрізь матеріалу вирізана деталь може:

- **Змістити позицію** під час роботи, спричиняючи неправильне вирівнювання наступних операцій
- **Впати крізь** решітку робочого столу або нахилитися, якщо підтримується тільки на краях
- **Зіткнутися з** лазерною головкою під час її руху
- **Загорітися** якщо впаде на гаряче сміття знизу
- **Бути пошкодженою** від падіння або вібрації

Утримуючі лапки вирішують ці проблеми, утримуючи деталь прикріпленою, поки ви не будете готові її видалити.

---

## Як працюють утримуючі лапки

Rayforge реалізує лапки шляхом створення **малих розривів у траєкторії різання**:

1. Ви позначаєте позиції вздовж траєкторії різання, де мають бути лапки
2. Під час генерації G-коду Rayforge перериває різання на кожній лапці
3. Лазер піднімається (або вимикається), пропускає ширину лапки, потім відновлює різання
4. Після завершення роботи ви вручну ламаєте або відрізаєте лапки, щоб звільнити деталь

---

## Додавання утримуючих лапок

### Швидке додавання

1. **Виберіть заготовку**, до якої ви хочете додати лапки (має бути операція різання/контуру)
2. **Натисніть інструмент лапок** на панелі інструментів або натисніть гарячу клавішу лапок
3. **Клацніть на траєкторії**, де ви хочете лапки:
   - Лапки з'являються як малі маркери на контурі траєкторії
   - Клацніть кілька разів, щоб додати більше лапок
   - Зазвичай 3-4 лапки для малих деталей, більше для більших частин
4. **Увімкніть лапки**, якщо ще не увімкнено (перемикач на панелі властивостей)

### Використання спливаючого вікна додавання лапок

Для більшого контролю:

1. **Клацніть правою кнопкою** на заготовці або використовуйте **Редагування → Додати лапки**
2. **Виберіть метод розміщення лапок:**
   - **Вручну:** Клацніть окремі місця
   - **Еквідистантно:** Автоматично розмістити лапки рівномірно навколо траєкторії
3. **Налаштуйте параметри лапок:**
   - **Кількість лапок:** Скільки лапок створити (для еквідистантних)
   - **Ширина лапки:** Довжина кожної нерізаної секції (зазвичай 2-5мм)
4. **Натисніть Застосувати**

## Властивості лапок

### Ширина лапки

**Ширина** — це довжина нерізаної секції вздовж траєкторії.

**Рекомендовані ширини:**

| Матеріал      | Товщина | Ширина лапки |
| ------------- | ------- | ------------ |
| **Картон**    | 1-3мм   | 2-3мм        |
| **Фанера**    | 3мм     | 3-4мм        |
| **Фанера**    | 6мм     | 4-6мм        |
| **Акрил**     | 3мм     | 2-3мм        |
| **Акрил**     | 6мм     | 3-5мм        |
| **МДФ**       | 3мм     | 3-4мм        |
| **МДФ**       | 6мм     | 5-7мм        |

**Настанови:**
- **Товстіші матеріали** потребують ширших лапок для міцності
- **Важчі деталі** потребують більше та/або ширших лапок
- **Крихкі матеріали** (акрил) можуть використовувати менші лапки (легше ламаються)
- **Волокнисті матеріали** (дерево) можуть потребувати ширших лапок

:::warning Ширина лапки проти товщини матеріалу
Лапки мають бути достатньо широкими, щоб підтримувати деталь, але достатньо малими, щоб чисто видалити. Занадто вузькі = деталь може відірватися; занадто широкі = важко видалити або пошкоджує деталь.
:::

### Позиція лапки

Лапки позиціонуються з використанням двох параметрів:

- **Індекс сегмента:** Який лінійний/дуговий сегмент траєкторії
- **Позиція (0.0 - 1.0):** Де вздовж цього сегмента (0 = початок, 1 = кінець)

**Поради щодо ручного розміщення:**
- Розміщуйте лапки на **прямих секціях** коли можливо (легше видалити)
- Уникайте лапок на **крутых вигинах** (концентрація напруги)
- Розподіляйте лапки **рівномірно** навколо деталі
- Розміщуйте лапки на **кутах** для максимальної підтримки за потреби

### Еквідистантні лапки

Функція **еквідистантності** автоматично розміщує лапки на рівних інтервалах:

**Переваги:**
- Рівномірний розподіл ваги
- Передбачуваний патерн ламання
- Швидке налаштування для регулярних форм

## Робота з лапками

### Редагування лапок

**Перемістити лапку:**
1. Виберіть заготовку
2. Перетягніть маркер лапки вздовж траєкторії
3. Відпустіть, щоб встановити нову позицію

**Змінити розмір лапки:**
- Використовуйте панель властивостей для налаштування ширини
- Всі лапки на заготовці поділяють однакову ширину

**Видалити лапку:**
1. Клацніть маркер лапки, щоб вибрати її
2. Натисніть Delete або використовуйте контекстне меню
3. Або очистіть всі лапки і почніть спочатку

### Увімкнення/Вимкнення лапок

Перемикайте лапки вкл/викл без їх видалення:

- **Панель властивостей заготовки:** Чекбокс "Увімкнути лапки"
- **Панель інструментів:** Іконка видимості лапок

**Коли вимкнено:**
- Лапки не генеруються в G-коді
- Маркери лапок приховані на полотні
- Траєкторія різає повністю наскрізь

**Випадок використання:** Тимчасово вимкніть лапки для тестування різання, потім знову увімкніть для виробництва.

## Видалення лапок після різання

**Інструменти:**
- Канцелярський ніж або різак
- Плоскогубці з плоскими губками
- Стамеска (для дерева)
- Тонка пилка для товстих матеріалів

**Техніка:**

1. **Надріжте лапку** з обох сторін, якщо доступно
2. **Обережно зігніть** деталь, щоб напружити лапку
3. **Проріжте** залишений матеріал
4. **Відшліфуйте або відфайлйте** залишок лапки запліч з краєм

**Для крихких матеріалів (акрил):**
- Використовуйте мінімум лапок (вони легко ламаються)
- Надріжте глибоко перед ламанням
- Підтримуйте деталь під час ламання лапок, щоб уникнути тріщин

**Для дерева:**
- Лапки можуть потребувати різання (не ламаються чисто)
- Використовуйте гострий ніж або стамеску
- Ріжте запліч, потім відшліфуйте гладко
