# Імпорт файлів

Rayforge підтримує імпорт різних форматів файлів, як векторних, так і растрових. Ця сторінка пояснює,
як імпортувати файли та оптимізувати їх для найкращих результатів.

## Підтримувані формати файлів

### Векторні формати

| Формат    | Розширення | Метод імпорту           | Найкраще для                        |
| --------- | --------- | ----------------------- | ------------------------------- |
| **SVG**   | `.svg`    | Прямі вектори або трасування | Векторна графіка, логотипи, дизайни |
| **DXF**   | `.dxf`    | Прямі вектори          | CAD- креслення, технічні дизайни |
| **PDF**   | `.pdf`    | Рендеринг та трасування        | Документи з векторним вмістом   |
| **Ruida** | `.rd`     | Прямі вектори          | Файли завдань контролера Ruida      |

### Растрові формати

| Формат   | Розширення       | Метод імпорту    | Найкраще для                         |
| -------- | --------------- | ---------------- | -------------------------------- |
| **PNG**  | `.png`          | Трасування у вектори | Фото, зображення з прозорістю |
| **JPEG** | `.jpg`, `.jpeg` | Трасування у вектори | Фото, зображення з суцільними тонами   |
| **BMP**  | `.bmp`          | Трасування у вектори | Проста графіка, знімки екрану     |

:::примітка Імпорт растру
:::

Всі растрові зображення **трасуються** для створення векторних шляхів, які можна використовувати для лазерних операцій. Якість залежить від конфігурації трасування.

## Імпорт файлів

### Діалог імпорту

Rayforge має уніфікований діалог імпорту, який забезпечує попередній перегляд у реальному часі та
параметри конфігурації для всіх підтримуваних типів файлів. Діалог дозволяє:

- **Попередній перегляд імпорту** перед додаванням до документа
- **Налаштування параметрів трасування** для растрових зображень
- **Вибір методу імпорту** для файлів SVG (прямі вектори або трасування)
- **Регулювання параметрів** як поріг, інверсія та авто-поріг

![Діалог імпорту](/screenshots/import-dialog.png)

### Метод 1: Меню "Файл"

1. **Файл Імпорт** (або Ctrl+I)
2. **Виберіть ваш файл** у діалозі вибору файлу
3. **Налаштуйте параметри імпорту** в діалозі імпорту
4. **Перегляньте** результат перед імпортом
5. **Натисніть "Імпортувати"** для додавання на полотно та в дерево документа

### Метод 2: Перетягування

1. **Перетягніть файл** з вашого файлового менеджера
2. **Опустіть на** полотно Rayforge
3. **Налаштуйте параметри імпорту** в діалозі імпорту
4. **Перегляньте** результат перед імпортом
5. **Натисніть "Імпортувати"** для додавання на полотно та в дерево документа

### Метод 3: Командний рядок

```bash
# Відкрити Rayforge з файлом
rayforge myfile.svg

# Кілька файлів
rayforge file1.svg file2.dxf
```

### Авто-зміна розміру при імпорті

При імпорті файлів, які більші за робочу область вашої машини, Rayforge
автоматично:

1. **Зменшує масштаб** імпортованого вмісту, щоб він помістився в межах машини
2. **Зберігає співвідношення сторін** під час масштабування
3. **Центрує** масштабований вміст у робочому просторі
4. **Показує сповіщення** з можливістю скасувати зміну розміру

Сповіщення про зміну розміру з'являється як тостове повідомлення:

- ⚠️ "Імпортований елемент був більшим за робочу область і був зменшений до відповідного розміру."
- Включає кнопку **"Скинути"** для скасування авто-зміни розміру
- Тост залишається видимим, доки не буде закритий або не буде виконано скидання

Це забезпечує, що ваші дизайни завжди поміщаються в межах можливостей вашої машини, даючи вам
гнучкість відновити початковий розмір за потреби.

## Імпорт SVG

SVG (Scalable Vector Graphics) - це **рекомендований формат** для векторних дизайнів.

### Параметри імпорту в діалозі

При імпорті SVG діалог імпорту надає перемикач для вибору
між двома методами:

#### 1. Використати оригінальні вектори (Рекомендовано)

Ця опція увімкнена за замовчуванням у діалозі імпорту.

**Як це працює:**

- Парсить SVG та конвертує шляхи безпосередньо в геометрію Rayforge
- Висока точність збереження кривих та форм
- Зберігає точні векторні дані

**Переваги:**

- Найкраща якість та точність
- Редаговані шляхи
- Малий розмір файлу

**Недоліки:**

- Деякі розширені функції SVG не підтримуються
- Складні SVG можуть мати проблеми

**Використовувати для:**

- Чистих векторних дизайнів з Inkscape, Illustrator
- Від простої до середньої складності
- Дизайнів без розширених функцій SVG

#### 2. Трасування растру

Вимкніть "Використати оригінальні вектори" для використання цього методу.

**Як це працює:**

- Спочатку рендерить SVG у растрове зображення
- Трасує відрендероване зображення для створення векторів
- Більш сумісний, але менш точний

**Переваги:**

- Обробляє складні функції SVG
- Надійний резервний метод
- Підтримує ефекти та фільтри

**Недоліки:**

- Втрата якості від растеризації
- Більші розміри файлів
- Менш точно

**Використовувати для:**

- SVG, які не вдається імпортувати напряму
- SVG з ефектами, фільтрами, градієнтами
- Коли прямий імпорт видає помилки

### Попередній перегляд у реальному часі

Діалог імпорту показує попередній перегляд у реальному часі того, як ваш SVG буде імпортовано:

- Векторні шляхи відображаються синім накладанням
- Для режиму трасування оригінальне зображення показується з векторизованими шляхами
- Попередній перегляд оновлюється в реальному часі при зміні налаштувань

### Найкращі практики SVG

**Підготуйте ваш SVG для найкращих результатів:**

1. **Конвертуйте текст у шляхи:**

    - Inkscape: `Шлях → Об'єкт у шлях`
    - Illustrator: `Type → Create Outlines`

2. **Спрощуйте складні шляхи:**

    - Inkscape: `Шлях → Спростити` (Ctrl+L)
    - Видаліть непотрібні вузли

3. **Розгрупуйте вкладені групи:**

    - Сплощуйте ієрархію де можливо
    - `Об'єкт → Розгрупувати` (Ctrl+Shift+G)

4. **Видаліть приховані елементи:**

    - Видаліть напрямні, сітки, конструкційні лінії
    - Видаліть невидимі/прозорі об'єкти

5. **Збережіть як Plain SVG:**

    - Inkscape: "Plain SVG" або "Optimized SVG"
    - Не "Inkscape SVG" (має зайві метадані)

6. **Перевірте одиниці документа:**
    - Встановіть у мм або дюйми за потреби
    - Rayforge використовує мм внутрішньо

**Поширені функції SVG, які можуть не імпортуватися:**

- Градієнти (конвертуйте у суцільні заливки або растр)
- Фільтри та ефекти (сплощуйте у шляхи)
- Маски та шляхи обрізки (розгорніть/сплощіть)
- Вбудовані растрові зображення (експортуйте окремо)
- Текст (спочатку конвертуйте у шляхи)

## Імпорт DXF

DXF (Drawing Exchange Format) поширений для CAD-програмного забезпечення.

### Версії DXF

Rayforge підтримує стандартні формати DXF:

- **R12/LT2** (рекомендовано) - Найкраща сумісність
- **R13, R14** - Гарна підтримка
- **R2000+** - Зазвичай працює, але R12 безпечніше

**Порада:** Експортуйте як R12/LT2 DXF для максимальної сумісності.

### Поради з імпорту DXF

**Перед експортом з CAD:**

1. **Спростіть креслення:**

    - Видаліть непотрібні шари
    - Видаліть розміри та анотації
    - Видаліть 3D-об'єкти (використовуйте 2D-проекцію)

2. **Перевірте одиниці:**

    - Перевірте одиниці креслення (мм проти дюймів)
    - Rayforge припускає мм за замовчуванням

3. **Сплощіть шари:**

    - Розгляньте експорт лише релевантних шарів
    - Приховайте або видаліть конструкційні шари

4. **Використовуйте відповідну точність:**
    - Точність лазера зазвичай 0.1мм
    - Не перевищуйте точність

**Після імпорту:**

- Перевірте масштаб (одиниці DXF можуть потребувати коригування)
- Переконайтеся, що всі шляхи імпортовано правильно
- Видаліть будь-які непажитані конструкційні елементи

## Імпорт PDF

Файли PDF можуть містити векторну графіку, растрові зображення або обидва.

### Як працює імпорт PDF

При імпорті файлів PDF через діалог імпорту Rayforge **рендерить PDF**
у зображення, потім **трасує** його для створення векторів.

**Процес:**

1. PDF рендериться та відображається у попередньому перегляді діалогу імпорту
2. Ви можете регулювати налаштування трасування в реальному часі
3. Відрендероване зображення трасується з вашими налаштуваннями
4. Отримані шляхи додаються до документа при натисканні "Імпортувати"

**Обмеження:**

- Текст растеризується (не редагується як шляхи)
- Векторна якість залежить від DPI рендерингу
- Багатосторінкові PDF: імпортується лише перша сторінка

### Поради з імпорту PDF

**Найкращі результати:**

1. **Використовуйте векторні PDF:**

    - PDF, створені у векторному ПЗ (Illustrator, Inkscape)
    - Не скановані документи або вбудовані зображення

2. **Експортуйте SVG натомість, якщо можливо:**

    - Більшість дизайнерського ПЗ може експортувати SVG напряму
    - SVG матиме кращу якість, ніж імпорт PDF

3. **Для документів з текстом:**

    - Експортуйте як SVG зі шрифтами, конвертованими у шляхи
    - Або рендеріть PDF з високим DPI (600+) і трасуйте

4. **Використовуйте попередній перегляд діалогу імпорту:**
    - Регулюйте налаштування порогу та інверсії для найкращих результатів
    - Попередній перегляд показує точно, як PDF буде векторизовано

## Імпорт Ruida

Файли Ruida (.rd) - це власні двійкові файли завдань, що використовуються контролерами Ruida у багатьох
лазерних різальних машинах. Ці файли містять як векторну геометрію, так і налаштування лазера,
організовані у шари (кольори).

**Після імпорту:**

- **Перевірте масштаб** - Переконайтеся, що розміри відповідають очікуваним
- **Перегляньте шари** - Переконайтеся, що всі шари імпортовано правильно
- **Валідуйте шляхи** - Підтвердіть, що всі шляхи різання присутні

### Обмеження

- **Імпорт лише для читання** - Файли Ruida можна лише імпортувати, не експортувати
- **Двійковий формат** - Пряме редагування оригінальних файлів .rd не підтримується
- **Власні функції** - Деякі розширені функції Ruida можуть не підтримуватися повністю

## Імпорт растрових зображень (PNG, JPG, BMP)

Растрові зображення **трасуються** для створення векторних шляхів за допомогою діалогу імпорту.

### Процес трасування в діалозі

**Як це працює:**

1. **Зображення завантажується** у діалог імпорту
2. **Попередній перегляд у реальному часі** показує результат трасування
3. **Налаштування трасування** можна регулювати в реальному часі
4. **Векторні шляхи створюються** з векторизованих країв
5. **Шляхи додаються** до документа як деталі при імпорті

### Конфігурація трасування в діалозі

Діалог імпорту надає ці регульовані параметри:

| Параметр          | Опис         | Ефект                                              |
| ------------------ | ------------------- | --------------------------------------------------- |
| **Авто-поріг** | Автоматичне виявлення | Коли увімкнено, автоматично знаходить оптимальний поріг |
| **Поріг**      | межа чорне/біле  | Нижче = більше деталей, вище = простіше               |
| **Інверсія**         | Зворотні кольори      | Трасувати світлі об'єкти на темному фоні              |

**Налаштування за замовчуванням** добре працюють для більшості зображень. Діалог показує попередній перегляд у реальному часі,
який оновлюється при регулюванні цих налаштувань, дозволяючи точно налаштувати трасування
перед імпортом.

### Підготовка зображень до трасування

**Для найкращих результатів:**

1. **Високий контраст:**

    - Регулюйте яскравість/контраст у редакторі зображень
    - Чітке розрізнення між переднім планом та фоном

2. **Чистий фон:**

    - Видаліть шум та артефакти
    - Суцільний білий або прозорий фон

3. **Відповідна роздільна здатність:**

    - 300-500 DPI для фото
    - Занадто висока = повільне трасування, занадто низька = погана якість

4. **Обріжте до вмісту:**

    - Видаліть непотрібні межі
    - Фокусуйтеся на області для гравірування/різання

5. **Конвертуйте у чорно-біле:**
    - Для різання: чисте чорно-біле
    - Для гравірування: градації сірого підходять

**Інструменти редагування зображень:**

- GIMP (безкоштовний)
- Photoshop
- Krita (безкоштовний)
- Paint.NET (безкоштовний, Windows)

### Якість трасування

**Добрі кандидати для трасування:**

- Логотипи з чіткими краями
- Висококонтрастні зображення
- Лінійна графіка та малюнки
- Текст (хоча краще як вектор)

**Погані кандидати для трасування:**

- Низькороздільні зображення
- Фото з м'якими краями
- Зображення з градієнтами
- Дуже деталізовані або складні фото
