# Налаштування лазера

Сторінка "Лазер" у налаштуваннях машини налаштовує ваші лазерні головки та їхні
властивості.

![Налаштування лазера](/screenshots/machine-laser.png)

## Лазерні головки

Rayforge підтримує машини з кількома лазерними головками. Кожна лазерна головка
має власну конфігурацію.

### Додавання лазерної головки

Натисніть кнопку **Додати лазер**, щоб створити нову конфігурацію лазерної
головки.

### Властивості лазерної головки

Кожна лазерна головка має наступні налаштування:

#### Назва

Описова назва для цієї лазерної головки.

Приклади:
- "10W Діод"
- "CO2 Трубка"
- "Інфрачервоний лазер"

#### Номер інструменту

Індекс інструменту для цієї лазерної головки. Використовується в G-code з
командою T.

- Машини з однією головкою: Використовуйте 0
- Машини з кількома головками: Призначте унікальні номери (0, 1, 2 тощо)

#### Максимальна потужність

Максимальне значення потужності для вашого лазера.

- **Типово для GRBL**: 1000 (діапазон S0-S1000)
- **Деякі контролери**: 255 (діапазон S0-S255)
- **Режим відсотків**: 100 (діапазон S0-S100)

Це значення має відповідати налаштуванню $30 вашої прошивки.

#### Потужність обведення

Рівень потужності, що використовується для операцій обведення (контур без
різання).

- Встановіть на 0, щоб вимкнути обведення
- Налаштуйте відповідно до вашого лазера та матеріалу

#### Швидкість обведення

Швидкість, з якою лазерна головка рухається під час обведення. Встановлюється
окремо для кожної лазерної головки, тому якщо ваша машина має кілька лазерів з
різними характеристиками, ви можете обрати відповідну швидкість для кожного.
Повільніші швидкості полегшують візуальне відстеження шляху обведення.

#### Потужність фокусу

Рівень потужності, що використовується коли увімкнено режим фокусу. Режим
фокусу вмикає лазер на низькій потужності, щоб діяти як "лазерний вказівник"
для позиціонування.

- Встановіть на 0, щоб вимкнути функцію режиму фокусу
- Використовуйте для візуального вирівнювання та позиціонування

:::tip Використання режиму фокусу
Натисніть кнопку фокусу (іконка лазера) на панелі інструментів, щоб
перемкнути режим фокусу. Лазер увімкнеться на цьому рівні потужності,
допомагаючи вам точно бачити, де позиціоновано лазер. Дивіться
[Позиціонування заготовки](../features/workpiece-positioning) для
отримання додаткової інформації.
:::

#### Розмір плями

Фізичний розмір сфокусованого променя вашого лазера в міліметрах.

- Введіть розміри X та Y
- Більшість лазерів мають круглу пляму (наприклад, 0.1 x 0.1)
- Впливає на розрахунки якості гравірування

:::tip Вимірювання розміру плями
Для вимірювання розміру плями:
1. Випустіть короткий імпульс на низькій потужності на тестовому матеріалі
2. Виміряйте отриману позначку штангенциркулем
3. Використовуйте середнє з кількох вимірювань
:::

#### Колір різання

Колір, що використовується для відображення операцій різання для цього лазера
на полотні та в 3D перегляді. Це допомагає візуально розрізняти, який лазер
виконуватиме кожну операцію різання при роботі з кількома лазерними головками.

- Натисніть на зразок кольору, щоб відкрити палітру кольорів
- Виберіть колір, який добре контрастує з попереднім переглядом матеріалу
- Кольори за замовчуванням призначаються автоматично

#### Колір растра

Колір, що використовується для відображення операцій растра/гравірування для
цього лазера на полотні та в 3D перегляді.

- Натисніть на зразок кольору, щоб відкрити палітру кольорів
- Корисно для розрізнення операцій растера від різань
- Кожен лазер може мати свій власний колір растра

:::tip Багатолазерні робочі процеси
При використанні кількох лазерних головок призначення різних кольорів кожному
лазеру полегшує перегляд того, які операції будуть виконуватися яким лазером.
Наприклад, використовуйте червоний для основного лазера різання та синій для
вторинного лазера гравірування.
:::
