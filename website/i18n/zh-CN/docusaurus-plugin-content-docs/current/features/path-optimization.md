# 路径优化

路径优化重新排序切割段以最小化行程距离。激光在切割之间高效移动，而不是在工作区域随机跳跃。

## 工作原理

没有优化时，路径按设计文件中出现的顺序切割。优化分析所有路径段并重新排列它们，使激光在切割之间行进的总距离最短。

**优化前：** 激光在材料上来回跳跃
**优化后：** 激光从一个切割顺序移动到下一个

## 设置

### 启用优化

开启或关闭路径优化。大多数操作默认启用。

## 何时使用优化

**启用用于：**

- 有许多单独形状的设计
- 减少总作业时间
- 最小化运动系统的磨损
- 复杂的嵌套布局

**禁用用于：**

- 切割顺序很重要的设计（例如，内部特征在外部之前）
- 调试路径问题
- 当您需要可预测、可重复的执行顺序时

## 如何影响您的作业

**时间节省：** 对于有许多单独切割的设计，可减少 20-50% 的作业时间。

**运动效率：** 较少的快速移动意味着皮带、电机和轴承的磨损更少。

**热量分布：** 优化的路径可能会在一个区域集中热量。对于热敏感材料，请考虑顺序是否重要。

:::tip
优化自动运行。只需启用它，软件会处理其余的工作。
:::
