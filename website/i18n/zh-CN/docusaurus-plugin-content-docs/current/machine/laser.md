# 激光设置

机器设置中的激光页面配置您的激光头及其属性。

![激光设置](/screenshots/machine-laser.png)

## 激光头

Rayforge 支持具有多个激光头的机器。每个激光头都有自己的配置。

### 添加激光头

点击 **添加激光** 按钮创建新的激光头配置。

### 激光头属性

每个激光头具有以下设置：

#### 名称

此激光头的描述性名称。

示例：
- "10W 二极管"
- "CO2 管"
- "红外激光"

#### 工具编号

此激光头的工具索引。在 G 代码中与 T 命令一起使用。

- 单头机器：使用 0
- 多头机器：分配唯一编号（0、1、2 等）

#### 最大功率

激光的最大功率值。

- **GRBL 典型**：1000（S0-S1000 范围）
- **某些控制器**：255（S0-S255 范围）
- **百分比模式**：100（S0-S100 范围）

此值应与您的固件 $30 设置匹配。

#### 框选功率

用于框选操作的功率级别（描绘轮廓而不切割）。

- 设置为 0 以禁用框选
- 典型值：5-20（刚好可见，不会标记材料）
- 根据您的激光和材料调整

#### 光斑大小

聚焦激光束的物理大小，以毫米为单位。

- 输入 X 和 Y 尺寸
- 大多数激光具有圆形光斑（例如，0.1 x 0.1）
- 影响雕刻质量计算

:::tip 测量光斑大小
要测量光斑大小：
1. 在测试材料上以低功率发射短脉冲
2. 用卡尺测量产生的标记
3. 使用多次测量的平均值
:::
