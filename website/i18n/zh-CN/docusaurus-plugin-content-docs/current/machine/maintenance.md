# 维护

机器设置中的维护页面帮助您跟踪机器使用情况并安排维护任务。

![维护设置](/screenshots/machine-maintenance.png)

## 使用跟踪

Rayforge 跟踪您的机器已使用多长时间。此信息帮助您在适当的间隔安排预防性维护。

### 总小时数

总小时数计数器跟踪在机器上运行作业的所有时间。此累积计数器无法重置，并提供机器使用的完整历史。

使用此来跟踪机器整体年龄并计划主要维护间隔。

## 自定义维护计数器

您可以创建自定义计数器来跟踪特定的维护间隔。每个计数器都有一个名称、跟踪小时数，并可以配置通知阈值。

### 创建计数器

1. 点击添加按钮创建新计数器
2. 输入描述性名称（例如，"激光管"、"皮带张力"、"镜片清洁"）
3. 如果需要，设置以小时为单位的通知阈值

### 计数器功能

- **自定义名称**：为任何维护任务标记计数器
- **小时跟踪**：在作业执行期间自动累积时间
- **通知阈值**：当需要维护时获得提醒
- **重置功能**：执行维护后重置计数器

### 示例计数器

**激光管**：跟踪 CO2 管小时数以计划更换（通常 1000-3000 小时）。在 2500 小时设置通知以提前计划。

**皮带张力**：跟踪自上次皮带张紧以来的小时数。执行维护后重置。

**镜片清洁**：跟踪自上次镜片清洁以来的使用情况。清洁后重置。

**轴承润滑**：跟踪轴承维护间隔的小时数。

## 重置计数器

执行维护后，重置相关计数器：

1. 点击计数器旁边的重置按钮
2. 在对话框中确认重置
3. 计数器归零

:::tip 维护计划
常见维护间隔：
- **每天**：清洁透镜，检查镜片对齐
- **每周**：清洁导轨，检查皮带张力
- **每月**：润滑轴承，检查电气连接
- **每年**：全面检查，更换磨损零件

根据您的使用模式和制造商建议调整间隔。
:::
