# 激光设置

机器设置中的激光页面配置您的激光头及其属性。

![激光设置](/screenshots/machine-laser.png)

## 激光头

Rayforge 支持具有多个激光头的机器。每个激光头都有自己的配置。

### 添加激光头

点击 **添加激光** 按钮创建新的激光头配置。

### 激光头属性

每个激光头具有以下设置：

#### 名称

此激光头的描述性名称。

示例：

- "10W 二极管"
- "CO2 管"
- "红外激光"

#### 工具编号

此激光头的工具索引。在 G 代码中与 T 命令一起使用。

- 单头机器：使用 0
- 多头机器：分配唯一编号（0、1、2 等）

#### 最大功率

激光的最大功率值。

- **GRBL 典型**：1000（S0-S1000 范围）
- **某些控制器**：255（S0-S255 范围）
- **百分比模式**：100（S0-S100 范围）

此值应与您的固件 $30 设置匹配。

#### 框选功率

用于框选操作的功率级别（描绘轮廓而不切割）。

- 设置为 0 以禁用框选
- 根据您的激光和材料调整

#### 框选速度

激光头在框选过程中移动的速度。此设置针对每个激光头单独配置，因此如果您的机器配备多个不同特性的激光器，可以为每个激光器选择合适的速度。较慢的速度使框选路径更容易用肉眼追踪。

#### 对焦功率

启用对焦模式时使用的功率级别。对焦模式以低功率开启激光，作为"激光指示器"用于定位。

- 设置为 0 以禁用对焦模式功能
- 用于视觉对齐和定位

:::tip 使用对焦模式
点击工具栏中的对焦按钮（激光图标）以切换对焦模式。激光将在此功率级别下开启，帮助您准确查看激光的位置。有关更多信息，请参阅[工件定位](../features/workpiece-positioning)。
:::

#### 光斑大小

聚焦激光束的物理大小，以毫米为单位。

- 输入 X 和 Y 尺寸
- 大多数激光具有圆形光斑（例如，0.1 x 0.1）
- 影响雕刻质量计算

:::tip 测量光斑大小
要测量光斑大小：
1. 在测试材料上以低功率发射短脉冲
2. 用卡尺测量产生的标记
3. 使用多次测量的平均值
:::

#### 切割颜色

用于在画布和 3D 预览中显示此激光切割操作的颜色。这有助于在使用多个激光头时直观地区分哪个激光将执行哪个切割操作。

- 点击颜色样本打开颜色选择器
- 选择与材料预览对比度好的颜色
- 默认颜色自动分配

#### 栅格颜色

用于在画布和 3D 预览中显示此激光栅格/雕刻操作的颜色。

- 点击颜色样本打开颜色选择器
- 用于区分栅格操作和切割操作
- 每个激光可以有自己的栅格颜色

:::tip 多激光工作流程
使用多个激光头时，为每个激光分配不同的颜色可以轻松查看哪些操作将由哪个激光执行。例如，主切割激光使用红色，次级雕刻激光使用蓝色。
:::

#### 激光类型

从下拉菜单中选择激光头的类型：

- **二极管**：标准二极管激光器（业余机器中最常见）
- **CO2**：CO2 管激光器
- **光纤**：光纤激光器

选择 CO2 或光纤时，会显示额外的 **PWM 设置**（见下文）。
对于二极管激光器，PWM 部分会隐藏，因为不适用。

#### PWM 设置

选择 CO2 或光纤激光类型时，会出现以下 PWM 控件：

- **PWM 频率**：此激光头的默认 PWM 频率（Hz）。
  典型值范围从 500 Hz 到数千 Hz，具体取决于您的控制器和电源。
- **最大 PWM 频率**：频率设置的上限。防止输入硬件无法处理的值。
- **脉冲宽度**：默认脉冲宽度（微秒）。控制每个周期中脉冲开启的时间长度。
- **最小/最大脉冲宽度**：脉冲宽度设置的边界。

这些默认值会传递到您的操作步骤中，如有需要可在每个步骤中覆盖。

#### 3D 模型

每个激光头可以分配一个 3D 模型。此模型在 [3D 视图](../ui/3d-preview) 中渲染，并在仿真过程中沿刀具路径移动。

点击模型选择行以浏览可用模型。选择模型后，您可以调整其比例、旋转（X/Y/Z）和焦距以匹配您的物理激光头。
