# Оверскан

Оверскан розширює лінії растрового гравіювання за фактичну область вмісту, щоб гарантувати, що лазер досягає постійної швидкості під час гравіювання, усуваючи артефакти прискорення.

## Проблема: Відмітини прискорення

Без оверскану растрове гравіювання страждає від **артефактів прискорення**:

- **Світлі краї** де починається прискорення (лазер рухається занадто швидко для рівня потужності)
- **Темні краї** де відбувається уповільнення (лазер затримується довше)
- **Непослідовна глибина/темрява гравіювання** по кожній лінії
- Видима смугастість або смужки на краях ліній

## Як працює оверскан

Оверскан **розширює траєкторію інструменту** до і після кожної растрової лінії:

**Процес:**

1. **Вхід:** Лазер переміщується в позицію _перед_ початком лінії
2. **Прискорення:** Лазер прискорюється до цільової швидкості (лазер ВИМК)
3. **Гравіювання:** Лазер вмикається і гравіює на постійній швидкості
4. **Уповільнення:** Лазер вимикається і уповільнюється _після_ закінчення лінії

**Результат:** Вся гравірована область отримує послідовну потужність на постійній швидкості.

**Переваги:**

- Рівномірна глибина гравіювання по всій растровій лінії
- Немає світлих/темних країв
- Вища якість фотогравіювання
- Професійний вигляд результатів

## Налаштування оверскану

Оверскан — це **трансформер** у конвеєрі робочого процесу Rayforge.

**Щоб увімкнути:**

1. **Виберіть шар** з растровим гравіюванням
2. **Відкрийте налаштування робочого процесу** (або налаштування операції)
3. **Додайте трансформер Оверскан** якщо його ще немає
4. **Налаштуйте відстань**

**Налаштування:**

| Параметр           | Опис                    | Типове значення |
| ------------------ | ----------------------- | --------------- |
| **Увімкнено**      | Перемкнути оверскан вкл/викл | УВІМК (для растра) |
| **Відстань (мм)**  | Наскільки розширити лінії | 2-5 мм          |

## Вибір відстані оверскану

Відстань оверскану повинна дозволити машині **повністю прискоритися** до цільової швидкості.

**Практичні настанови:**

| Макс. швидкість         | Прискорення | Рекомендований оверскан |
| ----------------------- | ----------- | ----------------------- |
| 3000 мм/хв (50 мм/с)    | Низьке      | 5 мм                    |
| 3000 мм/хв (50 мм/с)    | Середнє     | 3 мм                    |
| 3000 мм/хв (50 мм/с)    | Високе      | 2 мм                    |
| 6000 мм/хв (100 мм/с)   | Низьке      | 10 мм                   |
| 6000 мм/хв (100 мм/с)   | Середнє     | 6 мм                    |
| 6000 мм/хв (100 мм/с)   | Високе      | 4 мм                    |

**Фактори, що впливають на необхідну відстань:**

- **Швидкість:** Вища швидкість = потрібно більше відстані для прискорення
- **Прискорення:** Нижче прискорення = потрібно більше відстані
- **Механіка машини:** Ремінна передача проти прямого приводу впливає на прискорення

**Налаштування:**

- **Занадто мало:** Відмітини прискорення все ще видимі на краях
- **Занадто багато:** Витрачає час, може досягти меж машини
- **Почніть з 3мм** та коригуйте на основі результатів

## Тестування налаштувань оверскану

**Процедура тестування:**

1. **Створіть тестове гравіювання:**
   - Суцільний заповнений прямокутник (50мм x 20мм)
   - Використовуйте ваші типові налаштування гравіювання
   - Увімкніть оверскан на 3мм

2. **Вигравіюйте тест:**
   - Запустіть роботу
   - Дозвольте завершити

3. **Огляньте краї:**
   - Подивіться на лівий та правий краї прямокутника
   - Перевірте варіацію темряви на краях
   - Порівняйте темряву країв з темрявою центру

4. **Скоригуйте:**
   - **Якщо краї світліші/темніші:** Збільште оверскан
   - **Якщо краї відповідають центру:** Оверскан достатній
   - **Якщо краї ідеальні:** Спробуйте трохи зменшити оверскан щоб заощадити час

## Коли використовувати оверскан

**Завжди використовуйте для:**

- Фотогравіювання (растрове)
- Патерни заповнення
- Будь-яка високо-детальна растрова робота
- Гравіювання зображень у градаціях сірого
- Гравіювання тексту (растровий режим)

**Опціонально для:**

- Векторне різання (не потрібне)
- Дуже повільне гравіювання (прискорення менш помітне)
- Великі прості форми (краї менш критичні)

**Вимкніть для:**

- Векторних операцій
- Дуже малих робочих областей (може перевищити межі)
- Коли якість країв не важлива
