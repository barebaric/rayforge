# 设置

![常规设置](/screenshots/application-general.png)

自定义 Rayforge 以匹配您的工作流程和偏好。

## 访问设置

- **菜单**：编辑 → 设置
- **键盘**：<kbd>ctrl+comma</kbd>

## 常规

常规页面包含应用程序范围设置。

### 外观

- **主题**：在系统、浅色或深色主题之间选择

### 单位

配置整个应用程序中各种值的显示单位：

- **长度**：毫米、英寸和其他长度单位
- **速度**：mm/min、mm/sec、inches/min 和其他速度单位
- **加速度**：mm/s² 和其他加速度单位

### 启动

配置应用程序启动时发生的事情。命令行指定的文件将始终覆盖这些设置。

- **启动行为**：
  - 不打开任何内容
  - 打开上次的项目
  - 打开特定项目
- **项目路径**：启动时打开的特定项目路径
  （仅当选择"打开特定项目"时可见）

## 机器

机器页面允许您管理机器配置。

- **添加机器**：创建新的机器配置
- **移除机器**：删除选定的机器
- **机器列表**：显示所有配置的机器及其连接状态

有关详细的机器配置，请参阅[机器设置](../machine/general)部分。

## 材料

材料页面管理您的材料库。

- **添加材料**：创建新的材料预设
- **导入库**：从文件导入材料
- **导出库**：导出您的材料以备份或共享

有关详细信息，请参阅[材料](../application-settings/materials)。

## 配方

配方页面管理您的操作配方。

- **添加配方**：创建新配方
- **导入配方**：从文件导入配方
- **导出配方**：导出您的配方以备份或共享

有关详细信息，请参阅[配方](../application-settings/recipes)。

## 包

包页面显示已安装的扩展包。

- **包列表**：显示所有已安装的包
- **包详情**：查看有关每个包的信息
