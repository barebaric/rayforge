# 材料

![材料设置](/screenshots/application-materials.png)

Rayforge 中的材料库允许您组织和管理激光切割和雕刻项目的材料集合。本指南解释核心库和用户库之间的区别，以及如何创建自己的库和向其添加材料。

:::note
材料库目前没有功能用途 - 分配材料仅影响美观外观。在未来的版本中，材料将用于派生功能参数。
:::


## 创建新库

要创建自己的材料库：

1. 打开 **设置** 菜单并选择 **材料**
2. 点击 **添加新库** 按钮创建新库
3. 为库输入描述性名称（例如，"我的车间材料"）
4. 点击 **创建** 完成

您的新库将在用户数据目录中创建，并立即可用。


## 向库添加材料

### 创建新材料

1. 选择要添加材料的库
2. 点击材料列表中的 **添加新材料** 按钮
3. 填写材料属性：
   - **名称**：人类可读的名称
   - **类别**：分组类别（例如，"木材"、"亚克力"）
   - **外观**：视觉属性（颜色）
4. 点击 **保存** 将材料添加到库

### 材料属性说明

#### 名称
- 界面中显示的人类可读名称
- 可以包含空格和特殊字符

#### 类别
- 用于在库内组织材料
- 常见类别包括：木材、亚克力、金属、纸张、皮革
- 您可以根据需要创建自定义类别

#### 颜色

颜色仅用于工作表面上的视觉外观 - 它不会以任何方式影响激光路径。


## 管理现有材料

### 编辑材料

1. 选择要编辑的材料
2. 点击 **编辑** 按钮
3. 修改所需的属性
4. 点击 **保存** 应用更改

### 删除材料

1. 选择要删除的材料
2. 点击 **删除** 按钮
3. 在对话框中确认删除

:::warning
删除材料是永久性的，无法撤销。
:::
