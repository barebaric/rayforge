# Пакети

![Налаштування пакетів](/screenshots/application-packages.png)

Сторінка "Пакети" показує встановлені пакети розширень та дозволяє
керувати ними.

## Огляд

Пакети - це розширення, які додають функціональність до Rayforge. Вони можуть
надавати:

- Нові операції
- Фільтри імпорту/експорту
- Драйвери машин
- Покращення інтерфейсу
- Бібліотеки матеріалів

## Встановлені пакети

Список пакетів показує всі встановлені пакети з:

- **Назва**: Відображуване ім'я пакету
- **Версія**: Поточна встановлена версія
- **Опис**: Короткий опис функціональності
- **Статус**: Увімкнено або вимкнено

## Керування пакетами

### Встановлення пакетів

1. Натисніть кнопку **Встановити**
2. Виберіть файл пакету (`.zip` або репозиторій пакетів)
3. Підтвердьте встановлення
4. Дочекайтеся завершення встановлення

### Оновлення пакетів

Коли доступні оновлення:

1. Індикатор оновлення з'являється поруч з пакетом
2. Натисніть кнопку **Оновити** для встановлення оновлення
3. Перезапустіть Rayforge, якщо з'явиться запит

### Видалення пакетів

1. Виберіть пакет для видалення
2. Натисніть кнопку **Видалити**
3. Підтвердьте видалення

:::увага
Видалення пакету може вплинути на документи, які використовують його функції.
:::

## Джерела пакетів

Пакети можна встановити з:

- **Локальні файли**: `.zip` архіви, завантажені вручну
- **Репозиторії**: Git-репозиторії, що містять пакети
- **Офіційний реєстр**: Кураторські пакети від команди Rayforge
