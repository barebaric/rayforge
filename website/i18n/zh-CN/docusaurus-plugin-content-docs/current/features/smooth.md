# 平滑路径

路径平滑可减少切割路径中的锯齿状边缘和尖锐过渡，从而产生更平滑的曲线和更流畅的机器运动。

## 工作原理

平滑对路径几何应用滤镜，将棱角圆角化并平滑粗糙边缘。激光遵循更平缓的轨迹，而不是进行急剧的方向变化。

## 设置

### 启用平滑

为此操作开启或关闭平滑。平滑默认禁用。

### 平滑度

控制路径平滑的程度（0-100）。较高的值产生更圆的曲线，但可能偏离原始路径更多。

- **低（0-30）：** 最小平滑，保留锐利细节
- **中（30-60）：** 平衡平滑，适用于大多数设计
- **高（60-100）：** 激进平滑，最适用于有机形状

### 角度阈值

比此值更尖锐的角度保留为角而不是被平滑（0-179 度）。这可防止重要的尖锐特征被圆角化。

- **较低的值：** 更多角被平滑，结果更圆
- **较高的值：** 更多角被保留，结果更锐利

## 何时使用平滑

**适用于：**

- 从像素来源导入的设计，有阶梯效应
- 减少快速方向变化时的机械应力
- 改善曲线上的切割质量
- 有许多小线段的设计

**不需要用于：**

- 具有平滑贝塞尔曲线的干净矢量图稿
- 必须精确保留锐利角的设计
- 需要精确几何的技术图纸
