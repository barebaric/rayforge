# 框选作业

了解如何使用框选功能预览作业边界并确保切割前正确定位。

## 概述

框选允许您通过以低功率或激光关闭描绘轮廓来预览激光作业的精确边界。这有助于验证定位并防止昂贵的错误。

## 何时使用框选

- **首次设置**：验证材料放置
- **精确定位**：确保设计适合材料边界
- **多个作业**：每次运行前确认对齐
- **昂贵材料**：提交切割前仔细检查

## 如何框选

### 方法 1：仅轮廓

描绘作业边界而不开启激光：

1. 在 Rayforge 中**加载设计**
2. 在激光床上**放置材料**
3. **点击框选按钮**在工具栏中
4. **观察激光头**描绘边界矩形
5. **验证定位**并根据需要调整材料

### 方法 2：低功率预览

某些机器支持带可见光束的低功率框选：

1. 在机器设置中**启用低功率模式**
2. **设置框选功率**（通常 1-5%）
3. **运行框选操作**
4. **观察轮廓**在材料表面描绘

:::warning 检查您的机器
并非所有激光都安全支持低功率框选。使用此功能前请咨询机器文档。
:::


## 框选设置

在设置 → 机器中配置框选行为：

- **框选速度**：框选期间激光头移动的速度
- **框选功率**：框选期间的激光功率（0 表示关闭，低 % 表示可见描绘）
- **角落暂停**：在每个角落短暂暂停以提高可见性
- **重复次数**：描绘轮廓的次数

## 使用框选结果

框选后，您可以：

- 根据需要**调整材料位置**
- **重新框选**以验证新位置
- 一旦对放置满意**继续作业**

## 有效框选的提示

- **标记角落**：在角落放置小胶带作为参考
- **检查间隙**：确保设计周围有足够的空间
- **验证方向**：确认材料方向正确
- **考虑缝隙**：记住切割将比轮廓略宽

## 使用相机框选

如果您的机器有相机支持，您可以：

1. **捕获相机图像**材料放置
2. **叠加设计**在相机视图上
3. **虚拟调整位置**在框选前
4. **框选确认**物理对齐

有关详细信息，请参阅[相机集成](../machine/camera)。

## 故障排除

**框选与设计不匹配**：检查作业原点和坐标系设置

**框选期间激光发射**：禁用框选功率或检查机器设置

**框选太快无法看到**：在设置中降低框选速度

**头无法到达角落**：验证设计在机器工作区域内

## 安全说明

- 框选期间**切勿让机器无人看管**
- 如果使用零功率框选**验证激光关闭**
- **保持手远离**激光头路径
- **注意可能干扰移动的障碍物**
