# Atalhos de Teclado

Referência completa de atalhos de teclado no Rayforge.

:::note Convenções de Plataforma
- **Linux/Windows:** Tecla `Ctrl`
- **macOS:** Tecla `⌘ (Command)`
- Documentação usa `Ctrl` - usuários macOS substituam por `⌘`
:::


---

## Operações de Arquivo

| Atalho       | Ação      | Descrição              |
| -------------- | ----------- | ------------------------ |
| `Ctrl+N`       | Novo Esboço  | Criar novo esboço        |
| `Ctrl+O`       | Abrir        | Abrir projeto existente    |
| `Ctrl+S`       | Salvar        | Salvar projeto atual     |
| `Ctrl+Shift+S` | Salvar Como     | Salvar projeto como novo arquivo |
| `Ctrl+I`       | Importar      | Abrir diálogo de importação de arquivo  |
| `Ctrl+E`       | Exportar      | Exportar G-code            |
| `Ctrl+Q`       | Sair        | Sair do Rayforge            |
| `Ctrl+,`       | Preferências | Abrir diálogo de preferências  |

## Personalizando Atalhos

:::note Atalhos Personalizados
Atalhos de teclado atualmente são fixos no código. Configuração de atalhos personalizados pode ser adicionada em versões futuras.
:::


**Limitações atuais:**

- Atalhos não podem ser alterados sem modificar código fonte
- Sem GUI para personalização de atalhos
- Algumas ações podem não ter atalhos atribuídos

**Solicitação de recurso:** Se você precisa de atalhos personalizados, por favor abra uma issue no GitHub.

## Dicas e Truques

### Dicas de Eficiência

1. **Aprenda os atalhos de edição primeiro** - `Ctrl+Z/Y/C/V/D` são usados constantemente
2. **Use alternâncias de tecla única** - `H`, `T`, `P` para mudanças rápidas de visualização
3. **Atalhos de visão 3D** - `1`, `2`, `7` para troca instantânea de visão
4. **Atalhos de alinhamento** - `Shift+Seta` mais rápido que clicar botões de alinhamento
5. **Modo simulação** - `Ctrl+Shift+S` para verificar rapidamente a execução

### Atalhos de Fluxo de Trabalho

**Ciclo rápido de edição:**

```
1. Novo Esboço (Ctrl+N) ou Abrir (Ctrl+O)
2. Organizar itens (teclas de seta, Shift+Seta para alinhamento)
3. Duplicar peças (Ctrl+D)
4. Verificar em 3D (F12)
5. Simular (F11)
6. Salvar (Ctrl+S) ou Exportar (Ctrl+E)
```

**Posicionamento de precisão:**

```
1. Selecionar item
2. Teclas de seta para empurrões de 1mm
3. Shift+Seta para saltos de 10mm
4. Shift+Home/End para centralizar
```

### Atalhos Ocultos

Alguns atalhos menos conhecidos:

- `Ctrl+,` - Acesso rápido às preferências (padrão no macOS, funciona aqui também)
- `Ctrl+Shift+Z` - Refazer alternativo (para usuários que preferem isso ao Ctrl+Y)
- `Alt+A` - Encaixe pixel-perfeito para colocação precisa
- `Alt+T` - Colocação rápida de abas equidistantes

## Diferenças de Plataforma

### Linux

- Usa modificador `Ctrl` padrão
- Todos os atalhos funcionam como documentado
- Atalhos padrão GTK se aplicam em campos de texto

### Windows

- Usa modificador `Ctrl` padrão
- Idêntico aos atalhos Linux
- Teclas específicas do Windows (tecla Win) não são usadas

### macOS

- `Ctrl` mapeia para `⌘ (Command)`
- `Alt` mapeia para `⌥ (Option)`
- Convenções padrão do macOS se aplicam
- `Cmd+Q` para sair (em vez de Ctrl+Q)

## Solução de Problemas de Atalhos

### Atalho Não Funciona

**Problemas comuns:**

1. **Foco no elemento errado** - Certifique-se de que tela ou janela principal tem foco, não um campo de texto
2. **Aplicação conflitante** - Outro app pode estar interceptando o atalho
3. **Atalho do ambiente de desktop** - Atalhos do sistema podem substituir (ex: Alt+F4)
4. **Numpad vs linha de números** - Use linha de números para atalhos de visão 3D, não numpad

**Soluções:**

- Clique na tela para garantir que tem foco
- Verifique atalhos de teclado do sistema para conflitos
- Tente a ação do menu em vez disso para verificar funcionalidade

### Teclas Modificadoras

**Se Ctrl não funciona:**

- Verifique se Caps Lock está desligado (pode interferir em alguns sistemas)
- Tente Ctrl em ambos lados do teclado
- Verifique configurações de layout de teclado

**Se Alt não funciona:**

- Alguns gerenciadores de janela capturam Alt para arrastar janela
- Tente desabilitar atalhos Alt do gerenciador de janela
- Use itens de menu como alternativa
