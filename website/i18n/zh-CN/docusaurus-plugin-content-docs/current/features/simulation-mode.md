# 仿真模式

![仿真模式](/screenshots/main-simulation.png)

仿真模式显示激光作业在机器上运行之前将如何执行。您可以逐步浏览 G 代码并准确查看将会发生什么。

## 激活仿真模式

- **键盘**: 按 <kbd>F11</kbd>
- **菜单**: 转到 **视图 → 模拟执行**
- **工具栏**: 点击仿真按钮

## 可视化

### 速度热力图

操作根据速度着色：

| 速度 | 颜色 |
| ---- | ---- |
| 最慢 | 蓝色 |
| 慢   | 青色 |
| 中等 | 绿色 |
| 快   | 黄色 |
| 最快 | 红色 |

颜色相对于您作业的速度范围 - 蓝色是最小值，红色是最大值。

### 功率透明度

线条的不透明度显示激光功率：

- **淡线条** = 低功率（空走移动，轻度雕刻）
- **实线** = 高功率（切割）

## 播放控制

使用画布底部的控件：

- **播放/暂停** (<kbd>空格</kbd>): 开始或停止自动播放
- **进度滑块**: 拖动以浏览作业
- **方向键**: 逐条浏览指令

仿真和 G 代码视图保持同步 - 浏览仿真会高亮相应的 G 代码，点击 G 代码行会跳转到仿真中的该点。

## 仿真时编辑

您可以在仿真时编辑工件。移动、缩放或旋转对象，仿真会自动更新。
