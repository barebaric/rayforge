# Технічне обслуговування

Сторінка "Обслуговування" у налаштуваннях машини допомагає відстежувати використання машини та планувати завдання з технічного обслуговування.

![Налаштування обслуговування](/screenshots/machine-maintenance.png)

## Лічильники годин

Rayforge відстежує, як довго ваша машина використовується. Ці лічильники допомагають планувати профілактичне обслуговування.

### Години лазера

Відстежує загальний час роботи лазера. Використовуйте це для:

- Планування заміни лазерної трубки (CO2 трубки зазвичай служать 1000-3000 годин)
- Моніторингу деградації лазерного діода
- Відстеження гарантійних періодів

### Години завдань

Відстежує загальний час виконання завдань, включаючи переміщення. Використовуйте це для:

- Планування натягування ременів
- Планування змащування підшипників
- Відстеження загального використання машини

### Години переміщень

Відстежує час, витрачений на переміщення (не різання). Використовуйте це для:

- Планування обслуговування степерних двигунів
- Планування очищення та змащування рейок

## Скидання лічильників

Після виконання обслуговування ви можете скинути відповідний лічильник:

1. Натисніть кнопку скидання поруч із лічильником
2. Підтвердьте скидання в діалозі
3. Лічильник повернеться до нуля

:::порада Графік обслуговування
Типові інтервали обслуговження:
- **Щодня**: Очистіть лінзу, перевірте вирівнювання дзеркал
- **Щотижня**: Очистіть рейки, перевірте натяг ременів
- **Щомісяця**: Змастіть підшипники, перевірте електричні з'єднання
- **Щороку**: Повний огляд, заміна зношених частин
:::
