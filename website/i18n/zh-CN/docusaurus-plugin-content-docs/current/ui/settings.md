# 设置

![常规设置](/screenshots/application-general.png)

自定义 Rayforge 以匹配您的工作流程和偏好。

## 访问设置

- **菜单**：编辑 → 设置
- **键盘**：<kbd>ctrl+comma</kbd>

## 常规

常规页面包含应用程序范围设置。

### 外观

- **主题**：在系统、浅色或深色主题之间选择

### 操作

- **自动更新操作**：启用时（默认），操作在每次更改后自动重新计算。如果您更喜欢通过工具栏按钮手动触发重新计算，可以禁用此功能 — 这在较慢的机器上或处理非常复杂的文档时可能会有帮助。

### 单位

配置整个应用程序中各种值的显示单位：

- **长度**：毫米、英寸和其他长度单位
- **速度**：mm/min、mm/sec、inches/min 和其他速度单位
- **加速度**：mm/s² 和其他加速度单位

### 启动

配置应用程序启动时发生的事情。命令行指定的文件将始终覆盖这些设置。

- **启动行为**：
  - 不打开任何内容
  - 打开上次的项目
  - 打开特定项目
- **项目路径**：启动时打开的特定项目路径
  （仅当选择"打开特定项目"时可见）

## 机器

机器页面允许您管理机器配置。

- **添加机器**：创建新的机器配置
- **移除机器**：删除选定的机器
- **机器列表**：显示所有配置的机器及其连接状态

有关详细的机器配置，请参阅[机器设置](../machine/general)部分。

## 材料

材料页面管理您的材料库。

- **添加库**：创建新的材料库
- **编辑**：重命名选定的库
- **删除**：删除选定的库

有关详细信息，请参阅[材料](../application-settings/materials)。

## 配方

配方页面管理您的操作配方。

- **添加配方**：创建新配方
- **编辑**：修改选定的配方
- **删除**：删除选定的配方

有关详细信息，请参阅[配方](../application-settings/recipes)。

## 扩展

扩展页面显示已安装的扩展。

- **扩展列表**：显示所有已安装的扩展
- **扩展详情**：查看有关每个扩展的信息

## 隐私

隐私页面控制匿名使用报告。

- **报告匿名使用**：启用后，将发送匿名使用数据以帮助改进 Rayforge。不会收集任何个人信息。

## 许可证

许可证页面管理您的高级功能和扩展的许可证密钥。

- **关联 Patreon 账户**：连接您的 Patreon 账户以获取支持者福利
- **扩展许可证密钥**：输入高级扩展的许可证密钥
