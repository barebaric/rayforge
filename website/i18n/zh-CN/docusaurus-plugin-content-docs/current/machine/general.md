# 常规设置

机器设置中的"常规"页面包含机器名称、驱动程序选择和连接设置，以及速度参数。

![常规设置](/screenshots/machine-general.png)

## 机器名称

为你的机器起一个描述性的名称。当你配置了多台机器时，这有助于在
机器选择下拉菜单中识别该机器。

## 驱动程序

选择与你的机器控制器匹配的驱动程序。驱动程序负责 Rayforge 与硬件
之间的通信。

选择驱动程序后，选择器下方会出现特定于连接的设置（例如串口、波特率）。
这些设置因所选驱动程序而异。

:::tip
如果驱动程序未配置或遇到问题，页面顶部的错误横幅会发出警告。
:::

## 速度与加速度

这些设置控制最大速度和加速度，用于作业时间估算和路径优化。

### 最大空移速度

激光关闭且机头移动到新位置时，快速（非切割）移动的最大速度。

- **典型范围**：2000–5000 mm/min
- **注意**：实际速度还受固件设置的限制。如果所选 G-code 方言不支持
  指定空移速度，此字段将被禁用。

### 最大切割速度

切割或雕刻操作期间允许的最大速度。

- **典型范围**：500–2000 mm/min
- **注意**：个别操作可能会使用较低的速度

### 加速度

机器加速和减速的速率，用于时间估算和计算默认过扫描距离。

- **典型范围**：500–2000 mm/s²
- **注意**：必须与固件中的加速度设置一致或更低

:::tip
从保守的速度值开始，然后逐步增加。观察你的机器是否出现皮带跳齿、
电机失步或定位精度下降的情况。
:::

## 导出机器配置

点击设置对话框标题栏中的分享图标，导出当前的机器配置。选择一个保存
文件夹。系统将创建一个 ZIP 文件，其中包含机器设置及其 G-code 方言，
可以分享给其他用户或在另一台系统上导入。
