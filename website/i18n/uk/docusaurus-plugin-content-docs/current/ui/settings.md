# Налаштування

![Загальні налаштування](/screenshots/application-general.png)

Налаштуйте Rayforge відповідно до вашого робочого процесу та вподобань.
Відкрийте діалог налаштувань через **Редагування → Налаштування** або
натисніть <kbd>ctrl+кома</kbd>.

## Загальні

Сторінка Загальні містить налаштування програми.

### Зовнішній вигляд

Виберіть між **Системною**, **Світлою** або **Темною** темою, щоб відповідати
вашому робочому середовищу або особистим вподобанням. Ви також можете
налаштувати **Кольори операцій**, щоб використовувати колір лазера або колір
шару для візуального розрізнення на полотні.

### Одиниці

Налаштуйте одиниці відображення, що використовуються в програмі. Ви можете
задати окремі одиниці для **довжини** (міліметри, дюйми тощо), **швидкості**
(мм/хв, мм/сек, дюйми/хв тощо) та **прискорення** (мм/с² тощо).

### Поведінка

За замовчуванням операції перераховуються автоматично після кожної зміни. Якщо
ви працюєте на повільнішій машині або з дуже складними документами, ви можете
вимкнути **Автоматичне оновлення операцій** і ініціювати перерахунок вручну
через кнопку на панелі інструментів.

Rayforge може **Перевіряти оновлення** автоматично при запуску. Якщо увімкнено,
ви отримаєте сповіщення, коли буде доступна нова версія.

Ви також можете налаштувати **Поведінку при запуску** — починати з порожнього
робочого простору, повторно відкривати останній проект або завжди відкривати
певний файл проекту. Зверніть увагу, що файли, вказані в командному рядку,
завжди перекриватимуть ці налаштування.

### Приватність

Rayforge може надсилати анонімні дані про використання для допомоги в
покращенні програми. Жодна особиста інформація не збирається. Ви можете
вмикати або вимикати **Звітування про анонімне використання** у будь-який час.
Дивіться сторінку
[відстеження використання](https://rayforge.org/docs/general-info/usage-tracking),
щоб дізнатися більше про те, які дані збираються та як вони використовуються.

## Інші налаштування

Діалог налаштувань також містить сторінки для керування іншими частинами
програми. Кожна має власну документацію:

- [Машини](../application-settings/machines) — додавати, видаляти та
  налаштовувати ваші лазерні різаки
- [Матеріали](../application-settings/materials) — керувати вашими
  бібліотеками матеріалів
- [Рецепти](../application-settings/recipes) — керувати збереженими
  рецептами операцій
- [AI-провайдери](../application-settings/ai-provider) — налаштувати
  AI-провайдерів для використання аддонами
- [Аддони](../application-settings/addons) — встановлювати, оновлювати та
  видаляти аддони розширень
