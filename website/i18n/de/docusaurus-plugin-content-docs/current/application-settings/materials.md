# Materialien

![Material-Einstellungen](/screenshots/application-materials.png)

Material-Bibliotheken in Rayforge ermöglichen es Ihnen, Material-Sammlungen für Ihre Laserschneide- und Gravurprojekte zu organisieren und zu verwalten. Diese Anleitung erklärt den Unterschied zwischen Kern- und Benutzer-Bibliotheken und wie Sie Ihre eigenen Bibliotheken erstellen und Materialien hinzufügen.

:::note
Material-Bibliotheken haben derzeit keine funktionale Verwendung - das Zuweisen eines Materials betrifft lediglich das optische Erscheinungsbild. In zukünftigen Versionen werden Materialien verwendet, um funktionale Parameter abzuleiten.
:::


## Eine neue Bibliothek erstellen

Um Ihre eigene Materialbibliothek zu erstellen:

1. Öffnen Sie das Menü **Einstellungen** und wählen Sie **Materialien**
2. Klicken Sie auf die Schaltfläche **Neue Bibliothek hinzufügen**, um eine neue Bibliothek zu erstellen
3. Geben Sie einen beschreibenden Namen für Ihre Bibliothek ein (z.B. "Meine Werkstatt-Materialien")
4. Klicken Sie auf **Erstellen** zum Fertigstellen

Ihre neue Bibliothek wird im Benutzerdaten-Verzeichnis erstellt und ist sofort verfügbar.


## Materialien zu Bibliotheken hinzufügen

### Ein neues Material erstellen

1. Wählen Sie die Bibliothek aus, zu der Sie das Material hinzufügen möchten
2. Klicken Sie auf die Schaltfläche **Neues Material hinzufügen** in der Materialliste
3. Füllen Sie die Materialeigenschaften aus:
   - **Name**: Lesbarer Name
   - **Kategorie**: Gruppierungskategorie (z.B. "Holz", "Acryl")
   - **Aussehen**: Visuelle Eigenschaften (Farbe)
4. Klicken Sie auf **Speichern**, um das Material zur Bibliothek hinzuzufügen

### Materialeigenschaften erklärt

#### Name
- Lesbarer Name, der in der Schnittstelle angezeigt wird
- Kann Leerzeichen und Sonderzeichen enthalten

#### Kategorie
- Wird zum Organisieren von Materialien innerhalb der Bibliothek verwendet
- Häufige Kategorien: Holz, Acryl, Metall, Papier, Leder
- Sie können benutzerdefinierte Kategorien nach Bedarf erstellen

#### Farbe

Die Farbe wird nur für das visuelle Erscheinungsbild auf der Arbeitsfläche verwendet - sie beeinflusst den Laserpfad in keiner Weise.


## Bestehende Materialien verwalten

### Materialien bearbeiten

1. Wählen Sie das Material aus, das Sie bearbeiten möchten
2. Klicken Sie auf die Schaltfläche **Bearbeiten**
3. Ändern Sie die gewünschten Eigenschaften
4. Klicken Sie auf **Speichern**, um die Änderungen anzuwenden

### Materialien löschen

1. Wählen Sie das Material aus, das Sie löschen möchten
2. Klicken Sie auf die Schaltfläche **Löschen**
3. Bestätigen Sie das Löschen im Dialog

:::warning
Das Löschen eines Materials ist dauerhaft und kann nicht rückgängig gemacht werden.
:::
