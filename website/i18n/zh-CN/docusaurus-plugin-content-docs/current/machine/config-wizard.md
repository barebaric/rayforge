# 配置向导

配置向导通过连接设备并读取其固件设置来自动检测您的设备。它能在几秒钟内
创建一个完全配置好的机器配置文件，无需手动设置。

## 启动向导

1. 打开 **设置 → 机器**，点击 **Add Machine**
2. 在配置文件选择器中，点击底部的 **Other Device…**

这将打开向导。它**不**需要预先存在的设备配置文件——向导通过探测
已连接的硬件从头创建一个。

## 连接

第一页要求您选择驱动程序并提供连接参数。

![向导 连接](/screenshots/app-settings-machines-wizard-connect.png)

### 驱动程序选择

选择与您设备控制器匹配的驱动程序。只列出支持探测的驱动程序。通常是：

- **GRBL (串口)** — 通过 USB 连接的 GRBL 设备
- **GRBL (网络)** — WiFi/以太网 GRBL 设备

### 连接参数

选择驱动程序后，填写连接详情（串口、波特率、主机等）。这些参数与
[常规设置](general)中使用的相同。

点击 **Next** 开始探测。

## 发现

向导连接到设备并查询其固件以获取配置数据。这包括：

- 固件版本和构建信息（`$I`）
- 所有固件设置（`$$`）
- 轴行程、速度、加速度和激光功率范围

此步骤通常在几秒钟内完成。

## 审查

探测成功后，审查页面显示所有发现的设置。所有内容都已预填，但在创建
机器之前可以进行修改。

![向导 审查](/screenshots/app-settings-machines-wizard-review.png)

### 设备信息

从固件检测到的只读信息：

- **设备名称** — 从固件构建信息中提取
- **固件版本** — 例如 `1.1h`
- **RX 缓冲区大小** — 串行接收缓冲区大小
- **弧线容差** — 固件弧线插值容差

### 工作区域

- **X 行程** / **Y 行程** — 轴的最大行程（机器单位），
  从固件设置 `$130` 和 `$131` 读取

### 速度

- **最大空移速度** — 取 `$110` 和 `$111` 中的较小值
- **最大切割速度** — 默认与空移速度相同；根据需要调整

### 加速度

- **加速度** — 取 `$120` 和 `$121` 中的较小值，单位为
  机器单位每秒平方

### 激光

- **最大功率（S值）** — 从固件设置 `$30`（主轴最大值）获取

### 行为

- **启动时归零** — 如果固件归零（`$22`）已启用则开启
- **单轴归零** — 如果固件使用 `H` 标志编译则开启

### 警告

向导可能会显示关于潜在问题的警告，例如：

- 激光模式未启用（`$32=0`）
- 设备以英寸报告（`$13=1`）

## 创建机器

点击 **Create Machine** 完成。向导发出配置好的配置文件，正常的
机器创建流程继续——[机器设置对话框](general)打开以便进一步调整。

## 限制

- 向导仅适用于支持探测的驱动程序。如果您的驱动程序未列出，
  请改用选择器中的预置配置文件。
- 探测需要设备已开机并连接。
- 某些固件设置可能无法在所有设备上读取。
