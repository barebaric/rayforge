# Системи координат

Розуміння того, як Rayforge обробляє системи координат, є необхідним для правильного позиціонування вашої роботи.

## Робоча система координат (WCS) проти координат машини

Rayforge використовує дві основні системи координат:

### Робоча система координат (WCS)

WCS - це система координат вашого завдання. Коли ви позиціонуєте дизайн у (50, 100) на полотні, це координати WCS.

- **Початок**: Визначається вами (за замовчуванням G54)
- **Призначення**: Дизайн та позиціонування завдання
- **Кілька систем**: G54-G59 доступні для різних налаштувань

### Координати машини

Координати машини - це абсолютні позиції відносно позиції home машини.

- **Початок**: Home машини (0,0,0) - фіксовано обладнанням
- **Призначення**: Фізичне позиціонування на робочому столі
- **Фіксовано**: Не може бути змінено програмно

**Відношення**: Зміщення WCS визначають, як ваші координати завдання відображаються на координати машини. Якщо зміщення G54 дорівнює (100, 50, 0), то ваш дизайн у WCS (0, 0) ріже у позиції машини (100, 50).

## Налаштування координат у Rayforge

### Встановлення початку WCS

Для позиціонування вашого завдання на машині:

1. **Спочатку виконайте homing машини** (команда `$H` або кнопка Home)
2. **Перемістіть лазерну головку** у бажаний початок завдання
3. **Встановіть нуль WCS** за допомогою панелі керування:
    - Натисніть "Zero X" щоб встановити поточний X як початок
    - Натисніть "Zero Y" щоб встановити поточний Y як початок
4. Ваше завдання тепер починатиметься з цієї позиції

### Вибір WCS

Rayforge підтримує робочі системи координат G54-G59:

| Система | Випадок використання |
|--------|----------|
| G54 | За замовчуванням, основна робоча область |
| G55-G59 | Додаткові позиції оснащення |

Виберіть активний WCS на панелі керування. Кожна система зберігає власне зміщення від початку машини.

### Напрямок осі Y

Деякі машини мають Y, що збільшується вниз замість вгору. Налаштуйте це в:

**Налаштування → Машина → Обладнання → Осі**

Якщо ваші завдання виходять дзеркально відображеними по вертикалі, перемкніть налаштування напрямку осі Y.

## Поширені проблеми

### Завдання у неправильній позиції

- **Перевірте зміщення WCS**: Надішліть `G10 L20 P1` щоб переглянути зміщення G54
- **Перевірте homing**: Машина повинна бути homerована для узгодженого позиціонування
- **Перевірте напрямок осі Y**: Може бути інвертовано

### Дрейф координат між завданнями

- **Завжди виконуйте homing перед завданнями**: Встановлює узгоджену точку відліку
- **Перевірте наявність зміщень G92**: Очистіть командою `G92.1`
