# Paramètres

![Paramètres généraux](/screenshots/application-general.png)

Personnalisez Rayforge pour correspondre à votre flux de travail et vos préférences.

## Accéder aux paramètres

- **Menu** : Édition → Paramètres
- **Clavier** : <kbd>ctrl+virgule</kbd>

## Général

La page Général contient les paramètres à l'échelle de l'application.

### Apparence

- **Thème** : Choisissez entre le thème Système, Clair ou Sombre

### Unités

Configurez les unités d'affichage pour diverses valeurs dans l'application :

- **Longueur** : Millimètres, pouces et autres unités de longueur
- **Vitesse** : mm/min, mm/sec, pouces/min et autres unités de vitesse
- **Accélération** : mm/s² et autres unités d'accélération

### Démarrage

Configurez ce qui se passe au démarrage de l'application. Les fichiers spécifiés sur
la ligne de commande remplaceront toujours ces paramètres.

- **Comportement au démarrage** :
  - N'ouvrir rien
  - Ouvrir le dernier projet
  - Ouvrir un projet spécifique
- **Chemin du projet** : Chemin vers le projet spécifique à ouvrir au démarrage
  (visible uniquement lorsque "Ouvrir un projet spécifique" est sélectionné)

## Machines

La page Machines vous permet de gérer vos configurations machine.

- **Ajouter une machine** : Créer une nouvelle configuration machine
- **Supprimer une machine** : Supprimer une machine sélectionnée
- **Liste des machines** : Affiche toutes les machines configurées avec leur état de connexion

Pour une configuration détaillée de la machine, voir la section [Configuration machine](../machine/general).

## Matériaux

La page Matériaux gère votre bibliothèque de matériaux.

- **Ajouter un matériau** : Créer un nouveau préréglage de matériau
- **Importer une bibliothèque** : Importer des matériaux depuis un fichier
- **Exporter une bibliothèque** : Exporter vos matériaux pour sauvegarde ou partage

Voir [Matériaux](../application-settings/materials) pour plus de détails.

## Recettes

La page Recettes gère vos recettes d'opérations.

- **Ajouter une recette** : Créer une nouvelle recette
- **Importer des recettes** : Importer des recettes depuis un fichier
- **Exporter des recettes** : Exporter vos recettes pour sauvegarde ou partage

Voir [Recettes](../application-settings/recipes) pour plus de détails.

## Paquets

La page Paquets affiche les paquets d'extension installés.

- **Liste des paquets** : Affiche tous les paquets installés
- **Détails des paquets** : Voir des informations sur chaque paquet
