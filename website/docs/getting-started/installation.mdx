# Installation

import InstallGuide from '@site/src/components/InstallGuide';

Rayforge is available for Linux, Windows, and macOS. Choose your platform below
and follow the installation instructions.

## System Requirements

| Resource   | Installed | Recommended       |
|------------|-----------|-------------------|
| Disk Space | < 1 GB    | Varies by project |
| RAM        | 1 GB      | Varies by project |

Actual disk and memory requirements depend on project complexity. Simple
projects may need little beyond the installed footprint, while complex projects
can easily consume 10 GB of RAM or more and require significant additional disk
space.
