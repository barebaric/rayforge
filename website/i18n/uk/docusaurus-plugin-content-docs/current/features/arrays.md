# Масиви

Функція Масиву дозволяє створювати кілька копій вибраних робочих елементів
за допомогою трьох різних режимів компонування. Кожен режим відкриває
немодальне діалогове вікно, що дозволяє продовжувати роботу з полотном
під час налаштування параметрів — попередній перегляд оновлюється в
реальному часі.

Щоб відкрити діалогове вікно масиву, виберіть один або кілька робочих
елементів на полотні, потім виберіть режим масиву з панелі інструментів
або контекстного меню.

:::tip
Усі режими масиву є немодальними. Ви можете перетягувати робочі елементи
на полотні, поки діалогове вікно відкрите, і попередній перегляд
оновлюватиметься в реальному часі для відображення нових позицій.
:::

---

## Сітка

Режим Сітки організовує копії в прямокутну матрицю з рядків і стовпців
з налаштовуваним горизонтальним та вертикальним відстанню.

![Масив Сітки](/screenshots/main-array-grid.png)

### Налаштування

| Параметр | Опис |
|----------|------|
| **Рядки** | Кількість рядків (1–360) |
| **Стовпці** | Кількість стовпців (1–360) |
| **Режим відстані** | Виберіть між *Проміжок* (відстань між копіями) або *Крок* (відстань від краю до краю кожної копії) |
| **Відстань стовпців** | Горизонтальна відстань між стовпцями |
| **Відстань рядків** | Вертикальна відстань між рядками |

## Точковий оберт

Режим Точкового оберту створює копії, обертаючи їх на місці навколо
власного центру вибору. Це корисно для створення кругових візерунків,
де кожна копія залишається на своєму оригінальному місці, але обертається
на частку загального кута.

![Масив Точкового оберту](/screenshots/main-array-point-rotation.png)

### Налаштування

| Параметр | Опис |
|----------|------|
| **Кількість** | Кількість копій (1–360) |
| **Загальний кут (градуси)** | Загальний кутовий діапазон усіх копій (−360° до 360°) |

:::info
Оскільки обертання відбувається навколо власного центру вибору, перетягування
робочого елемента на полотні переміщує всі копії разом, поки діалогове
вікно залишається відкритим.
:::

---

## Круговий

Режим Кругового розміщує копії вздовж кругової дуги навколо центральної
точки. Мітка у вигляді перехрестя на полотні показує центр, і ви можете
перетягнути її на нову позицію, поки діалогове вікно відкрите.

![Масив Кругового](/screenshots/main-array-circular.png)

### Налаштування

| Параметр | Опис |
|----------|------|
| **Кількість** | Кількість копій (1–360) |
| **Загальний кут (градуси)** | Кутовий діапазон дуги (−360° до 360°) |
| **Центр X** | Координата X центру кола |
| **Центр Y** | Координата Y центру кола |
| **Радіус** | Радіус кругової траєкторії |
| **Обертати копії** | Якщо увімкнено, кожна копія обертається для відображення дотичної дуги |

:::tip Перетягування центру
Перехрестя на полотні представляє центр кола. Перетягніть його для
інтерактивного переміщення масиву — поля Центр X і Центр Y у діалоговому
вікні оновляться автоматично.
:::

:::tip Перетягування робочих елементів
Ви також можете перетягувати оригінальний робочий елемент на полотні.
Радіус оновиться автоматично, щоб підтримувати копії на їхній поточній
відстані від центру.
:::
