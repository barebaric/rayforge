# Обведення вашого завдання

Дізнайтеся, як використовувати функцію обведення для попереднього перегляду
меж вашого завдання та забезпечення правильного вирівнювання перед різанням.

## Огляд

Обведення дозволяє попередньо переглянути точні межі вашого лазерного
завдання, обводячи контур лазером на малій потужності або з вимкненим лазером.
Це допомагає перевірити позиціонування та запобігти дорогим помилкам.

## Коли використовувати обведення

- **Перші налаштування**: Перевірити розміщення матеріалу
- **Точне позиціонування**: Переконатися, що дизайн вміщується в межі
  матеріалу
- **Кілька завдань**: Підтвердити вирівнювання перед кожним запуском
- **Дорогі матеріали**: Перевірити перед виконанням різів

## Як виконати обведення

### Метод 1: Тільки контур

Обвести межу завдання без увімкнення лазера:

1. **Завантажте дизайн** у Rayforge
2. **Розташуйте матеріал** на робочому столі лазера
3. **Натисніть кнопку Обведення** на панелі інструментів
4. **Спостерігайте за лазерною головкою**, як вона обводить прямокутник меж
5. **Перевірте позиціонування** та за потреби скоригуйте матеріал

### Метод 2: Попередній перегляд на малій потужності

Деякі машини підтримують обведення на малій потужності з видимим променем:

1. **Увімкніть режим малої потужності** в налаштуваннях машини
2. **Встановіть потужність обведення** (зазвичай 1–5 %)
3. **Виконайте операцію обведення**
4. **Спостерігайте за контуром**, що обводиться на поверхні матеріалу

:::warning Перевірте вашу машину
Не всі лазери безпечно підтримують обведення на малій потужності. Зверніться
до документації вашої машини перед використанням цієї функції.
:::

## Налаштування обведення

Налаштуйте поведінку обведення в налаштуваннях лазерної головки вашої машини:

- **Швидкість обведення**: Як швидко рухається лазерна головка під час
  обведення. Налаштовується для кожної лазерної головки окремо, тому якщо
  ваша машина має кілька лазерів, ви можете використовувати різні швидкості
  для кожного.
- **Потужність обведення**: Потужність лазера під час обведення (0 для
  вимкненого, малий % для видимого сліду)
- **Час затримки на кутах**: Коротка пауза на кожному куті контуру обведення.
  Це дає вам змогу побачити, де саме знаходиться кожен кут — особливо
  корисно при вищих швидкостях обведення.
- **Кількість повторень**: Кількість разів обведення контуру. Встановлення
  значення більше одного робить шлях обведення легшим для візуального
  відстеження.

## Використання результатів обведення

Після обведення ви можете:

- **Скоригувати положення матеріалу** за потреби
- **Повторити обведення** для перевірки нового положення
- **Продовжити завдання** після підтвердження розміщення

## Поради для ефективного обведення

- **Позначте кути**: Розмістіть невеликі шматочки стрічки на кутах як
  орієнтири
- **Перевірте зазор**: Переконайтеся в достатньому просторі навколо дизайну
- **Підтвердьте орієнтацію**: Перевірте правильність орієнтації матеріалу
- **Врахуйте пропил**: Пам'ятайте, що різи будуть трохи ширшими за контури

## Обведення з камерою

Якщо ваша машина підтримує камеру, ви можете:

1. **Захопити зображення з камери** розміщення матеріалу
2. **Накласти дизайн** на зображення з камери
3. **Скоригувати положення** віртуально перед обведенням
4. **Виконати обведення** для підтвердження фізичного вирівнювання

Див. [Інтеграція з камерою](../machine/camera) для деталей.

## Вирішення проблем

**Обведення не відповідає дизайну**: Перевірте початок координат завдання та
налаштування системи координат

**Лазер спрацьовує під час обведення**: Вимкніть потужність обведення або
перевірте налаштування машини

**Обведення занадто швидке**: Зменште швидкість обведення в налаштуваннях

**Головка не досягає кутів**: Перевірте, що дизайн знаходиться в робочій
зоні машини

## Зауваження з безпеки

- **Ніколи не залишайте машину без нагляду** під час обведення
- **Переконайтеся, що лазер вимкнений** при обведенні без потужності
- **Тримайте руки подалі** від траєкторії руху лазерної головки
- **Слідкуйте за перешкодами**, які можуть заважати руху
