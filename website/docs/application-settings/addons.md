# Addons

![Addons Settings](/screenshots/application-addons.png)

The Addons page shows installed extension addons and allows you to
manage them.

## Overview

Addons are extensions that add functionality to Rayforge. They can
provide:

- New operations
- Import/export filters
- Machine drivers
- UI enhancements
- Material libraries

## Installed Addons

The addon list shows all installed addons with:

- **Name**: The addon's display name
- **Version**: Current installed version
- **Description**: Brief description of functionality
- **Status**: Whether enabled or disabled

## Managing Addons

### Installing Addons

1. Click the **Install** button
2. Select an addon file (`.zip` or addon repository)
3. Confirm the installation
4. Wait for the installation to complete

### Updating Addons

When updates are available:

1. An update indicator appears next to the addon
2. Click the **Update** button to install the update
3. Restart Rayforge if prompted

### Removing Addons

1. Select the addon to remove
2. Click the **Remove** button
3. Confirm the removal

:::warning
Removing an addon may affect documents that use its features.
:::

## Addon Sources

Addons can be installed from:

- **Local files**: `.zip` archives downloaded manually
- **Repositories**: Git repositories containing addons
- **Official registry**: Curated addons from the Rayforge team
