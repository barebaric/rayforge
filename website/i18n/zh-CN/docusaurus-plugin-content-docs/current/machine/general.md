# 常规设置

机器设置中的常规页面包含基本机器信息和速度设置。

![常规设置](/screenshots/machine-general.png)

## 机器名称

给您的机器一个描述性名称。这有助于在配置多台机器时在机器选择器下拉菜单中识别机器。

示例：
- "车间 K40"
- "车库二极管激光"
- "Ortur LM2 Pro"

## 速度和加速度

这些设置控制运动规划和时间估计的最大速度和加速度。

### 最大空行速度

快速（非切割）移动的最大速度。这在激光关闭且头正在移动到新位置时使用。

- **典型范围**：2000-5000 mm/min
- **目的**：运动规划和时间估计
- **注意**：实际速度也受您的固件设置限制

### 最大切割速度

切割或雕刻操作期间允许的最大速度。

- **典型范围**：500-2000 mm/min
- **目的**：限制操作速度以确保安全
- **注意**：个别操作可能使用较低速度

### 加速度

机器加速和减速的速率。

- **典型范围**：500-2000 mm/s²
- **目的**：时间估计和运动规划
- **注意**：必须与固件加速度设置匹配或更低

:::tip
从保守的速度值开始并逐渐增加。观察您的机器是否有皮带跳步、电机失步或定位精度丢失。
:::
