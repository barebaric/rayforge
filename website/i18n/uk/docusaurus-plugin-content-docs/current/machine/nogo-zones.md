# Зони обмеження

Зони обмеження визначають заборонені області на робочій поверхні, куди лазер
не повинен заходити. Перед запуском або експортом завдання Rayforge перевіряє,
чи не заходять траєкторії у ввімкнену зону обмеження, і попереджає вас,
якщо виявлено зіткнення.

![Зони обмеження](/screenshots/machine-nogo-zones.png)

## Додавання зони обмеження

Відкрийте **Налаштування → Машина** та перейдіть на сторінку **Зони обмеження**.
Натисніть кнопку додавання, щоб створити нову зону, потім виберіть її форму та
позицію.

Кожна зона має наступні налаштування:

- **Назва**: Описова мітка для зони
- **Увімкнено**: Перемкнути зону увімк/вимк без її видалення
- **Форма**: Прямокутник, Коробка або Циліндр
- **Позиція (X, Y, Z)**: Де зона розміщена на робочій поверхні
- **Розміри**: Ширина, висота та глибина (або радіус для циліндрів)

## Попередження про зіткнення

При запуску або експорті завдання Rayforge перевіряє всі траєкторії проти
увімкнених зон обмеження. Якщо траєкторія проходить через зону, з'являється
діалогове вікно з попередженням та можливістю скасувати або продовжити на власний
ризик.

## Видимість

Зони обмеження відображаються як на 2D, так і на 3D полотні як напівпрозорі
накладання. Використовуйте кнопку перемикання зон обмеження в накладанні полотна,
щоб показати або приховати їх. Налаштування видимості зберігається між сесіями.
