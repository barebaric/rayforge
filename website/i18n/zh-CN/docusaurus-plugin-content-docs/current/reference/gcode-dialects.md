# G 代码方言支持

Rayforge 支持多种 G 代码方言以配合不同的控制器固件。

## 支持的方言

Rayforge 目前支持这些 G 代码方言：

| 方言                        | 固件        | 常见用途             | 状态                   |
| --------------------------- | ----------- | -------------------- | ---------------------- |
| **GRBL（通用）**            | GRBL 1.1+   | 二极管激光、业余 CNC | 主要，完全支持         |
| **GRBL（无 Z 轴）**         | GRBL 1.1+   | 无 Z 的 2D 激光切割机| 优化变体               |
| **GRBL 动态（深度感知）**   | GRBL 1.1+   | 深度感知激光雕刻     | 推荐用于动态功率       |
| **GRBL 动态（无 Z 轴）**    | GRBL 1.1+   | 深度感知激光雕刻     | 优化变体               |
| **Smoothieware**            | Smoothieware| 激光切割机、CNC      | 实验性                 |
| **Marlin**                  | Marlin 2.0+ | 带激光的 3D 打印机   | 实验性                 |

:::note 推荐方言
:::

**GRBL（通用）** 是最测试和推荐的标准激光应用方言。

    **GRBL 动态（深度感知）** 推荐用于深度感知激光雕刻，其中功率在切割期间变化（例如，可变深度雕刻）。
---

## 创建自定义方言

要基于内置方言创建自定义 G 代码方言：

1. 打开 **机器设置** → **G 代码方言**
2. 在内置方言上点击 **复制** 图标以创建新的自定义方言
3. 根据需要编辑方言设置
4. 保存您的自定义方言

自定义方言存储在您的配置目录中，可以共享。
