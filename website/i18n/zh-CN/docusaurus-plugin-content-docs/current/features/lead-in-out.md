# 引入 / 引出

引入和引出运动通过在切割开始前和结束后添加短的无功率段来扩展每个轮廓路径。这使激光头有时间在实际切割开始前达到稳定速度，并在切割结束后逐渐减速，从而在每个切割的起点和终点产生更干净的结果。

## 工作原理

启用引入/引出时，Rayforge 会查看每个轮廓路径在起点和终点处的切线方向。然后沿该切线在第一个切割点之前插入一段短的无功率直线运动，在最后一个切割点之后再插入一段。在这些额外的段中激光是关闭的，因此不会在预定路径之外移除任何材料。

## 设置

### 启用引入/引出

为操作开启或关闭此功能。禁用时，切割精确地在路径端点开始和结束，没有额外的接近或退出运动。

### 自动距离

启用此选项时，Rayforge 会根据切割速度和机器加速度设置自动计算引入和引出距离。公式使用安全系数 2 来确保激光头有足够的空间达到全速。每当您更改切割速度或机器加速度更新时，距离都会重新计算。

### 引入距离

切割开始前的无功率接近运动长度，单位为毫米。默认值为 2 mm。仅在关闭自动距离时此字段可编辑。

### 引出距离

切割结束后的无功率退出运动长度，单位为毫米。默认值为 2 mm。仅在关闭自动距离时此字段可编辑。

## 何时使用引入/引出

当您注意到轮廓起点和终点处出现烧痕、过度烧焦或切割质量不一致时，引入/引出最为有用。无功率接近给机器时间加速到切割速度，使激光以全速到达材料；无功率引出让机器平稳减速，而不是在最后一个点上保持满功率。

它可作为轮廓切割、框架轮廓和收缩包络操作的后处理选项。
