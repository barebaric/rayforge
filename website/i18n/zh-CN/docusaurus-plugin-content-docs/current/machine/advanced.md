# 高级设置

机器设置中的高级页面包含用于专用用例的额外配置选项。

![高级设置](/screenshots/machine-advanced.png)

## 连接行为

控制 Rayforge 在连接期间如何与您的机器交互的设置。

### 连接时归零

启用后，Rayforge 在连接到机器时自动发送归零命令（$H）。

- **启用如果**：您的机器有可靠的限位开关
- **禁用如果**：您的机器无限位开关或归零不可靠

### 连接时清除警报

启用后，Rayforge 在连接时自动清除任何警报状态。

- **启用如果**：您的机器经常以警报状态启动
- **禁用如果**：您想在清除之前手动调查警报

## 翻转轴

这些设置反转轴移动的方向。

### 翻转 X 轴

反转 X 轴方向。启用后，正 X 向左移动而不是向右。

### 翻转 Y 轴

反转 Y 轴方向。启用后，正 Y 向下移动而不是向上。

:::info
翻转轴在以下情况下有用：
- 您的机器坐标系与预期行为不匹配
- 您的电机接线反了
- 您想匹配另一台机器的行为
:::
