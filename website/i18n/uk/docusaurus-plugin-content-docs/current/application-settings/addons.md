# Аддони

![Налаштування аддонів](/screenshots/application-addons.png)

Сторінка "Аддони" показує встановлені аддони розширень та дозволяє
керувати ними.

## Огляд

Аддони - це розширення, які додають функціональність до Rayforge. Вони можуть
надавати:

- Нові операції
- Фільтри імпорту/експорту
- Драйвери машин
- Покращення інтерфейсу
- Бібліотеки матеріалів

## Встановлені аддони

Список аддонів показує всі встановлені аддони з:

- **Назва**: Відображуване ім'я аддону
- **Версія**: Поточна встановлена версія
- **Опис**: Короткий опис функціональності
- **Статус**: Увімкнено або вимкнено

## Керування аддонами

### Встановлення аддонів

1. Натисніть кнопку **Встановити**
2. Виберіть файл аддону (`.zip` або репозиторій аддонів)
3. Підтвердьте встановлення
4. Дочекайтеся завершення встановлення

### Оновлення аддонів

Коли доступні оновлення:

1. Індикатор оновлення з'являється поруч з аддоном
2. Натисніть кнопку **Оновити** для встановлення оновлення
3. Перезапустіть Rayforge, якщо з'явиться запит

### Видалення аддонів

1. Виберіть аддон для видалення
2. Натисніть кнопку **Видалити**
3. Підтвердьте видалення

:::увага
Видалення аддону може вплинути на документи, які використовують його функції.
:::

## Джерела аддонів

Аддони можна встановити з:

- **Локальні файли**: `.zip` архіви, завантажені вручну
- **Репозиторії**: Git-репозиторії, що містять аддони
- **Офіційний реєстр**: Кураторські аддони від команди Rayforge
