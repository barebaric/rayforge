# Головне вікно

Головне вікно Rayforge — це ваш основний робочий простір для створення та керування лазерними роботами.

## Макет вікна

![Головне вікно](/screenshots/main-standard.png)

### 1. Рядок меню

Доступ до всіх функцій Rayforge через організовані меню:

- **Файл**: Відкрити, зберегти, імпортувати, експортувати та нещодавні файли
- **Редагування**: Скасувати, повторити, копіювати, вставити, налаштування
- **Вид**: Масштаб, сітка, лінійки, панелі та режими перегляду
- **Операції**: Додати, редагувати та керувати операціями
- **Машина**: Підключити, переміщення, калібрування, запуск/зупинка робіт
- **Допомога**: Документація, про програму та підтримка

### 2. Панель інструментів

Швидкий доступ до часто використовуваних інструментів:

- **Інструмент виділення**: Виділяти та переміщувати об'єкти
- **Інструмент панорами**: Навігація по полотну
- **Інструмент масштабування**: Збільшувати/зменшувати певні області
- **Інструмент вимірювання**: Вимірювати відстані та кути
- **Інструменти вирівнювання**: Вирівнювати та розподіляти об'єкти
- **Випадаючий список WCS**: Вибрати активну робочу систему координат (G53-G59)

Випадаючий список WCS дозволяє швидко перемикатися між системами координат.
Дивіться [Робочі системи координат](../general-info/work-coordinate-systems) для
отримання додаткової інформації.

### 3. Полотно

Основний робочий простір, де ви:

- Імпортуєте та розташовуєте дизайни
- Переглядаєте траєкторії інструменту
- Розміщуєте об'єкти відносно початку координат машини
- Тестуєте межі обведення

**Керування полотном:**

- **Панорама**: Середня кнопка миші та перетягування або <kbd>пробіл</kbd> + перетягування
- **Масштаб**: Колесо миші або <kbd>ctrl+"+"</kbd> / <kbd>ctrl+"-"</kbd>
- **Скинути вид**: <kbd>ctrl+0</kbd> або Вид → Скинути масштаб

### 4. Панель шарів

Керування операціями та призначення шарів:

- Перегляд усіх операцій у вашому проекті
- Призначення операцій елементам дизайну
- Зміна порядку виконання операцій
- Увімкнення/вимкнення окремих операцій
- Налаштування параметрів операцій

### 5. Панель властивостей

Налаштування параметрів для виділених об'єктів або операцій:

- Тип операції (Контур, Растр тощо)
- Налаштування потужності та швидкості
- Кількість проходів
- Розширені параметри (оверкан, керф, лапки)

### 6. Панель керування

Панель керування в нижній частині вікна надає:

- **Контролі переміщення**: Ручне переміщення та позиціонування машини
- **Статус машини**: Позиція та стан з'єднання в реальному часі
- **Перегляд журналу**: Комунікація G-коду та історія операцій
- **Керування WCS**: Вибір та обнулення робочої системи координат

Дивіться [Панель керування](control-panel) для детальної інформації.

## Керування вікном

### Панелі

Показати/приховати панелі за потребою:

- **Панель шарів**: Вид → Панель шарів (<kbd>ctrl+l</kbd>)
- **Панель властивостей**: Вид → Панель властивостей (<kbd>ctrl+i</kbd>)

### Повноекранний режим

Зосередьтеся на роботі в повноекранному режимі:

- Увійти: <kbd>f11</kbd> або Вид → Повний екран
- Вийти: <kbd>f11</kbd> або <kbd>esc</kbd>

## Налаштування

Налаштуйте інтерфейс у **Редагування → Налаштування**:

- **Тема**: Світла, темна або системна
- **Одиниці**: Міліметри або дюйми
- **Сітка**: Показати/приховати та налаштувати крок сітки
- **Лінійки**: Показати/приховати лінійки на полотні
- **Панель інструментів**: Налаштувати видимі кнопки
