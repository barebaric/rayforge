# Встановлення

import InstallGuide from '@site/src/components/InstallGuide';

Rayforge доступний для Linux, Windows та macOS. Виберіть свою платформу нижче
та дотримуйтесь інструкцій з встановлення.

## Системні вимоги

| Ресурс              | Встановлено   | Рекомендовано           |
|---------------------|---------------|-------------------------|
| Дисковий простір    | < 1 ГБ        | Залежить від проекту    |
| Оперативна пам'ять  | 1 ГБ          | Залежить від проекту    |

Фактичні вимоги до диска та пам'яті залежать від складності проекту. Прості проекти
можуть потребувати небагато більше встановленого обсягу, тоді як складні проекти можуть
легко спожити 10 ГБ оперативної пам'яті або більше та вимагати значного додаткового
дискового простору.
