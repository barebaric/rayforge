# Instalação

import InstallGuide from '@site/src/components/InstallGuide';

O Rayforge está disponível para Linux, Windows e macOS. Escolha sua plataforma abaixo
e siga as instruções de instalação.

## Requisitos do Sistema

| Recurso         | Instalado   | Recomendado          |
|-----------------|-------------|----------------------|
| Espaço em disco | < 1 GB      | Varia pelo projeto   |
| RAM             | 1 GB        | Varia pelo projeto   |

Os requisitos reais de disco e memória dependem da complexidade do projeto.
Projetos simples podem precisar de pouco além do instalado, enquanto projetos
complexos podem facilmente consumir 10 GB de RAM ou mais e exigir espaço em
disco adicional significativo.
