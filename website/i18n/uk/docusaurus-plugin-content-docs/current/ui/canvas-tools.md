# Інструменти полотна

Полотно надає інструменти для маніпулювання дизайнами та підготовки ваших лазерних робіт.

## Інструмент виділення

Виділяйте, переміщуйте та трансформуйте об'єкти на полотні.

**Використання:**

- Клацніть для виділення одного об'єкта
- <kbd>ctrl</kbd> + клацання для виділення кількох об'єктів
- Перетягніть для створення рамки виділення
- Клацніть та перетягніть виділені об'єкти для переміщення

**Клавіші-модифікатори під час перетягування:**

Утримання клавіш-модифікаторів під час перетягування змінює поведінку:

- <kbd>ctrl</kbd> під час **переміщення**: Прилипає до сітки 5мм, полегшуючи точне вирівнювання
- <kbd>ctrl</kbd> під час **обертання**: Прилипає до кута з кроком 5 градусів для узгоджених кутів
- <kbd>shift</kbd> під час **масштабування**: Обмежує пропорції (рівномірне масштабування)
- <kbd>ctrl</kbd> під час **масштабування**: Масштабує від центру замість протилежного краю

**Маркери трансформації:**

- **Кутові маркери**: Масштабувати пропорційно (утримуйте <kbd>shift</kbd> для непропорційного масштабування)
- **Бокові маркери**: Масштабувати в одному напрямку
- **Маркер обертання**: Клацніть один раз на виділеному об'єкті для перемикання режиму обертання, потім перетягніть для обертання навколо центральної точки

**Гарячі клавіші:**

- <kbd>стрілки</kbd>: Перемістити виділені об'єкти на 1 одиницю
- <kbd>shift+стрілки</kbd>: Перемістити на 10 одиниць
- <kbd>ctrl+d</kbd>: Дублювати виділення

## Навігація

Навігуйте по полотну за допомогою миші та клавіатури:

**Панорама:**

- Середня кнопка миші та перетягування
- Утримуйте <kbd>пробіл</kbd> з будь-яким активним інструментом і перетягуйте

**Масштаб:**

- <kbd>ctrl+"+"</kbd>: Збільшити
- <kbd>ctrl"-"</kbd>: Зменшити
- <kbd>ctrl+0</kbd>: Скинути масштаб (до вікна)
- Колесо миші: Збільшити/зменшити в позиції курсора

## Інструменти вирівнювання

Вирівнюйте та розподіляйте кілька об'єктів.

**Параметри вирівнювання:**

- Вирівняти по лівих краях
- Вирівняти по правих краях
- Вирівняти по верхніх краях
- Вирівняти по нижніх краях
- Вирівняти по горизонтальних центрах
- Вирівняти по вертикальних центрах

**Параметри розподілу:**

- Розподілити горизонтально
- Розподілити вертикально
- Рівний інтервал

**Використання:**

1. Виділіть кілька об'єктів
2. Виберіть параметр вирівнювання/розподілу з панелі інструментів або меню Редагування
3. Об'єкти вирівнюються/розподіляються негайно

## Трансформація об'єктів

Трансформуйте об'єкти чисельно для точності.

**Доступно через панель властивостей:**

- **Позиція (X, Y)**: Точні координати
- **Розмір (Ш, В)**: Точні розміри
- **Обертання**: Градуси від горизонталі
- **Масштаб**: Відсоток від початкового розміру

## Поради для ефективного використання полотна

1. **Використовуйте гарячі клавіші**: Набагато швидше, ніж клацання по панелі інструментів
2. **Опануйте панораму та масштабування**: Необхідно для великих або детальних дизайнів
3. **Групуйте пов'язані об'єкти**: Легше переміщувати та організовувати (<kbd>ctrl+g</kbd> для групування)

## Редагування векторів

Двічі клацніть на робочу деталь на полотні, щоб увійти в режим редагування
векторів. У цьому режимі ви можете вибирати та видаляти окремі векторні
сегменти робочої деталі.

Щоб вийти з режиму редагування векторів, клацніть поза робочою деталлю або
натисніть <kbd>Escape</kbd>.
