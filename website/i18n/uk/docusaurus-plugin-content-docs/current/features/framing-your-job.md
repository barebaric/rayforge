# Обведення вашої роботи

Дізнайтеся, як використовувати функцію обведення для попереднього перегляду меж вашої роботи та забезпечення правильного вирівнювання перед різанням.

## Огляд

Обведення дозволяє вам попередньо переглянути точні межі вашої лазерної роботи шляхом трасування контуру з лазером на низькій потужності або з вимкненим лазером. Це допомагає перевірити позиціонування та запобігти дорогим помилкам.

## Коли використовувати обведення

- **Перші налаштування**: Перевірити розміщення матеріалу
- **Точне позиціонування**: Переконатися, що дизайн поміщається в межах матеріалу
- **Кілька робіт**: Підтвердити вирівнювання перед кожним запуском
- **Дорогі матеріали**: Перевірити двічі перед тим, як робити різи

## Як обводити

### Метод 1: Тільки контур

Трасувати межу роботи без вмикання лазера:

1. **Завантажте ваш дизайн** в Rayforge
2. **Розмістіть матеріал** на робочому столі лазера
3. **Натисніть кнопку Обвести** на панелі інструментів
4. **Спостерігайте, як лазерна головка** трасує прямокутник межі
5. **Перевірте позиціонування** та скоригуйте матеріал за потреби

### Метод 2: Попередній перегляд на низькій потужності

Деякі машини підтримують обведення на низькій потужності з видимим променем:

1. **Увімкніть режим низької потужності** в налаштуваннях машини
2. **Встановіть потужність обведення** (зазвичай 1-5%)
3. **Запустіть операцію обведення**
4. **Спостерігайте контур**, що трасується на поверхні матеріалу

:::warning Перевірте вашу машину
Не всі лазери безпечно підтримують обведення на низькій потужності. Зверніться до документації вашої машини перед використанням цієї функції.
:::


## Налаштування обведення

Налаштуйте поведінку обведення в Налаштування → Машина:

- **Швидкість обведення**: Як швидко переміщується лазерна головка під час обведення
- **Потужність обведення**: Потужність лазера під час обведення (0 для вимкнення, низький % для видимого сліду)
- **Пауза на кутах**: Коротка пауза на кожному куті для видимості
- **Кількість повторень**: Кількість разів для трасування контуру

## Використання результатів обведення

Після обведення ви можете:

- **Скоригувати позицію матеріалу** за потреби
- **Обвести знову** для перевірки нової позиції
- **Продовжити роботу** коли задоволені розміщенням

## Поради для ефективного обведення

- **Позначте кути**: Розмістіть малі шматочки стрічки на кутах для орієнтира
- **Перевірте зазор**: Забезпечте адекватний простір навколо вашого дизайну
- **Перевірте орієнтацію**: Підтвердіть, що матеріал орієнтований правильно
- **Врахуйте керф**: Пам'ятайте, що різи будуть трохи ширші за контури

## Обведення з камерою

Якщо ваша машина підтримує камеру, ви можете:

1. **Захопити зображення камери** розміщення матеріалу
2. **Накласти дизайн** на вид з камери
3. **Скоригувати позицію** віртуально перед обведенням
4. **Обвести для підтвердження** фізичного вирівнювання

Дивіться [Інтеграція камери](../machine/camera) для деталей.

## Виправлення проблем

**Обведення не відповідає дизайну**: Перевірте початок роботи та налаштування системи координат

**Лазер стріляє під час обведення**: Вимкніть потужність обведення або перевірте налаштування машини

**Обведення занадто швидке щоб побачити**: Зменшіть швидкість обведення в налаштуваннях

**Головка не досягає кутів**: Перевірте, що дизайн знаходиться в межах робочої області машини

## Примітки безпеки

- **Ніколи не залишайте машину без нагляду** під час обведення
- **Перевірте, що лазер вимкнено** якщо використовуєте обведення з нульовою потужністю
- **Тримайте руки подалі** від траєкторії лазерної головки
- **Слідкуйте за перешкодами**, які можуть завадити русу
