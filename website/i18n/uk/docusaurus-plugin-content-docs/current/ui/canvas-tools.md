# Інструменти полотна

Полотно надає комплексний набір інструментів для маніпулювання дизайнами, вимірювання та підготовки ваших лазерних робіт.

## Інструмент виділення

Виділяйте, переміщуйте та трансформуйте об'єкти на полотні.

**Використання:**

- Клацніть для виділення одного об'єкта
- <kbd>ctrl</kbd> + клацання для виділення кількох об'єктів
- Перетягніть для створення рамки виділення
- Клацніть та перетягніть виділені об'єкти для переміщення

**Маркери трансформації:**

- **Кутові маркери**: Масштабувати пропорційно (утримуйте <kbd>shift</kbd> для непропорційного масштабування)
- **Бокові маркери**: Масштабувати в одному напрямку
- **Маркер обертання**: Обертати навколо центральної точки

**Гарячі клавіші:**

- <kbd>стрілки</kbd>: Перемістити виділені об'єкти на 1 одиницю
- <kbd>shift+стрілки</kbd>: Перемістити на 10 одиниць
- <kbd>ctrl+d</kbd>: Дублювати виділення

## Інструмент панорами

Навігуйте по полотну без випадкового переміщення об'єктів.

**Використання:**

- Клацніть та перетягніть для панорамування
- Або утримуйте <kbd>пробіл</kbd> з будь-яким активним інструментом і перетягуйте

## Інструмент масштабування

Збільшуйте певні області вашого дизайну.

**Використання:**

- Клацніть для збільшення в точці
- <kbd>alt</kbd> + клацання для зменшення
- Клацніть та перетягніть для масштабування до певної області

**Гарячі клавіші:**

- <kbd>ctrl+"+"</kbd>: Збільшити
- <kbd>ctrl+"-"</kbd>: Зменшити
- <kbd>ctrl+0</kbd>: Скинути масштаб (до вікна)
- Колесо миші: Збільшити/зменшити в позиції курсора

## Інструмент вимірювання

Вимірюйте відстані та кути на полотні.

**Використання:**

- Клацніть початкову точку
- Клацніть кінцеву точку для завершення вимірювання
- Вимірювання відображається в поточних одиницях (мм або дюйми)

**Функції:**

- Відстань між двома точками
- Кут відносно горизонталі
- Попередній перегляд в реальному часі під час вимірювання

## Інструменти вирівнювання

Вирівнюйте та розподіляйте кілька об'єктів.

**Параметри вирівнювання:**

- Вирівняти по лівих краях
- Вирівняти по правих краях
- Вирівняти по верхніх краях
- Вирівняти по нижніх краях
- Вирівняти по горизонтальних центрах
- Вирівняти по вертикальних центрах

**Параметри розподілу:**

- Розподілити горизонтально
- Розподілити вертикально
- Рівний інтервал

**Використання:**

1. Виділіть кілька об'єктів
2. Виберіть параметр вирівнювання/розподілу з панелі інструментів або меню Редагування
3. Об'єкти вирівнюються/розподіляються негайно

## Сітка та прив'язка

Допомагають точному позиціонуванню з сіткою та прив'язкою.

**Сітка:**

- Перемикання: Вид → Показати сітку (<kbd>ctrl+g</kbd>)
- Налаштувати крок у Налаштуваннях
- Тільки візуальний посібник (прив'язка опціональна)

**Прив'язка:**

- **Прив'язка до сітки**: Вирівнювати об'єкти до точок сітки
- **Прив'язка до об'єктів**: Вирівнювати до країв інших об'єктів
- **Прив'язка до початку**: Вирівнювати до початку координат машини (0,0)

Перемикання прив'язки: Вид → Прив'язка (<kbd>ctrl+shift+g</kbd>)

## Трансформація об'єктів

Трансформуйте об'єкти чисельно для точності.

**Доступно через панель властивостей:**

- **Позиція (X, Y)**: Точні координати
- **Розмір (Ш, В)**: Точні розміри
- **Обертання**: Градуси від горизонталі
- **Масштаб**: Відсоток від початкового розміру

## Булеві операції

Об'єднуйте або віднімайте форми:

- **Об'єднання**: Злити перекриваючі форми
- **Різниця**: Відняти одну форму від іншої
- **Перетин**: Залишити тільки перекриваючі області
- **Виключення**: Видалити перекриваючі області

**Використання:**

1. Виділіть два або більше векторних об'єктів
2. Виберіть булеву операцію з меню Редагування
3. Результат замінює виділені об'єкти

## Поради для ефективного використання полотна

1. **Використовуйте гарячі клавіші**: Набагато швидше, ніж клацання по панелі інструментів
2. **Опануйте панораму та масштабування**: Необхідно для великих або детальних дизайнів
3. **Прив'язка до сітки**: Прискорює вирівнювання для прямокутних макетів
4. **Спочатку виміряйте**: Перевірте розміри перед генерацією G-коду
5. **Групуйте пов'язані об'єкти**: Легше переміщувати та організовувати (<kbd>ctrl+g</kbd> для групування)
