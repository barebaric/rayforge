# Deepnest

Deepnest 自动将您的工作件以紧凑的布局排列在您的材料或机器工作区域上。它使用遗传算法寻找形状的高效排列方式，最大限度地减少浪费，并在每张板材上放置更多零件。

![Deepnest 设置对话框](/screenshots/addon-deepnest.png)

## 前提条件

在运行嵌套之前，请在画布上选择一个或多个工作件。您还可以选择库存项目来定义板材边界。如果未选择库存，插件将使用文档库存或回退到机器工作区域。

## 运行嵌套布局

从**排列**菜单、工具栏按钮或键盘快捷键 **Ctrl+Alt+N** 触发嵌套布局。在算法运行之前会打开一个设置对话框。

## 嵌套设置

设置对话框在嵌套算法开始之前提供以下选项。

**间距**设置嵌套形状之间的距离，单位为毫米。默认值取自您机器的激光光斑大小。增大此值可在零件之间添加安全余量。

**约束旋转**使所有零件保持原始方向。当此选项关闭时，算法以10度为增量旋转零件以寻找更紧密的贴合。不约束旋转可以产生更好的材料利用率，但计算时间更长。

**允许水平翻转**在嵌套过程中水平镜像零件。这可以帮助更紧密地排列零件，但产生的切割结果将被镜像。

**允许垂直翻转**在嵌套过程中垂直镜像零件。关于镜像输出的同样注意事项也适用。

点击**开始嵌套**开始。对话框关闭，算法在后台运行。嵌套进行期间，底部面板中会出现进度指示器。

## 嵌套完成后

当算法完成时，画布上的所有工作件都会重新定位到其嵌套位置。位置作为单个可撤销操作应用，因此如果结果不符合您的需要，可以通过一步撤销布局。

如果算法无法将所有工作件放置在可用库存上，未放置的项目会被移到库存区域的右侧，以便保持可见且易于识别。

如果嵌套结果比原始布局更差——例如，零件已经排列得很好——工作件将保持在原始位置。
