# 固定标签

固定标签（也称为桥接或标签）是沿切割路径留下的小未切割段，用于保持零件与周围材料连接。这可以防止切割件在作业期间移动，这可能导致错位、损坏或火灾危险。

## 为什么要使用固定标签？

切割穿透材料时，切割件可能会：

- **移位** - 作业中途，导致后续操作错位
- **掉落** - 穿过床格栅或如果仅在边缘支撑则倾斜
- **碰撞** - 移动时与激光头碰撞
- **着火** - 如果掉落在下方热废料上
- **损坏** - 由于掉落或振动

固定标签通过保持零件连接直到您准备移除它来解决这些问题。

---

## 固定标签如何工作

Rayforge 通过**在切割路径中创建小间隙**来实现标签：

1. 您在切割路径上标记标签应该在哪里
2. 在 G 代码生成期间，Rayforge 在每个标签处中断切割
3. 激光抬起（或关闭），跳过标签宽度，然后恢复切割
4. 作业完成后，您手动折断或切割标签以释放零件

---

## 添加固定标签

### 快速添加

1. **选择工件**要添加标签的（必须是切割/轮廓操作）
2. **点击标签工具**在工具栏中或按标签快捷键
3. **点击路径**在您想要标签的位置：
   - 标签在路径轮廓上显示为小手柄
   - 多次点击以添加更多标签
   - 小零件通常 3-4 个标签，较大的零件更多
4. **启用标签**如果尚未启用（在属性面板中切换）

### 使用添加标签弹出窗口

更多控制：

1. **右键点击**工件或使用 **编辑 → 添加标签**
2. **选择标签放置方法：**
   - **手动：** 点击单独的位置
   - **等距：** 自动沿路径均匀分布标签
3. **配置标签设置：**
   - **标签数量：** 创建多少标签（用于等距）
   - **标签宽度：** 每个未切割段的长度（通常 2-5mm）
4. **点击应用**

---

## 标签属性

### 标签宽度

**宽度**是沿路径的未切割段的长度。

**推荐宽度：**

| 材料     | 厚度  | 标签宽度 |
| -------- | ----- | -------- |
| **纸板** | 1-3mm | 2-3mm    |
| **胶合板**| 3mm   | 3-4mm    |
| **胶合板**| 6mm   | 4-6mm    |
| **亚克力**| 3mm   | 2-3mm    |
| **亚克力**| 6mm   | 3-5mm    |
| **MDF**  | 3mm   | 3-4mm    |
| **MDF**  | 6mm   | 5-7mm    |

**指南：**
- **较厚的材料**需要更宽的标签以获得强度
- **较重的零件**需要更多和/或更宽的标签
- **脆性材料**（亚克力）可以使用较小的标签（更容易折断）
- **纤维材料**（木材）可能需要更宽的标签

:::warning 标签宽度与材料厚度
标签必须足够宽以支撑零件，但又要足够小以便干净地移除。太窄 = 零件可能脱落；太宽 = 难以移除或损坏零件。
:::

### 标签位置

标签使用两个参数定位：

- **段索引：** 路径的哪个线/圆弧段
- **位置 (0.0 - 1.0)：** 沿该段的位置（0 = 开始，1 = 结束）

**手动放置提示：**
- 尽可能在**直线段**上放置标签（更容易移除）
- 避免在**紧曲线**上放置标签（应力集中）
- 在零件周围**均匀**分布标签
- 如需要，在**角落**放置标签以获得最大支撑

### 等距标签

**等距**功能自动以均匀间隔放置标签：

**好处：**
- 均匀的重量分布
- 可预测的断裂模式
- 规则形状的快速设置

## 使用标签

### 编辑标签

**移动标签：**
1. 选择工件
2. 沿路径拖动标签手柄
3. 释放以设置新位置

**调整标签大小：**
- 使用属性面板调整宽度
- 工件上的所有标签共享相同的宽度

**删除标签：**
1. 点击标签手柄选择它
2. 按 Delete 或使用上下文菜单
3. 或清除所有标签并重新开始

### 启用/禁用标签

切换标签开/关而不删除它们：

- **工件属性面板：**"启用标签"复选框
- **工具栏：**标签可见性切换图标

**禁用时：**
- 标签不在 G 代码中生成
- 标签手柄在画布中隐藏
- 路径完全切割

**用例：**暂时禁用标签以测试切割，然后为生产重新启用。

---

## 切割后移除标签

**工具：**
- 工艺刀或美工刀
- 斜口钳
- 凿子（用于木材）
- 厚材料用细锯

**技术：**

1. 如果可接触，从两侧**划标签**
2. **轻轻弯曲**零件以施加标签应力
3. **切穿**剩余材料
4. **打磨或锉**标签残余与边缘齐平

**对于脆性材料（亚克力）：**
- 使用最少的标签（它们容易折断）
- 折断前深划
- 折断标签时支撑零件以避免裂纹

**对于木材：**
- 标签可能需要切割（不会干净折断）
- 使用锋利的刀或凿子
- 切齐，然后打磨光滑
