# Розширені налаштування

Сторінка "Розширені" у налаштуваннях машини містить додаткові параметри конфігурації для спеціалізованих випадків використання.

![Розширені налаштування](/screenshots/machine-advanced.png)

## Поведінка підключення

Налаштування, що контролюють, як Rayforge взаємодіє з вашою машиною під час підключення.

### Homing при підключенні

Коли увімкнено, Rayforge автоматично надсилає команду homing ($H) при підключенні до машини.

- **Увімкніть, якщо**: Ваша машина має надійні кінцеві вимикачі
- **Вимкніть, якщо**: Ваша машина не має кінцевих вимикачів або homing ненадійний

### Очищення аварій при підключенні

Коли увімкнено, Rayforge автоматично очищає будь-який стан аварії при підключенні.

- **Увімкніть, якщо**: Ваша машина часто запускається в стані аварії
- **Вимкніть, якщо**: Ви хочете вручну дослідити аварії перед очищенням

### Дозволити Homing окремої осі

Коли увімкнено, ви можете виконувати homing окремих осей незалежно (X, Y або Z) замість вимоги всіх осей разом. Це корисно для машин, де одна вісь вже може бути правильно позиціонована.

## Налаштування дуг та кривих

Налаштування для контролю того, як криві шляхи перетворюються на рухи G-code.

### Підтримка дуг

Коли увімкнено, Rayforge генерує команди дуг (G2/G3) для кривих шляхів замість розбиття їх на багато малих лінійних рухів. Це виробляє більш компактний G-code і плавніший рух на більшості контролерів.

Коли вимкнено, всі криві перетворюються на лінійні сегменти (команди G1), що забезпечує максимальну сумісність з контролерами, які не підтримують дуги.

### Підтримка кривих Bézier

Коли увімкнено, Rayforge генерує нативні кубічні команди Bézier (такі як команда G5, що використовується в LinuxCNC) для кривих шляхів. Це забезпечує дуже плавний рух і компактний G-code на контролерах, які це підтримують. Вам слід вимкнути це налаштування, якщо мікропрограма вашої машини не розуміє команди Bézier, у такому разі криві будуть розбиті на лінійні сегменти.

### Толерантність дуги та кривої

Це налаштування контролює максимальне дозволене відхилення при підгонці дуг та кривих до кривих шляхів, вказане в міліметрах. Менше значення виробляє більш точні шляхи, але може вимагати більше команд. Більше значення дозволяє більше відхилення, але генерує менше команд.

Типові значення варіюються від 0.01мм для точної роботи до 0.1мм для швидшої обробки.
