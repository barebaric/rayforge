# 工作坐标系（WCS）

工作坐标系（WCS）允许您在机器工作区域上定义多个参考点。这使得在同一作业的不同位置运行相同作业变得容易，而无需重新设计或重新定位工件。

## 理解 WCS

将 WCS 视为可自定义的"零点"用于您的工作。虽然您的机器有固定的原点位置（由限位开关确定），但 WCS 让您定义希望工作从哪里开始。

### 为什么使用 WCS？

- **多个夹具**：在床上设置多个工作区域并在它们之间切换
- **可重复定位**：在不同位置运行相同作业
- **快速对齐**：根据材料或工件设置参考点
- **生产工作流程**：在工作区域组织多个作业

## WCS 类型

Rayforge 支持以下坐标系统：

| 系统    | 类型     | 描述                                             |
| ------- | -------- | ------------------------------------------------ |
| **G53** | 机器     | 绝对机器坐标（固定，无法更改）                   |
| **G54** | 工作 1   | 第一个工作坐标系（默认）                         |
| **G55** | 工作 2   | 第二个工作坐标系                                 |
| **G56** | 工作 3   | 第三个工作坐标系                                 |
| **G57** | 工作 4   | 第四个工作坐标系                                 |
| **G58** | 工作 5   | 第五个工作坐标系                                 |
| **G59** | 工作 6   | 第六个工作坐标系                                 |

### 机器坐标（G53）

G53 代表机器的绝对位置，零点在机器的原点位置。这由您的硬件固定，无法更改。

**何时使用：**

- 归零和校准
- 相对于机器限位的绝对定位
- 当您需要引用物理机器位置时

### 工作坐标（G54-G59）

这些是您可以定义的偏移坐标系统。每个都有自己的零点，可以设置在工作区域的任何位置。

**何时使用：**

- 设置多个工作夹具
- 对齐材料位置
- 在不同位置运行相同作业

## 在界面中可视化 WCS

### 2D 画布

2D 画布显示带有绿色标记的 WCS 原点：

- **绿线**：指示当前 WCS 原点 (0, 0) 位置
- **网格对齐**：网格线与 WCS 原点对齐，而不是机器原点

当您更改活动的 WCS 或其偏移时，原点标记会移动，准确显示工作将从哪里开始。

### 3D 预览

在 3D 预览中，WCS 显示方式不同：

- **网格和轴**：整个网格显示为 WCS 原点是世界原点
- **隔离视图**：WCS"隔离"显示 - 看起来网格以 WCS 为中心，而不是机器
- **标签**：坐标标签相对于 WCS 原点

这使得可视化作业相对于所选工作坐标系的运行位置变得容易。

## 选择和更改 WCS

### 通过工具栏

1. 在主工具栏中找到 WCS 下拉菜单（默认标记为"G53"）
2. 点击查看可用的坐标系统
3. 选择要使用的 WCS

### 通过控制面板

1. 打开控制面板（视图 → 控制面板或 Ctrl+L）
2. 在机器状态部分找到 WCS 下拉菜单
3. 从下拉菜单中选择所需的 WCS

## 设置 WCS 偏移

您可以定义每个 WCS 原点在机器上的位置。

### 在当前位置设置零点

1. 连接到您的机器
2. 选择要配置的 WCS（例如 G54）
3. 将激光头点动到希望为 (0, 0) 的位置
4. 在控制面板中，点击零按钮：
   - **零 X**：将当前 X 位置设置为活动 WCS 的 0
   - **零 Y**：将当前 Y 位置设置为活动 WCS 的 0
   - **零 Z**：将当前 Z 位置设置为活动 WCS 的 0

偏移存储在您的机器控制器中，并在会话之间持续存在。

### 查看当前偏移

控制面板显示活动 WCS 的当前偏移：

- **当前偏移**：显示距机器原点的 (X, Y, Z) 偏移
- **当前位置**：显示激光头在活动 WCS 中的位置

## 作业中的 WCS

运行作业时，Rayforge 使用活动的 WCS 来定位您的工作：

1. 在画布中设计作业
2. 选择要使用的 WCS
3. 运行作业 - 它将根据 WCS 偏移定位

只需更改活动的 WCS，即可在不同位置运行相同作业。

## 实用工作流程

### 工作流程 1：多个夹具位置

您有一个大床并希望设置三个工作区域：

1. **归零机器**以建立参考
2. **点动到第一个工作区域**并设置 G54 偏移（零 X、零 Y）
3. **点动到第二个工作区域**并设置 G55 偏移
4. **点动到第三个工作区域**并设置 G56 偏移
5. 现在您可以在 G54、G55 和 G56 之间切换以在每个区域运行作业

### 工作流程 2：对齐材料

您有一块材料放在床上的某处：

1. 将激光头**点动**到材料的角落
2. **选择 G54**（或您首选的 WCS）
3. **点击零 X 和零 Y** 将材料角设置为 (0, 0)
4. 以 (0, 0) 为原点**设计作业**
5. **运行作业** - 它将从材料角开始

### 工作流程 3：生产网格

您需要在不同的位置切割同一零件 10 次：

1. 在 Rayforge 中**设计一个零件**
2. 为所需位置**设置 G54-G59** 偏移
3. **运行作业**，G54 活动
4. **切换到 G55** 并再次运行
5. 对每个 WCS 位置**重复**

## 重要说明

### WCS 限制

- **G53 无法更改**：机器坐标由硬件固定
- **偏移持续**：WCS 偏移存储在您的机器控制器中
- **需要连接**：您必须连接到机器才能设置 WCS 偏移

### WCS 和作业原点

WCS 独立于您的作业原点设置工作。作业原点确定作业在画布上的开始位置，而 WCS 确定该画布位置映射到机器上的位置。

### 机器兼容性

并非所有机器都支持所有 WCS 功能：

- **GRBL（v1.1+）**：完全支持 G53-G59
- **Smoothieware**：支持 G54-G59（偏移读取可能有限）
- **自定义控制器**：因实现而异
