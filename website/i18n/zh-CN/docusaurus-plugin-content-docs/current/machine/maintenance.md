# 维护

机器设置中的维护页面帮助您跟踪机器使用情况并安排维护任务。

![维护设置](/screenshots/machine-maintenance.png)

## 小时计数器

Rayforge 跟踪您的机器已使用多长时间。这些计数器帮助您安排预防性维护。

### 激光小时数

跟踪激光已发射的总时间。使用此来：

- 安排激光管更换（CO2 管通常可持续 1000-3000 小时）
- 监控激光二极管退化
- 跟踪保修期

### 作业小时数

跟踪运行作业所花费的总时间，包括行程移动。使用此来：

- 安排皮带张紧
- 计划轴承润滑
- 跟踪整体机器使用情况

### 移动小时数

跟踪移动（非切割）所花费的时间。使用此来：

- 安排步进电机维护
- 计划导轨清洁和润滑

## 重置计数器

执行维护后，您可以重置相关计数器：

1. 点击计数器旁边的重置按钮
2. 在对话框中确认重置
3. 计数器归零

:::tip 维护计划
常见维护间隔：
- **每天**：清洁透镜，检查镜片对齐
- **每周**：清洁导轨，检查皮带张力
- **每月**：润滑轴承，检查电气连接
- **每年**：全面检查，更换磨损零件
:::
